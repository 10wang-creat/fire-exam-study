## 一、四種系統比較（必背）

```
    ┌─────────┐
    │ 自動撒水 │
    └────┬────┘
   ┌─────┼──────┬──────────┐
密閉濕式  密閉乾式  預動式   開放式
(管充水)  (管充空氣) (雙確認)  (空管)
 ↓         ↓        ↓        ↓
最常見    低溫場所  電腦房   舞台停車場
反應最快  防結冰   防誤噴   一齊開放閥
```

| 系統 | 管內 | 啟動方式 | 放水方式 |
|------|------|---------|---------|
| 密閉濕式 | 充水 | 撒水頭熱熔/破裂 | 逐一開放 |
| 密閉乾式 | 壓縮空氣 | 撒水頭動作→洩壓→進水 | 逐一開放 |
| 預動式 | 壓縮空氣 | 探測器+撒水頭雙確認 | 逐一開放 |
| 開放式 | 空管 | 探測器→一齊開放閥 | **全區同時** |

## 二、撒水頭裝置位置規定（103年考題）

### 2-1 淨空間

| 場所 | 迴水板下方 | 水平方向 |
|------|----------|---------|
| **一般場所** | **45** cm | **30** cm |
| **易燃物品場所** | **90** cm | **30** cm |

### 2-2 配置間距

| 場所/型式 | 任一點至撒水頭水平距離 |
|----------|---------------------|
| 防火構造一般場所 | ≤ **2.3** m |
| 非防火構造 | ≤ **2.1** m |
| 公共危險物品場所 | ≤ **1.7** m |
| 高架倉庫（貨架上） | ≤ **2.5** m，**交錯**設置 |
| 快速反應型（防火構造） | ≤ **2.3** m |
| 小區劃型 | ≤ **2.6** m，防護面積 ≤ 13 m² |
| 放水型（天花板>6m可燃物/10m其他）| 依設計 |

### 2-3 密閉式撒水頭其他規定

- 迴水板與裝置面間距 ≤ **30** cm
- 樑下安裝：迴水板與樑底間距 ≤ **10** cm，且與樓板間距 ≤ **50** cm
- 風管寬度 > **110** cm → 風管下方亦應設撒水頭
- 不得傾斜裝置（軸心與裝置面成 **90°**）

## 三、水源容量（104年考題）

### 密閉式撒水（一般場所）

| 場所 | 快速反應型 | 一般反應型 |
|------|----------|----------|
| **11F↑/地下建築物** | 15個×20min | 30個×20min |
| **10F↓ 甲類第四目** | 12個×20min | 15個×20min |
| **10F↓ 地下層** | 12個×20min | 15個×20min |
| **10F↓ 其他** | 8個×20min | 15個×20min |
| **高架倉庫（易燃）** | — | 30個×20min |
| **高架倉庫（其他）** | — | 20個×20min |

> 每個撒水頭 **80 L/min**（一般）或 **60 L/min**（快速反應型輕度場所）

### 公共危險物品場所

密閉式：**30** 個撒水頭 × **20** 分鐘

### 水道連結型

- 撒水頭數：側壁型/小區劃型，10F↓ = **8** 個 / 11F↑ = **12** 個
- 放水時間：**20** 分鐘

## 四、開放式撒水——一齊開放閥系統（106年考題）

### 系統圖構件

```
     加壓送水裝置（幫浦）
          ↓
     ┌──主管──┐
     │        │
  一齊開放閥  壓力開關(PS)
     │
  ┌──┼──┐
  支管  支管  支管
  │    │    │
  開放式撒水頭（全區）
```

一齊開放閥附近五大構件：
1. **一齊開放閥**本體
2. **壓力開關（PS）**——啟動加壓送水裝置
3. **感知撒水頭**（或探測器）——偵測火災
4. **手動啟動開關**
5. **排水閥**——測試用

### 一齊開放閥性能檢查

| 檢查項目 | 方法 |
|---------|------|
| 外觀 | 無變形、損傷、洩漏 |
| 閥體動作 | 操作手動啟動裝置確認閥體開啟 |
| 排水功能 | 開啟排水閥確認排水正常 |
| 壓力開關 | 閥體開啟後 PS 應連動啟動加壓送水裝置 |
| 復歸 | 關閉後應能確實復歸 |

## 五、密閉式撒水綜合檢查（107年考題）

### 檢查方法

1. 使用**正常電源**供電
2. 開啟**末端查驗閥**
3. 確認加壓送水裝置自動啟動
4. 測量末端查驗閥之放水壓力
5. 確認流水檢知裝置之警報功能
6. 確認壓力開關連動

### 判定方法

| 項目 | 合格標準 |
|------|---------|
| 末端查驗閥放水壓力 | **1** kgf/cm² 以上（幫浦啟動後） |
| 加壓送水裝置 | 應確實自動啟動 |
| 流水檢知裝置 | 警報應正常鳴動 |
| 表示裝置 | 該區域之表示應正常 |

### 注意事項

- 放水壓力超過 **7** kgf/cm² 應確認減壓措施
- 乾式/預動式：撒水頭動作後 **1** 分鐘內應放水
- 乾式空氣壓試驗：空氣壓力達 **0.28** MPa，持續 **24** 小時，漏氣減壓量 ≤ **0.01** MPa
- 測試後應確實復歸

## 六、水道連結型自動撒水（108、110年考題）

### 適用場所

樓地板面積合計 **未達 1,000 m²** 之避難弱勢場所：
- 產後護理之家
- 團體家屋
- 精神護理之家
- 老人福利機構

### 與密閉濕式比較

| 項目 | 密閉濕式（一般） | 水道連結型 |
|------|----------------|-----------|
| 水源 | 專用蓄水池/水箱 | **自來水直結** |
| 放水壓力 | ≥ 1 kgf/cm² | ≥ **0.5** kgf/cm² |
| 放水量 | 60~80 L/min | 依設計 |
| 撒水頭數（水源） | 8~30個 | 8~12個 |
| 放水時間 | 20 min | 20 min |
| 特點 | 需幫浦/水箱 | 免幫浦，直接接自來水 |

## 七、高架儲存倉庫撒水頭（111年考題）

### 定義

**高架儲存倉庫** = 儲區高度 **超過 10 m** 且樓地板面積 ≥ **700 m²** 之高架式倉庫

### 撒水頭配置

| 項目 | 規定 |
|------|------|
| 天花板撒水頭 | 任一點至撒水頭 ≤ **2.3** m |
| 貨架撒水頭 | 任一點至撒水頭 ≤ **2.5** m，**交錯**設置 |
| 易燃物品 | 每 **4** m 高度至少設一個 |
| 撒水障礙 | 物品側面亦應設置 |
| 放水量 | ≥ **114** L/min |
