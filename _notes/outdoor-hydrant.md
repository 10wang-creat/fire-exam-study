## 室外消防栓規格

| 項目 | 規定 |
|------|------|
| 口徑 | 63mm |
| 水帶口徑 | 63mm |
| 水帶長度 | 20m |
| 瞄子口徑 | 19mm |
| **放水量** | ≥ **350 L/min** |
| **放水壓力** | ≥ 2.5 kgf/cm²（0.25 MPa）|
| 同時使用數 | **2支** |

## 水源容量

`水源 = 350 × 2 × 20min = 14,000 L = 14 m³`

## 設置條件

- 建築物各層任一點至室外消防栓的步行距離 ≤ **40m**（第一種）
- 設於建築物外牆面或基地內

## 室內 vs 室外比較

| 項目 | 室內（第一種） | 室外 |
|------|-------------|------|
| 放水量 | 130 L/min | **350 L/min** |
| 水帶口徑 | 38/50mm | **63mm** |
| 瞄子口徑 | 13mm | **19mm** |
| 同時使用 | 2支 | 2支 |
| 水源 | 5.2 m³ | **14 m³** |
| 操作人數 | 2人 | 2人以上 |
| 步行距離 | 25m | 40m |

## 送水口

- 供消防車連結送水之用
- 口徑 63mm 雙口陽式快速接頭
- 設於消防車容易接近處
- 離地面 **1m** 以下

## 記憶口訣

> **「室外350比室內130大很多」**
>
> 室外消防栓：350 L/min × 2支 × 20min = 14噸水
> 水帶63mm、瞄子19mm、步行40m
