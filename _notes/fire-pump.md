## 消防幫浦功能

將水從水源（蓄水池）加壓送至各樓層消防栓或撒水頭，維持足夠壓力與水量。

## 出水量規定

| 設備類型 | 最低出水量 |
|---------|----------|
| 室內消防栓（第一種×2）| ≥ 260 L/min |
| 室內消防栓（第二種×2）| ≥ 120 L/min |
| 自動撒水（第一種）| ≥ 480 L/min（60×8）|
| 自動撒水（第二種）| ≥ 1,200 L/min（80×15）|

## 全揚程計算

`H = h₁ + h₂ + h₃ + h₄`

| 項目 | 說明 |
|------|------|
| h₁ | 落差（幫浦至最高或最遠消防栓的垂直距離）|
| h₂ | 配管摩擦損失水頭 |
| h₃ | 消防栓（或撒水頭）放水壓力換算水頭 |
| h₄ | 其他損失（閥類、接頭等）|

## 啟動方式

| 方式 | 說明 |
|------|------|
| 自動啟動 | 配管壓力下降 → 壓力開關動作 → 幫浦自動啟動 |
| 遠隔啟動 | 消防栓箱旁按鈕（啟動表示燈）→ 啟動幫浦 |
| 手動啟動 | 在幫浦室直接操作 |

## 呼水裝置

- 幫浦位置**低於水源**時：不需呼水裝置（自然引水）
- 幫浦位置**高於水源**時：需設呼水裝置（防止空轉）
- 呼水槽容量 ≥ **100 L**

## 性能試驗

| 測試項目 | 規定 |
|---------|------|
| 額定出水量時 | 壓力 ≥ 額定全揚程 |
| 出水量150%時 | 壓力 ≥ 額定全揚程的 **65%** |
| 締切運轉（關閉出水）| 壓力 ≤ 額定全揚程的 **140%** |

## 備用電源

- 消防幫浦須設**緊急電源**
- 緊急發電機或蓄電池
- 可運轉 **30分鐘** 以上

## 記憶口訣

> **「幫浦四揚程：落差+摩擦+放壓+損失」**
>
> 150%出水量→65%揚程，締切→140%揚程
> 呼水槽100L，緊急電源30分鐘
