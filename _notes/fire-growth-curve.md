## 火災四階段

```
     熱釋放率(HRR)
      ↑
      │        ┌──最盛期──┐
      │       /            \
      │      / 閃燃         \
      │     /                \
      │    / 成長期    衰退期  \
      │   /                    \
      │──/ 初期                  \──
      └─────────────────────────────→ 時間
```

| 階段 | 特徵 | 燃燒控制 |
|------|------|---------|
| 初期 | 小火源、局部燃燒 | 燃料控制 |
| 成長期 | 火勢擴大、溫度上升 | 燃料控制 |
| **閃燃** | 全面燃燒的轉折點 | — |
| 最盛期 | 最高溫度、最大HRR | **通風控制** |
| 衰退期 | 可燃物耗盡、溫度下降 | 燃料控制 |

## 熱釋放率 HRR

HRR（Heat Release Rate）= 單位時間釋放的熱量，單位 **kW**。

- 是描述火災嚴重度的最重要參數
- 決定火災成長速度和最大規模

## t²火災模型

成長期的HRR可近似為時間的平方函數：

`Q = α × t²`

| 參數 | 說明 |
|------|------|
| Q | 熱釋放率 (kW) |
| α | 成長係數 (kW/s²) |
| t | 時間 (s) |

## 四種成長速度

| 成長速度 | α (kW/s²) | 達到1 MW的時間 | 代表場景 |
|---------|-----------|-------------|---------|
| **超快速** | 0.1876 | **75 秒** | 易燃液體池火、堆積紙箱 |
| **快速** | 0.0469 | **150 秒** | 塑膠家具、聖誕樹 |
| **中速** | 0.0117 | **300 秒** | 一般家具、辦公室 |
| **慢速** | 0.0029 | **600 秒** | 厚實木家具 |

## NFPA標準

- NFPA 72（火警系統）：用t²火災模型設計探測器
- NFPA 92（排煙）：用HRR計算排煙量
- 設計火災時常參考α值選擇

## 常考重點

- α值越大 → 成長越快 → 越危險
- 超快速75秒達1MW vs 慢速600秒達1MW → 差8倍
- 閃燃發生在成長期末端
- 最盛期的HRR取決於通風條件

## 記憶口訣

> **「超75快150中300慢600」**
>
> Q = αt²，α越大長越快
> 超快速：易燃液體(0.19)
> 慢速：厚木家具(0.003)
