## 緊急電源種類

| 種類 | 特點 | 適用 |
|------|------|------|
| **蓄電池** | 立即供電、免啟動 | 火警設備、照明 |
| **緊急發電機** | 大容量、長時間 | 消防幫浦、昇降機 |
| **蓄電池+發電機** | 雙重保障 | 重要場所 |

## 各設備緊急電源持續時間

| 設備 | 電源種類 | 持續時間 |
|------|---------|---------|
| 受信總機 | 蓄電池 | 監視 **10分鐘** + 警報 **10分鐘** |
| 緊急廣播 | 蓄電池/發電機 | ≥ **10 分鐘** |
| 緊急照明 | 蓄電池 | ≥ **30 分鐘** |
| 出口標示燈 | 蓄電池 | ≥ **30 分鐘**（常亮型20分鐘）|
| 排煙設備 | 緊急發電機 | ≥ **30 分鐘** |
| 消防幫浦 | 緊急發電機 | ≥ **30 分鐘** |
| 緊急昇降機 | 緊急發電機 | ≥ **30 分鐘** |
| 瓦斯漏氣警報 | 蓄電池 | ≥ **10 分鐘** |
| 無線電通信 | 蓄電池/發電機 | ≥ **30 分鐘** |

## 蓄電池規定

| 項目 | 規定 |
|------|------|
| 種類 | 鉛蓄電池或密閉型鎳鎘電池 |
| 充電方式 | **自動充電裝置**（浮動充電或定期充電）|
| 容量 | 足以供應規定持續時間 |
| 設置位置 | 受信總機內或專用箱 |
| 溫度 | 避免高溫環境（影響壽命）|
| 檢查 | 定期測量電壓/比重 |

## 充電裝置

| 方式 | 原理 |
|------|------|
| **浮動充電** | 常時微量充電，保持滿電（最常用）|
| 定期充電 | 定時充電至滿 |
| 自動切換 | 停電時自動切換至蓄電池 |

## 緊急發電機

| 項目 | 規定 |
|------|------|
| 燃料 | 柴油為主 |
| 啟動時間 | 停電後 **40秒** 內自動啟動 |
| 容量 | 足以供應所有消防設備同時使用 |
| 設置位置 | 專用發電機室 |
| 燃料儲量 | 足以運轉 **2小時** 以上 |

## 常考重點

- 受信總機：10分鐘監視+10分鐘警報 = **共20分鐘**
- 照明和標示：**30分鐘**
- 幫浦和排煙：需要**發電機**（蓄電池不夠力）
- 發電機40秒內啟動

## 記憶口訣

> **「火警10+10，照明標示30，幫浦排煙靠發電機30」**
>
> 蓄電池=立即供電（火警、照明）
> 發電機=大功率（幫浦、排煙、昇降機）
> 發電機40秒內啟動，燃料撐2小時
