## GHS 易燃液體分級

GHS（Globally Harmonized System）聯合國全球調和制度：

| GHS分級 | 閃火點 | 沸點 | 危險程度 |
|---------|--------|------|---------|
| 第1級 | < **23°C** | ≤ 35°C | 極度易燃 |
| 第2級 | < **23°C** | > 35°C | 高度易燃 |
| 第3級 | **23-60°C** | - | 易燃 |
| 第4級 | **60-93°C** | - | 可燃 |

## 台灣公共危險物品第四類分級

| 分級 | 閃火點 | 代表物質 |
|------|--------|---------|
| 特殊引火物 | <-20°C 或 自燃溫度<100°C | 乙醚、二硫化碳 |
| 第一石油類 | < 21°C | 汽油、丙酮 |
| 醇類 | - | 甲醇、乙醇 |
| 第二石油類 | 21-70°C | 煤油、柴油 |
| 第三石油類 | 70-200°C | 重油 |
| 第四石油類 | 200-250°C | 潤滑油 |

## GHS vs 台灣對照

| GHS | 對應台灣分級 |
|-----|------------|
| 第1級(FP<23°C+BP≤35°C) | 特殊引火物、第一石油類 |
| 第2級(FP<23°C+BP>35°C) | 第一石油類 |
| 第3級(23-60°C) | 第二石油類 |
| 第4級(60-93°C) | 第三石油類（部分）|

## GHS標示

| 圖示 | 意義 |
|------|------|
| 火焰符號 🔥 | 易燃性 |
| 紅色菱形框 | 危險警示 |
| 信號語 | 「危險」或「警告」|

## 記憶口訣

> **「GHS用23和60分三級」**
>
> GHS：<23°C(1&2級)→23-60°C(3級)→60-93°C(4級)
> 台灣：21°C和70°C是主要分界
