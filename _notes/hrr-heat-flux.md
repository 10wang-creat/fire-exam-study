## 熱釋放率 HRR

### 定義
單位時間內火源釋放的**總熱量**，單位 **kW**。

`HRR = ṁ × ΔHc × χ`

| 參數 | 說明 |
|------|------|
| ṁ | 質量燃燒速率 (kg/s) |
| ΔHc | 燃燒熱 (kJ/kg) |
| χ | 燃燒效率 (0-1) |

### 典型火源HRR

| 火源 | HRR (kW) |
|------|----------|
| 蠟燭 | 0.08 |
| 廢紙簍 | 4-18 |
| 聖誕樹 | 500-650 |
| 沙發 | 1,000-3,000 |
| 房間全面燃燒 | 2,000-8,000 |
| 閃燃臨界 | 約 **1,000-2,000** |

## 熱通量 Heat Flux

### 定義
單位**面積**接收到的熱功率，單位 **kW/m²**。

### 危險判定基準

| 熱通量 (kW/m²) | 效果 |
|----------------|------|
| 1 | 長時間曝曬會不適 |
| 2.5 | **避難安全界限**（不可超過）|
| 4 | 皮膚30秒開始燒傷 |
| 12.5 | 木材引燃 |
| **20** | **閃燃發生基準**（地板熱通量）|
| 25 | 鋼結構開始弱化 |

### 重要判定值

| 值 | 用途 |
|----|------|
| **2.5 kW/m²** | 人員避難安全上限 |
| **20 kW/m²** | 閃燃發生條件（地板面） |

## 單位換算

| 換算 | 數值 |
|------|------|
| 1 kW | = 1 kJ/s |
| 1 kcal | = 4.186 kJ |
| 1 BTU | = 1.055 kJ |
| 1 MW | = 1,000 kW |
| 1 W/m² | = 0.001 kW/m² |

## 與其他概念的關係

| 概念 | 關係 |
|------|------|
| HRR與閃燃 | HRR達到臨界值→閃燃 |
| 熱通量與避難 | >2.5 kW/m² → 不安全 |
| 熱通量與閃燃 | 地板面≥20 kW/m² → 閃燃 |
| HRR與排煙設計 | 排煙量依HRR計算 |

## 記憶口訣

> **「2.5避難上限，20閃燃地板」**
>
> HRR = 質量燃燒率 × 燃燒熱 × 效率
> 沙發燒1000-3000kW，閃燃約需1000-2000kW
