## 定義

建築物之各樓層供人員進出或排煙採光的**有效開口面積**未達規定標準者，視為無開口樓層。

## 有效開口條件（須同時滿足）

| 條件 | 規定 |
|------|------|
| 開口面積 | 面積 ≥ **0.5 m²** 以上 |
| 開口高度 | 下端距地面 ≤ **1.2m** |
| 開口寬度 | ≥ **75 cm** |
| 可開啟 | 可自外面開啟或易於破壞 |
| 無格柵 | 未設有格柵或固定式鐵窗 |

> 任一條件不符 → 該開口不算「有效開口」

## 判定標準

**有效開口面積合計** < 該樓層樓地板面積 × **1/30** → **無開口樓層**

### 計算範例
某樓層面積 300m²：
- 需要有效開口 = 300 × 1/30 = **10 m²**
- 如果該層有效開口 < 10 m² → 無開口樓層

## 為什麼重要？

無開口樓層比一般樓層**更危險**：
- 排煙困難（無自然通風）
- 消防搶救進入困難
- 熱累積快

## 加嚴設備要求

| 設備 | 無開口樓層加嚴 |
|------|-------------|
| 火警探測器 | 必須設置（不得免設）|
| 受信總機 | 必須使用**非蓄積**模式 |
| 排煙設備 | 必須設置 |
| 自動撒水 | 面積門檻降低 |
| 緊急照明 | 必須設置 |

## 常見無開口樓層

- 地下室（無外牆開口）
- 裝設固定式鐵窗的樓層
- 大型密閉商場/倉庫
- 無外牆的內部樓層

## 記憶口訣

> **「有效開口 < 面積的1/30 → 無開口樓層」**
>
> 有效開口條件：≥0.5m²、下端≤1.2m、寬≥75cm、可開、無格柵
> 無開口 = 加嚴對待（非蓄積+排煙+撒水+照明）
