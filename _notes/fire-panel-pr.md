## P型 vs R型 比較

| 項目 | P型（共通線式） | R型（固有信號式） |
|------|---------------|-----------------|
| **信號方式** | 共通信號（只知哪個迴路）| 固有信號（知道哪個探測器）|
| **配線** | 每迴路一對線 | 共用幹線+中繼器 |
| **定位精度** | 只能定位到**迴路（區域）** | 可定位到**個別探測器** |
| **迴路數** | P1: ≤7 / P2: ≤2 | 無限制 |
| **適用規模** | 中小型建築 | 大型、高層建築 |
| **成本** | 較低 | 較高 |
| **維修** | 較簡單 | 需專業技術 |
| **配線量** | 較多（每迴路獨立）| 較少（共用幹線）|

## P型受信總機

### P型第一種
- 迴路數 ≤ **7** 回路
- 有火災燈、地區燈、主音響
- 適用：中型建築

### P型第二種
- 迴路數 ≤ **2** 回路
- 簡易型，適用小型場所

## R型受信總機

- 透過**中繼器**（副受信總機）收集各探測器信號
- 以數位編碼方式傳輸 → 可辨識個別探測器
- 適合大規模建築（醫院、百貨、高層建築）
- 可結合大樓自動化管理系統（BAS）

## 受信總機基本功能

| 功能 | 說明 |
|------|------|
| 火災表示 | 顯示哪個區域/探測器動作 |
| 主音響裝置 | 發出警報音 |
| 地區音響裝置 | 該區域發出警報 |
| 電話連絡 | 與各樓層通話 |
| 電源監視 | 監視主電源及蓄電池狀態 |
| 回路斷線監視 | 偵測配線是否斷線 |

## 設置位置

- **守衛室**或**管理室**等有人常駐處
- 便於操作、監視之位置
- 不受日光直射或潮濕影響

## 記憶口訣

> **「P型看迴路，R型看個體」**
>
> P = 便宜簡單，定位到區域（P1≤7迴路，P2≤2迴路）
> R = 精準昂貴，定位到探測器，用中繼器
