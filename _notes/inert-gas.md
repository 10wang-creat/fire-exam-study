## 惰性氣體種類

| 藥劑 | 組成 | 商品名 |
|------|------|--------|
| **IG-541** | 52%N₂ + 40%Ar + 8%CO₂ | Inergen |
| IG-55 | 50%N₂ + 50%Ar | Argonite |
| IG-100 | 100%N₂ | - |
| IG-01 | 100%Ar | - |

## 滅火原理

- 釋放惰性氣體，將防護區域氧濃度**降至12-14%**
- 氧濃度<15% → 燃燒無法持續
- 純物理性窒息，**無化學反應**

## IG-541的特殊設計

8% CO₂ 的作用：
- 人在低氧環境中呼吸減緩
- CO₂刺激呼吸中樞 → **加快呼吸**
- 即使氧濃度降低，仍能攝取足夠氧氣存活
- 提高人員安全性（有一定時間可避難）

## 設計參數

| 項目 | IG-541 |
|------|--------|
| 設計濃度 | 約 **37-43%**（將O₂降至12%）|
| 放射時間 | ≤ **60秒**（或120秒視規定）|
| 儲存壓力 | 150-300 bar |
| 儲存方式 | 高壓鋼瓶 |

## 安全措施

| 措施 | 說明 |
|------|------|
| 延遲放射 | 放射前 **30秒** 以上警報 |
| 人員偵測 | 確認無人後才放射 |
| 排氣裝置 | 放射後排除過量氣體 |
| 壓力洩放口 | 防止室內正壓過大 |

## vs CO₂滅火

| 項目 | 惰性氣體(IG-541) | CO₂ |
|------|-----------------|-----|
| 毒性 | 低（含CO₂助呼吸）| **高**（窒息危險大）|
| 環保性 | **ODP=0, GWP=0** | GWP=1 |
| 人員安全 | 相對安全 | 危險（需確認無人）|
| 殘留 | 無 | 無 |
| 成本 | 高（鋼瓶多）| 較低 |

## 記憶口訣

> **「IG-541 = 52氮+40氬+8碳，碳讓你喘氣逃命」**
>
> ODP=0, GWP=0，最環保的滅火藥劑
> 降O₂到12-14%→火熄人活
