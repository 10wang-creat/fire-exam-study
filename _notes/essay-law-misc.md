## 一、無開口樓層 vs 無窗戶居室（103、106年）

### 無開口樓層（設置標準）

**定義**：建築物之各樓層供居室使用之部分，具可內切直徑 **50 cm** 以上圓孔之開口，合計面積未達該樓地板面積 **1/30** 以上者。（11F以上為 1/30）

| 開口條件 | 數值 |
|---------|------|
| 開口可內切圓直徑 | ≥ **50** cm |
| 開口面積合計 | < 樓地板面積 × **1/30** |

→ 無開口樓層須**加設**：室內消防栓、自動撒水、火警自動警報（依面積門檻）

### 無窗戶居室（建築技術規則）

| 條件（符合任一） | 判定為無窗戶居室 |
|----------------|---------------|
| 有效採光面積 < 樓地板面積 **1/10** | ✓ |
| 防火避難開口高 < **120** cm 或 寬 < **75** cm | ✓ |
| 圓形開口直徑 < **100** cm | ✓ |
| 面積 >50 m² 且有效通風面積 < 樓地板面積 **2%** | ✓ |

### 差別

| 比較 | 無開口樓層 | 無窗戶居室 |
|------|----------|----------|
| 法源 | 設置標準 | 建築技術規則 |
| 對象 | **整個樓層** | **個別居室** |
| 開口大小 | ≥50cm圓 | ≥120×75cm 或 ≥100cm圓 |
| 面積比 | <1/30 | <1/10（採光）或 <2%（通風）|
| 效果 | 加設消防設備 | 加設排煙/照明等 |

## 二、消防設備人員（105、113年）

### 資格取得

| 人員 | 資格 |
|------|------|
| **消防設備師** | 通過高等考試 |
| **消防設備士** | 通過普通考試 |

### 業務分工

| 工作 | 設備師 | 設備士 |
|------|--------|--------|
| **設計** | ✓ | ✗ |
| **監造** | ✓ | ✗ |
| **裝置** | ✓ | ✓ |
| **檢修** | ✓ | ✓ |

### 定期講習（消防設備師及消防設備士管理辦法）

- 自取得證書日起每 **3年** 應接受講習 **1次**
- 或取得累計積分達 **160分** 以上之訓練證明
- 業務登記簿至少保存 **5年**

### 不得充任情形

1. 受禁治產宣告尚未撤銷
2. 受1年以上有期徒刑確定（緩刑除外）
3. 經撤銷或廢止證書未滿3年
4. 受停業處分期間

## 三、圖說審查與竣工查驗（105、107年）

### 審查程序

```
起造人（或消防設備設計人）
  ↓ 檢附消防圖說
消防機關受理
  ↓ 7~10日內結案（複雜者≤20日）
審查結果
  ├─ 合格 → 核發審查合格通知
  └─ 不合格 → 一次告知補正 → 限期補正
```

### 竣工查驗程序

```
起造人
  ↓ 檢附：測試報告書+施工照片+原審圖說
消防機關排定日期
  ↓ 通知起造人+裝置人
裝置人攜帶資格證件會同查驗
  ↓
合格 → 核發合格通知
不合格 → 列舉不合規定項目 → 改善 → 複查
```

### 監造人因應

- 施工中須經**試驗或勘驗**事項之查核
- 製作紀錄（拍照建檔）
- 確認與審訖圖說一致

## 四、罰則——逕行舉發（106年）

### 不須限期改善，應逕行舉發之情形

| 違規態樣 | 罰鍰 |
|---------|------|
| 消防設備師/士**檢修不實** | **2萬~10萬**，按次處罰 |
| 未依規定從事設計/監造/裝置/檢修 | 1萬~5萬 |
| 規避/妨礙/拒絕**檢查** | 1萬~5萬，按次+強制執行 |

### 須經限期改善之情形

| 違規態樣 | 處理流程 |
|---------|---------|
| 未設置/維護消防設備 | 限期改善→屆期未改善→**6萬~30萬** |
| 未使用防焰物品 | 限期改善→屆期未改善→1萬~5萬 |
| 未辦理檢修申報 | 限期改善→屆期未改善→1萬~5萬 |

## 五、防火管理（113年）

### 防火管理人

| 項目 | 規定 |
|------|------|
| 遴用條件 | 管理或監督層次人員 |
| 初訓 | ≥ **12** 小時 |
| 複訓 | 每 **2年** 至少 1次，≥ **6** 小時 |
| 異動 | 報消防機關備查 |

### 消防防護計畫

- 自衛消防編組：10人↑→滅火班+通報班+避難引導班；50人↑→增安全防護班+救護班
- 防火避難設施自行檢查：每**月**至少 1次
- 滅火通報避難訓練：每**半年**至少 1次，每次≥**4小時**

### 供營業使用場所認定

依消防法施行細則定義，包含營利性質之場所（餐廳、KTV、商場等），須遴用防火管理人。

## 六、增建改建變更用途（114年）

### 原則

增建、改建或變更用途時，消防安全設備應適用**增改建後之標準**。

### 適用增改建後標準之設備

- 火警自動警報設備
- 緊急廣播設備
- 標示設備
- 緊急照明設備
- 避難器具
- 排煙設備等**警報避難設備**

### 原有合法建築物（104年）

- 原有合法建築物=取得使用執照時合法，但**現行法規已變更**
- 依消防法第6條：應依**現行標準**改善
- 但得依「原有合法建築物消防安全設備改善計畫」分階段改善

## 七、防災中心（109年）

### 設置條件

**高層建築物**（建築技術規則：高度 >50m 或 ≥16F）應設防災中心

### 規定

| 項目 | 規定 |
|------|------|
| 位置 | 消防人員自外面**容易進出** |
| 位置 | 便於通達緊急昇降機間及特別安全梯 |
| 出入口至屋外 | 步行距離 ≤ **30** m |
| 樓地板面積 | ≥ **40** m² |
| 空調 | 冷暖換氣為**專用** |
| 防火構造 | 牆壁/樓板以防火構造區劃 |
| 設備 | 受信總機、緊急廣播、排煙控制、監視設備 |

## 八、第五種滅火設備滅火效能值（111年）

| 設備 | 核算方式 |
|------|---------|
| 滅火器 | 依型式認可之滅火效能值 |
| 消防專用水桶 6L | 每 **3** 個 = 1 滅火效能值 |
| 水槽 | 每 **80** L = 1.5 滅火效能值 |
| 乾燥砂 | 每 **50** L = 0.5 滅火效能值 |
| 膨脹蛭石/珍珠岩 | 每 **160** L = 1 滅火效能值 |
