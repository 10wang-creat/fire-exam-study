## 消防法重要條文速查

| 條文 | 核心內容 | 關鍵字 |
|------|---------|--------|
| 第1條 | 立法目的：預防火災、搶救災害、緊急救護 | 三大目的 |
| 第6條 | 消防設備之設置依**設置標準** | 設置標準 |
| **第7條** | 消防設備之設計由**消防設備師**為之，裝置、檢修由**設備師或設備士**為之 | 師/士業務 |
| **第9條** | 消防設備應定期**檢修申報** | 檢修申報 |
| **第10條** | 供公眾使用建築物圖說應經消防機關**審查** | 圖說審查 |
| **第11條** | 指定場所應使用**防焰物品** | 防焰 |
| **第13條** | 一定規模場所應由管理權人遴用**防火管理人** | 防火管理 |
| 第14條 | 田野引火燃燒、施放天燈等管制 | 引火管制 |
| **第15條** | **公共危險物品**之管理 | 危險物品 |
| 第15-1條 | **液化石油氣**之管理 | LPG |
| 第38條 | 消防設備師/士違規之懲處 | 懲處 |

## 罰則整理

| 違規行為 | 罰鍰 |
|---------|------|
| 未依規定設置消防設備 | **6-30萬** |
| 未檢修申報 | 1-5萬 |
| 未改善（經通知後）| 2-10萬，連續處罰 |
| 未使用防焰物品 | 1-5萬 |
| 未遴用防火管理人 | 1-5萬 |
| 拒絕/妨礙消防檢查 | 3千-1.5萬 |
| 謊報火警 | 3千-1.5萬 |

## 消防法 vs 設置標準 vs 管理辦法

| 法規層級 | 名稱 | 性質 |
|---------|------|------|
| 法律 | 消防法 | 母法 |
| 法規命令 | 消防法施行細則 | 細節規定 |
| 法規命令 | 各類場所消防安全設備設置標準 | 設備規格 |
| 法規命令 | 公共危險物品管理辦法 | 危險物品 |
| 法規命令 | 消防設備師及設備士管理辦法 | 人員管理 |

## 記憶口訣

> **「6設7師9檢10審11焰13管15危」**
>
> 第6→設置標準、第7→師士、第9→檢修申報
> 第10→圖說審查、第11→防焰、第13→防火管理
> 第15→公共危險物品
