## 一、應設置場所

### 地下層使用瓦斯場所

| 場所條件 | 面積要求 |
|---------|---------|
| 甲類場所之地下層 | 樓地板面積合計 ≥ **1,000** m² 且甲類 ≥ **500** m² |
| 溫泉浴室等特定場所 | 樓地板面積合計 ≥ **500** m² |
| 地下建築物（使用瓦斯） | **全部** |

### 不需設置條件

- 不使用瓦斯之場所
- 非屬地下層之場所（地上層免設）

## 二、檢知器裝設位置（核心必背）

```
┌─────────────────────────────────────────┐
│                天花板                     │
│  ┌──────────────────────┐                │
│  │ 天然氣（比重<1）↑上浮 │                │
│  │ 檢知器下端：天花板下方  │                │
│  │     ≤ 30 cm 範圍內     │                │
│  └──────────────────────┘                │
│                                          │
│                                          │
│                                          │
│  ┌──────────────────────┐                │
│  │ 液化石油氣(比重>1)↓下沉│               │
│  │ 檢知器上端：距樓地板面  │                │
│  │     ≤ 30 cm 範圍內     │                │
│  └──────────────────────┘                │
│                樓地板                     │
└─────────────────────────────────────────┘
```

| 瓦斯種類 | 對空氣比重 | 檢知器位置 | 水平距離 |
|---------|----------|----------|---------|
| **天然氣**（甲烷） | < 1（上浮） | **下端**在天花板下方 **30** cm內 | 距瓦斯器具 ≤ **8** m |
| **液化石油氣**（丙烷/丁烷） | > 1（下沉） | **上端**在距樓地板面 **30** cm內 | 距瓦斯器具 ≤ **4** m |

### 水平距離起算點

- 以瓦斯**燃燒器**為中心點
- 或以瓦斯導管**貫穿牆壁處**起算

### 有樑區隔時

檢知器設於距瓦斯器具水平距離內，且於該距離內**無淨高 60 cm 以上之樑**區隔之吸氣口 **1.5** m 範圍內

## 四、加瓦斯試驗器

| 瓦斯比重 | 試驗用氣體 |
|---------|----------|
| < 1（天然氣） | **甲烷** |
| > 1（液化石油氣） | **異丁烷** |

> 瓦斯濃度測試值一般取燃燒下限之 **1/4** 值

---

## 五、受信機性能檢查（103年考題）

### 檢查方法

| 項目 | 方法 |
|------|------|
| 火災表示 | 使檢知器動作，確認瓦斯漏氣燈亮 |
| 音響裝置 | 確認主音響正常鳴動 |
| 切換裝置 | 遮斷常用電源，確認自動切換至緊急電源 |
| 充電裝置 | 目視確認無變形/腐蝕/發熱 |
| 端子電壓 | 操作電源試驗開關，由電壓表確認 |
| 結線接續 | 目視或螺絲起子確認無斷線/端子鬆動 |
| 通話裝置 | 相互通話確認（設有時） |

### 判定方法

- 瓦斯漏氣燈、主音響、分區表示應正常
- 切換至緊急電源後應正常動作
- 緊急電源容量 ≥ **10** 分鐘有效動作

---

## 六、綜合檢查（112、114年考題）

### 檢查方法

1. 使用**加瓦斯試驗器**對選取之檢知器加瓦斯
2. 確認從檢知器動作到受信總機表示之**時間**
3. 確認中繼器、瓦斯漏氣表示燈、檢知區域警報裝置之連動

### 動作時間判定

| 確認方式 | 無中繼器 | 有中繼器 |
|---------|---------|---------|
| 由檢知器動作確認燈 → 瓦斯漏氣燈 | ≤ **60** 秒 | ≤ **65** 秒 |
| 由檢知區域警報/中繼器確認燈 → 漏氣燈 | ≤ **80** 秒 | ≤ **85** 秒 |

### 警報音壓

| 裝置 | 距離 | 音壓 |
|------|------|------|
| 檢知區域警報裝置 | 距 1m | ≥ **70** dB |
| 瓦斯漏氣表示燈 | 前方 3m | 能明確識別 |
| 表示燈距樓地板高度 | — | ≤ **4.5** m |
