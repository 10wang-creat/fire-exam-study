## 系統組成

```
火警探測器 ──┐
             ├──→ 受信總機 ──→ 地區音響(警鈴)
手動報警機 ──┘        │
                      ├──→ 緊急廣播設備
                      ├──→ 119火災通報裝置
                      ├──→ 防火門/捲門關閉
                      ├──→ 排煙設備啟動
                      ├──→ 空調停止
                      └──→ 緊急照明啟動
```

## 各組件功能

| 組件 | 功能 | 類型 |
|------|------|------|
| 火警探測器 | 自動偵測火災 | 差動/定溫/偵煙/火焰 |
| 手動報警機 | 人員手動啟動警報 | 按壓式 |
| 受信總機 | 接收+顯示+控制中樞 | P型/R型 |
| 地區音響裝置 | 發出警報音 | 警鈴/蜂鳴器 |
| 中繼器 | R型系統的信號中繼 | 副受信總機 |
| 緊急廣播 | 語音引導避難 | 揚聲器 |

## 設置場所

| 條件 | 設備 |
|------|------|
| 甲類場所≥300m² | 火警自動警報設備 |
| 11層以上 | +緊急廣播+119通報 |
| 地下建築物 | +緊急廣播+119通報 |

## 連動控制（重要！）

受信總機確認火警後，自動連動：

| 連動設備 | 動作 |
|---------|------|
| 防火門/捲門 | **關閉** |
| 排煙設備 | **啟動** |
| 空調系統 | **停止** |
| 緊急照明 | 啟動（停電時）|
| 緊急廣播 | 播放警報 |
| 消防幫浦 | 待命啟動 |

## 蓄電池容量

| 設備 | 備用電源持續時間 |
|------|---------------|
| 受信總機 | ≥ **10 分鐘** 監視 + **10 分鐘** 警報 |
| 緊急廣播 | ≥ 10 分鐘 |
| 緊急照明 | ≥ 30 分鐘 |

## 記憶口訣

> **「探測+手動 → 受信總機 → 音響+廣播+119+連動」**
>
> 連動三關閉：防火門關、排煙開、空調停
> 受信總機是整個系統的「大腦」
