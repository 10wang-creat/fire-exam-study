## 三個溫度定義

| 名詞 | 英文 | 定義 | 需要點火源？ |
|------|------|------|-----------|
| 閃火點 | Flash Point | 可燃液體蒸氣遇火源**瞬間閃燃**的最低溫度 | ✅ 需要 |
| 燃點（著火點）| Fire Point | 可燃液體蒸氣遇火源能**持續燃燒5秒以上**的最低溫度 | ✅ 需要 |
| 發火點（自燃溫度）| Auto-ignition Temp | **不需要**外部火源，自行著火的最低溫度 | ❌ 不需要 |

## 溫度高低關係

`閃火點 < 燃點 < 發火點（自燃溫度）`

- 閃火點最低（只閃一下就熄）
- 燃點稍高（能持續燃燒）
- 發火點最高（不需火源就能燃）

## 閃火點的重要性

- 閃火點是判斷液體**危險性**的主要指標
- 閃火點越低 → 越危險
- 公共危險物品第四類（易燃液體）的分級就是依**閃火點**

### 第四類易燃液體分級

| 分級 | 閃火點 | 代表物質 |
|------|--------|---------|
| 特殊引火物 | < -20°C（或自燃溫度<100°C）| 乙醚(-45°C)、二硫化碳 |
| 第一石油類 | < 21°C | 汽油(-43°C)、丙酮(-20°C) |
| 醇類 | - | 甲醇(11°C)、乙醇(13°C) |
| 第二石油類 | 21-70°C | 煤油(43°C)、柴油(52°C) |
| 第三石油類 | 70-200°C | 重油(70°C) |
| 第四石油類 | 200-250°C | 潤滑油、齒輪油 |

## 測試方法

| 方法 | 適用 |
|------|------|
| 密閉杯法（Pensky-Martens） | 閃火點 > 80°C |
| 密閉杯法（Tag） | 閃火點 < 80°C |
| 開放杯法（Cleveland） | 閃火點 > 80°C |

## 常考陷阱

- 閃火點≠發火點：閃火點需要火源，發火點不需要
- 汽油閃火點很低(-43°C)但發火點很高(約300°C)
- 碳的發火點比紙低？→ 錯！紙的發火點約230°C，碳約400°C

## 記憶口訣

> **「閃一下就熄，燒5秒才算燃，不點自己著是發火」**
>
> 閃火點 < 燃點 < 發火點
> 汽油閃火點-43°C → 特別危險
