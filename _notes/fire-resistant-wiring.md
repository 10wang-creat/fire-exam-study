## 為什麼需要耐燃耐熱配線？

火災時溫度極高，一般電線會被燒斷 → 消防設備失效。消防設備的配線必須在火災中維持一定時間的功能。

## 耐燃 vs 耐熱

| 項目 | 耐燃配線 | 耐熱配線 |
|------|---------|---------|
| 耐火時間 | ≥ **15 分鐘**（750°C）| ≥ **30 分鐘**（840°C）|
| 要求程度 | 較低 | **較高** |
| 適用設備 | 一般消防設備 | 關鍵消防設備 |
| 成本 | 較低 | 較高 |

## 設備配線要求

### 須使用**耐熱**配線的設備（高要求）

| 設備 | 原因 |
|------|------|
| 緊急廣播設備 | 火災中須持續運作引導避難 |
| 排煙設備 | 火災中須排煙 |
| 緊急照明設備 | 停電+火災時照明 |
| 消防幫浦 | 持續供水滅火 |
| 緊急昇降機 | 消防員搶救使用 |

### 須使用**耐燃**配線的設備（基本要求）

| 設備 |
|------|
| 火警自動警報設備 |
| 手動報警機 |
| 瓦斯漏氣警報 |
| 自動撒水啟動裝置 |

## 配線施工

| 方式 | 說明 |
|------|------|
| 金屬導線管 | 最常見，管內穿線 |
| MI電纜 | 礦物絕緣電纜，耐火性最佳 |
| 耐火電纜 | 專用耐火被覆 |
| 金屬線槽 | 大量配線用 |

## 記憶口訣

> **「耐熱30分>耐燃15分」**
>
> 火災中要持續運作的設備→耐熱（廣播、排煙、照明、幫浦）
> 只需初期偵測通報的設備→耐燃（火警、手動、瓦斯）
