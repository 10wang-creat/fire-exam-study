## CO₂滅火設備藥劑量

### 全區放射

`W = V × K + A_o × 10`

| 參數 | 說明 |
|------|------|
| W | 藥劑量 (kg) |
| V | 防護區域容積 (m³) |
| K | 基本藥劑量 (kg/m³) |
| A_o | 開口部面積 (m²) |
| 10 | 開口部補正係數 (kg/m²) |

### K值

| 防護對象 | K (kg/m³) |
|---------|----------|
| 電氣設備 | **1.0** |
| 可燃液體 | **1.0** |
| 棉花、木材 | 1.3 |
| 其他 | **0.8** |

### 局部放射
`W = Q × t`
- Q = 放射率 (kg/min)
- t = 放射時間 (min)

## 乾粉滅火設備藥劑量

### 全區放射

`W = V × K`

| 防護對象 | K (kg/m³) |
|---------|----------|
| 第一種乾粉(NaHCO₃) | 0.60 |
| 第二種乾粉(KHCO₃) | 0.36 |
| **第三種乾粉(NH₄H₂PO₄)** | **0.36** |
| 第四種乾粉 | 0.24 |

### 局部放射
- 防護面積每m² ≥ 一定量

## 潔淨藥劑（FM-200）

### 全區放射

`W = V × C / (100 - C) × S`

| 參數 | 說明 |
|------|------|
| V | 防護區域容積 (m³) |
| C | 設計濃度 (%) |
| S | 藥劑比容 (m³/kg) |

### FM-200設計濃度

| 防護對象 | 設計濃度 |
|---------|---------|
| A類（表面火災）| 7.0% |
| B類（可燃液體）| 依物質 |
| C類（電氣設備）| 7.0% |

## IG-541

`W = V × C / (100 - C) × (1/S)`

- 設計濃度約 37-43%
- S = 比容（依溫度）

## 開口部補正（重要！）

有開口部時，藥劑會從開口洩漏，需加算補正：

| 系統 | 開口部補正 |
|------|----------|
| CO₂ | +A_o × **10 kg/m²** |
| 乾粉 | +A_o × 相應補正值 |
| 潔淨藥劑 | 開口部應在放射前**自動關閉** |

> 潔淨藥劑通常要求開口部自動關閉，若無法關閉需加算洩漏量

## 放射時間

| 系統 | 放射時間 |
|------|---------|
| CO₂（電氣） | ≤ **1 分鐘** |
| CO₂（其他） | ≤ **3.5 分鐘** |
| 乾粉 | ≤ **30 秒** |
| FM-200 | ≤ **10 秒** |
| IG-541 | ≤ **60 秒** |

## 計算範例

CO₂全區放射：電氣室 5m×4m×3m，開口 1m²
- V = 60 m³
- W = 60 × 1.0 + 1 × 10 = **70 kg**
- 需要鋼瓶：70 ÷ 45 ≈ 2支（45kg/支）

## 記憶口訣

> **「CO₂藥劑 = 容積×K + 開口×10」**
>
> CO₂電氣K=1.0，放射1分鐘
> 乾粉30秒放完，FM-200最快10秒
> 開口部一定要補正！
