## 差動式探測器家族

| 型式 | 適用 | 特點 |
|------|------|------|
| 差動式局限型 | 一般房間 | 最常見 |
| **補償式局限型** | 溫度緩慢變化場所 | 差動+定溫雙功能 |
| **空氣管式** | **大空間** | 管路佈設涵蓋大面積 |

## 補償式探測器

### 原理
結合**差動式**和**定溫式**的雙重功能：
- 溫度**快速**上升 → 差動式機制動作
- 溫度**緩慢**上升到設定值 → 定溫式機制動作

### 優點
- 解決差動式在緩慢升溫場所可能不動作的問題
- 兼具兩種探測器的優點

### 適用場所
- 溫度可能緩慢升高的場所
- 需要較高可靠性的場所
- 暖氣房等

## 空氣管式探測器

### 原理
在天花板下佈設**細金屬管（空氣管）**，管內空氣受熱膨脹 → 推動接點動作。

### 特點

| 項目 | 規定 |
|------|------|
| 管路總長 | 每一迴路 ≤ **100m** |
| 佈設方式 | 蛇行狀鋪設於天花板下 |
| 管路間距 | ≤ **6m**（相鄰管路中心間距）|
| 離牆距離 | ≤ **3m** |
| 適用高度 | ≤ **15m** |
| 適用面積 | 大空間（倉庫、工廠、展示廳）|

### 為什麼適合大空間？
- 一根管路可以涵蓋大面積
- 比裝數十個局限型探測器更經濟
- 類似「線型」偵測概念

## 探測器型式選擇總整理

| 場所 | 建議型式 | 原因 |
|------|---------|------|
| 一般居室 | 偵煙式 | 早期偵測 |
| 廚房 | 定溫式 | 避免差動式誤報 |
| 大空間(<15m) | **空氣管式** | 經濟涵蓋大面積 |
| 大空間(15-20m) | 光電分離型 | 線型偵煙 |
| 大空間(>20m) | 火焰式 | 煙熱到不了天花板 |
| 溫度緩變場所 | **補償式** | 差動+定溫雙保險 |

## 記憶口訣

> **「補償 = 差動+定溫雙重保險」**
> **「空氣管 = 大空間專用，管路≤100m，間距≤6m」**
>
> 補償式解決緩慢升溫問題
> 空氣管式解決大面積偵測問題
