## 定義

陰燃（Smoldering）：可燃固體在**無可見火焰**的情況下，表面或內部緩慢氧化燃燒的現象。又稱**悶燒、無焰燃燒**。

## 特性

| 特性 | 說明 |
|------|------|
| 無火焰 | 只有灼熱發光，無可見火焰 |
| 溫度較低 | 約 300-600°C（比有焰燃燒低）|
| 速度慢 | 燃燒傳播速度極慢 |
| 持續時間 | 可能持續**數小時到數天** |
| 難以發現 | 無火焰、煙量初期少 |
| 產物 | 大量**CO**和有毒氣體 |

## 常見陰燃物質

| 物質 | 場景 |
|------|------|
| 棉花、棉被 | 香菸引燃→陰燃數小時 |
| 木材深層 | 火場殘火 |
| 泥炭 | 地下泥炭層 |
| 鋸木屑 | 堆積自燃 |
| 聚氨酯泡棉 | 沙發、床墊 |

## 危險性

### 1. 轉為有焰燃燒
陰燃持續 → 溫度逐漸升高 → 突然轉為**有焰燃燒**（flashover前兆之一）

### 2. CO中毒
陰燃是**不完全燃燒** → 產生大量CO → 睡眠中CO中毒

### 3. 火災調查困難
陰燃痕跡不明顯 → 難以判定起火點

## 陰燃 vs 有焰燃燒

| 項目 | 陰燃 | 有焰燃燒 |
|------|------|---------|
| 火焰 | **無** | 有 |
| 溫度 | 300-600°C | 600°C以上 |
| O₂需求 | 少量 | 較多 |
| 速度 | 極慢 | 較快 |
| CO產量 | **高**（危險！）| 較低（完全燃燒產CO₂）|
| 偵測 | 偵煙式有效 | 差動式/偵煙式均可 |

## 消防對策

- 火場殘火處理：確認**深層無陰燃**後才撤離
- 偵煙式探測器對陰燃最有效（產煙→偵測）
- 香菸引火預防：不在床上/沙發上吸菸

## 記憶口訣

> **「陰燃無焰慢慢燒，CO多最危險」**
>
> 香菸引燃棉被→陰燃數小時→CO中毒
> 可能突然轉為有焰燃燒→閃燃
