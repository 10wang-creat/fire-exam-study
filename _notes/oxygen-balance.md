## 氧平衡（Oxygen Balance, O.B.）

### 定義
爆炸性物質分子內的氧含量，與完全氧化所需氧量的差值。

### 公式（CₐHᵦNᵧOᵟ）

`O.B.(%) = -1600/M × (2a + b/2 - d)`

| 參數 | 說明 |
|------|------|
| M | 分子量 |
| a | 碳原子數 |
| b | 氫原子數 |
| d | 氧原子數 |

### O.B.值意義

| O.B. | 意義 | 爆炸效率 |
|------|------|---------|
| O.B. = 0 | 含氧量剛好完全氧化 | **最大爆炸威力** |
| O.B. > 0 | 氧過剩（多餘的氧）| 威力較低 |
| O.B. < 0 | 氧不足（缺氧）| 威力較低，產生CO |

### TNT的O.B.
TNT = C₇H₅N₃O₆（M=227）：
- O.B. = -1600/227 × (2×7 + 5/2 - 6) = -1600/227 × 10.5 = **-74%**
- 嚴重缺氧 → 爆炸時產生大量黑煙

## 硝化度

### 定義
硝化纖維（Nitrocellulose）中含氮量的百分比，表示硝化程度。

### 計算
`硝化度(%) = (硝基中的氮原子質量/分子量) × 100`

### 範例
硝化纖維 C₂₄H₃₂O₁₂(NO₃)₈：
- 氮原子數 = 8
- 氮原子質量 = 8 × 14 = 112
- 分子量 = 24×12 + 32×1 + 12×16 + 8×(14+48) = 288+32+192+496 = 1008
- 硝化度 = 112/1008 × 100 = **11.11%**

### 硝化度與危險性

| 硝化度 | 名稱 | 用途 |
|--------|------|------|
| <12.5% | 弱棉 | 塑膠、漆料 |
| ≥12.5% | 強棉 | 炸藥、火藥 |

## 記憶口訣

> **「O.B.=0最猛，正多氧負缺氧」**
>
> TNT的O.B.=-74%→嚴重缺氧→黑煙
> 硝化度>12.5%=強棉(炸藥用)
