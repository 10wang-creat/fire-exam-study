## 壓力單位換算

| 單位 | 換算關係 |
|------|---------|
| 1 MPa | = **10 kgf/cm²** |
| 1 kgf/cm² | = 0.1 MPa = **98.07 kPa** ≈ 100 kPa |
| 1 atm | = 1.013 kgf/cm² ≈ 0.1013 MPa |
| 1 bar | ≈ 1.02 kgf/cm² |
| 10m 水柱 | = **1 kgf/cm²** = 0.1 MPa |

### 常用記憶

> **10m 水頭 = 1 kgf/cm² = 0.1 MPa**

## 消防設備壓力規定

| 設備 | 放水壓力 | 換算 |
|------|---------|------|
| 室內消防栓(一種) | ≥ 1.7 kgf/cm² | = 0.17 MPa = 17m水頭 |
| 室內消防栓(二種) | ≥ 2.5 kgf/cm² | = 0.25 MPa = 25m水頭 |
| 室外消防栓 | ≥ 2.5 kgf/cm² | = 0.25 MPa |
| 撒水頭(一般) | ≥ 1.0 kgf/cm² | = 0.1 MPa = 10m水頭 |
| 消防栓上限 | ≤ 7.0 kgf/cm² | 超過需裝減壓閥 |

## 配管摩擦損失

### Hazen-Williams 公式

消防配管最常用的摩擦損失計算公式：

`h_f = (6.05 × 10⁵ × Q^1.85) / (C^1.85 × d^4.87) × L`

| 參數 | 說明 |
|------|------|
| h_f | 摩擦損失水頭 (m) |
| Q | 流量 (L/min) |
| C | 管路係數（鍍鋅鋼管≈120，銅管≈150）|
| d | 管內徑 (mm) |
| L | 管路長度 (m) |

### 等值管長

彎管、閥門的摩擦損失換算成等值直管長度：

| 管件 | 等值管長(約) |
|------|------------|
| 90°彎管 | 管徑×30倍 |
| T型接頭 | 管徑×60倍 |
| 閘閥（全開）| 管徑×8倍 |
| 逆止閥 | 管徑×100倍 |

## 幫浦全揚程

`H = h₁ + h₂ + h₃ + h₄`

| 項目 | 說明 | 計算 |
|------|------|------|
| h₁ | 落差 | 幫浦至最高消防栓的垂直高度 |
| h₂ | 摩擦損失 | Hazen-Williams公式 |
| h₃ | 放水壓力 | 規定最低壓力×10m/kgf |
| h₄ | 附屬損失 | 閥門、接頭等(約10%) |

## 配管口徑選擇

- 流速不得超過管路容許流速
- 消防栓配管流速一般 ≤ **4 m/s**
- 撒水設備配管流速一般 ≤ **6 m/s**
- 口徑太小→摩擦損失大→壓力不足

## 記憶口訣

> **「10m水頭=1公斤力=0.1MPa」**
>
> 消防栓一種1.7、二種2.5 kgf/cm²
> 超過7 kgf/cm²要裝減壓閥
> 幫浦揚程=落差+摩擦+放壓+損失
