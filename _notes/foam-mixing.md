## 泡沫原液（泡沫濃縮液）

### 種類與混合比

| 原液種類 | 混合比 | 特點 |
|---------|--------|------|
| 蛋白泡沫 | 3%或6% | 耐熱性佳，流動性差 |
| 氟蛋白泡沫 | 3%或6% | 蛋白+氟界面活性劑 |
| **水成膜(AFFF)** | **3%** 或6% | 最常用，流動性佳 |
| 合成界面活性劑 | 3% | 高發泡用 |
| 耐醇型 | 3%或6% | 抗極性溶劑 |

### 混合比意義
- 3%混合比 = 100L泡沫液中含 **3L原液 + 97L水**
- 6%混合比 = 100L泡沫液中含 **6L原液 + 94L水**

### 原液管理

| 項目 | 規定 |
|------|------|
| 儲存 | 不鏽鋼或防腐蝕容器 |
| 有效期限 | 定期檢查（通常5-10年）|
| 溫度 | 避免凍結和高溫 |
| 混用 | **不同種類不可混用！** |

## 混合裝置

### 壓力比例混合器（最常見）

| 原理 | 說明 |
|------|------|
| 構造 | 壓力水流經文乘利管→產生負壓→吸入原液 |
| 混合比 | 固定比例（3%或6%）|
| 優點 | 結構簡單、不需外部動力 |
| 設置 | 消防幫浦出口到泡沫頭之間 |

### 壓力旁通混合器

| 原理 | 說明 |
|------|------|
| 構造 | 加壓原液槽+旁通管路 |
| 混合比 | 可調整 |
| 優點 | 適合中大型系統 |

### 管線比例混合器

| 原理 | 說明 |
|------|------|
| 構造 | 利用管路內水流速度差異混合 |
| 混合比 | 較精確 |
| 適用 | 固定式消防系統 |

## 泡沫頭（泡沫噴頭）

| 種類 | 適用 |
|------|------|
| 泡沫噴頭 | 低發泡，直接覆蓋 |
| 高發泡產生器 | 高發泡，充滿空間 |
| 泡沫室（foam chamber） | 油槽頂部注入 |

### 泡沫頭放射量

| 場所 | 放射量 |
|------|--------|
| 停車場 | ≥ 6.5 L/min·m² |
| 飛機修護廠 | ≥ 6.5 L/min·m² |

## 原液量計算

`原液量 = 放射量 × 面積 × 放射時間 × 混合比`

範例（停車場200m²，AFFF 3%，放射10分鐘）：
- 泡沫液量 = 6.5 × 200 × 10 = 13,000 L
- 原液量 = 13,000 × 0.03 = **390 L**

## 記憶口訣

> **「AFFF 3%最常用，壓力比例混合器最常見」**
>
> 3%混合=每100L泡沫液含3L原液
> 不同種類原液不可混用！
> 泡沫頭放射量：停車場≥6.5 L/min·m²
