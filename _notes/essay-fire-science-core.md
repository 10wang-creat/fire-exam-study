## 一、燃燒四面體與滅火方法（107年）

### 燃燒四面體

| 要素 | 說明 | 對應滅火方法 |
|------|------|------------|
| **可燃物** | 固/液/氣態可燃物 | **移除法**（搬移可燃物） |
| **氧氣** | 空氣中≥15%即可燃 | **窒息法**（CO₂、泡沫覆蓋） |
| **熱能** | 達到引燃溫度 | **冷卻法**（水、細水霧） |
| **連鎖反應** | 自由基鏈鎖反應 | **抑制法**（乾粉、鹵化烴） |

> 移除任一要素即可滅火。前三面=燃燒三角形，加連鎖反應=四面體

### 細水霧滅火機制（108年）

| 機制 | 說明 |
|------|------|
| **冷卻** | 水滴表面積大→蒸發吸熱效率極高 |
| **窒息** | 水蒸氣體積膨脹1,700倍→排擠氧氣 |
| **遮斷輻射** | 水霧遮擋輻射熱回饋 |
| **稀釋** | 水蒸氣稀釋可燃蒸氣濃度 |

### 水滅火原理（114年）

- 比熱大（4.18 kJ/kg·°C）→ 吸收大量熱
- 蒸發潛熱大（2,257 kJ/kg）→ 氣化時吸收更多熱
- 水蒸氣膨脹排擠氧氣

## 二、閃火點 vs 發火點（106年）

| 項目 | 閃火點 Flash Point | 發火點 Fire Point |
|------|-------------------|------------------|
| 定義 | 液體蒸氣與空氣混合，遇火源產生**閃燃**之最低溫度 | 遇火源後能**持續燃燒5秒以上**之最低溫度 |
| 溫度 | 較低 | 較高（通常高10~30°C） |
| 燃燒持續 | **瞬間閃燃**即熄滅 | **持續燃燒** |
| 意義 | 表示液體產生可燃蒸氣之難易度 | 表示液體持續燃燒之條件 |

## 三、煙囪效應與中性帶（109年）

### 煙囪效應

高層建築物內外溫差造成空氣浮力驅動之**垂直氣流**現象。

```
冬季（室內暖、室外冷）：
  室內暖空氣上升 → 高層正壓（煙往外）
  外部冷空氣由低層進入 → 低層負壓（空氣往內）

         屋頂
    ↑  ← 煙/暖空氣流出
    ↑
  ─────── 中性帶（壓力=0）───────
    ↓
    ↓  → 冷空氣流入
         地面
```

### 中性帶（Neutral Plane）

- 室內外壓力差為零的水平面
- 中性帶**以上**：正壓→煙往外擴散
- 中性帶**以下**：負壓→外部空氣補入

### 補充空氣

- 火災消耗氧氣→低層負壓→外部新鮮空氣由低處開口補入
- 補充空氣供給燃燒所需氧氣→火勢持續

### 高層建築火災特性（113年）

1. **煙囪效應**加劇煙的垂直擴散
2. **人員疏散困難**（樓層多、人數多）
3. **消防搶救困難**（雲梯車高度限制）
4. **風壓影響**（高樓風速大→助長火勢）
5. **排煙困難**
6. **連結送水管壓力損失大**

## 四、BLEVE（110年）

**BLEVE** = Boiling Liquid Expanding Vapor Explosion（沸騰液體膨脹蒸氣爆炸）

### 發生機制中的熱傳方式

| 熱傳方式 | 在BLEVE中的角色 |
|---------|---------------|
| **輻射** | 外部火焰輻射加熱儲槽壁面（主要） |
| **對流** | 火焰熱氣流對流加熱槽壁上部（氣相區） |
| **傳導** | 槽壁金屬傳導熱量至液面以上區域 |

### 過程

1. 外部火災→輻射加熱儲槽
2. 液面以上槽壁無液體冷卻→**局部過熱弱化**
3. 內部液體溫度上升→壓力上升
4. 槽壁在弱化處**破裂**
5. 內部高壓液體瞬間釋壓→**急速汽化膨脹**
6. 產生火球（fireball）+碎片飛射

## 五、閃火 vs 蒸氣雲爆炸（110年）

| 比較 | 閃火 Flash Fire | 蒸氣雲爆炸 VCE |
|------|----------------|---------------|
| 定義 | 可燃氣雲被點燃後**火焰快速掃過** | 可燃氣雲被點燃後**產生壓力波** |
| 壓力波 | **無**顯著壓力波 | **有**破壞性壓力波 |
| 危害 | 主要為**熱輻射** | 壓力波+熱輻射 |
| 條件 | 無密閉或障礙物 | 有**部分密閉**或障礙物→火焰加速 |
| 火焰速度 | 層流~弱紊流 | 高度紊流→爆轟 |

## 六、燃燒計算（105、106年）

### 爆炸上下限（Le Chatelier公式）

混合氣體爆炸下限：**1/LEL_mix = Σ(yi/LELi)**

### 限氧濃度（LOC）

**LOC = LEL × (理論空氣中氧比例) × (化學計量氧莫耳數)**

### 斷熱壓縮（106年）

**T₂ = T₁ × (P₂/P₁)^((γ-1)/γ)**

γ = 比熱比（空氣 ≈ 1.4）

## 七、禁水性物質化學式（112年）

| 物質 | 與水反應 | 產物 |
|------|---------|------|
| **K（鉀）** | 2K + 2H₂O → 2KOH + **H₂↑** | 氫氣（可燃） |
| **Na₂O₂（過氧化鈉）** | 2Na₂O₂ + 2H₂O → 4NaOH + **O₂↑** | 氧氣（助燃） |
| **MgO₂（過氧化鎂）** | 2MgO₂ + 2H₂O → 2Mg(OH)₂ + **O₂↑** | 氧氣 |
| **Li₂C₂（碳化鋰）** | Li₂C₂ + 2H₂O → 2LiOH + **C₂H₂↑** | 乙炔（可燃） |
| **Al₄C₃（碳化鋁）** | Al₄C₃ + 12H₂O → 4Al(OH)₃ + 3**CH₄↑** | 甲烷（可燃） |

> 共同特徵：與水反應產生**可燃氣體或助燃氣體**→嚴禁用水滅火

### 視覺密度與遮光率（103年）

- **遮光率 OD = 1 - 透過率T**
- **視覺密度 Cs = -log₁₀(T) = -log₁₀(1-OD)**
- 視覺密度 Cs = 1 時：T = 0.1，遮光率 = **90%**
