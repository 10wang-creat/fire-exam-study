## 功能

火警信號確認後，在火災區域發出**聽覺警報**，通知該區域人員火災發生。

## 種類

| 種類 | 特點 |
|------|------|
| 火警鈴（bell） | 機械式鈴聲，最傳統 |
| 蜂鳴器（buzzer） | 電子式蜂鳴音 |
| 喇叭（horn） | 電子式警報音 |

## 音量規定

| 項目 | 規定 |
|------|------|
| 音量 | 距 **1m** 處 ≥ **90 dB** |
| 與背景音關係 | 高於背景音 **15 dB** 以上 |
| 可聞性 | 該樓層任何角落都能聽到 |

## 設置規定

| 項目 | 規定 |
|------|------|
| 設置位置 | 各樓層走廊、通道 |
| 設置高度 | 距地面 **2.5m** 以上（壁掛式）|
| 涵蓋範圍 | 該樓層全區域都能聽到 |
| 與受信總機連動 | 受信總機動作→自動鳴動 |

## 鳴動方式

| 方式 | 說明 |
|------|------|
| 全區鳴動 | 全棟所有地區音響同時響 |
| 區域鳴動 | 僅火災樓層+相關樓層響 |

### 區域鳴動原則（與緊急廣播相同）

| 火災樓層 | 鳴動範圍 |
|---------|---------|
| 2F以上 | 火災層 + 直上層 |
| 1F | 火災層 + 直上層 + 全地下層 |
| 地下層 | 全地下層 + 1F |

## 地區音響 vs 主音響

| 項目 | 地區音響 | 主音響 |
|------|---------|--------|
| 設置位置 | 各樓層 | 受信總機上 |
| 功能 | 通知該區域人員 | 通知管理人員 |
| 音量 | ≥90 dB@1m | ≥70 dB@1m |
| 可停止 | 一般不可 | 管理人員可操作停止 |

## 地區音響 vs 緊急廣播

| 項目 | 地區音響 | 緊急廣播 |
|------|---------|---------|
| 信號 | 警報音（鈴聲/蜂鳴）| **語音**廣播 |
| 設置條件 | 設有火警自動警報時 | 11層以上/地下建築物 |
| 功能 | 通知火災發生 | 引導避難方向 |
| 優先 | 廣播啟動後可停止音響 | 廣播優先 |

## 記憶口訣

> **「地區音響90dB@1m，掛2.5m高」**
>
> 主音響在受信總機70dB，地區音響在各樓層90dB
> 2F以上→本層+上層，1F→本+上+全地下
