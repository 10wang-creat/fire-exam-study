## 沸溢（Boilover）

### 發生條件
1. 大型原油儲槽火災
2. 原油中含有**水分層**（底部沉水）
3. 燃燒產生**熱波**（Hot Zone）向下傳播
4. 熱波到達底部水層

### 機制
```
油面燃燒 → 熱波向下傳播(約1m/hr)
→ 到達底部水層 → 水被加熱至100°C+
→ 水瞬間氣化 → 體積膨脹1700倍
→ 油被噴出 → 巨大火焰噴射
```

### 特點

| 項目 | 說明 |
|------|------|
| 發生時機 | 燃燒**數小時**後（熱波需時間傳播）|
| 規模 | **極大**（整個油面噴出）|
| 危險性 | **極高**（噴射高度可達100m+）|
| 可預測性 | 可透過**測溫**監控熱波位置 |
| 典型物質 | 原油（含水+多成分）|

## 濺溢（Slopover）

### 發生條件
1. 含水油品燃燒
2. 射水滅火時水落入高溫油面
3. 水接觸高溫油→瞬間沸騰

### 機制
```
射水 → 水落入高溫油面(>100°C)
→ 水沸騰蒸發 → 油被「濺」出槽外
→ 溢出的油持續燃燒
```

### 特點

| 項目 | 說明 |
|------|------|
| 發生時機 | 射水滅火時**立即** |
| 規模 | 較小（局部溢出）|
| 危險性 | 中等 |
| 原因 | 滅火用水造成 |

## 沸溢 vs 濺溢 比較

| 項目 | 沸溢 Boilover | 濺溢 Slopover |
|------|-------------|--------------|
| **水的來源** | 油品底部的**原有水分** | **滅火用水**落入油面 |
| **發生時間** | 燃燒數小時後 | 射水時立即 |
| **規模** | **極大**（全面噴出） | 較小（局部濺出）|
| **可預測** | 可以（測溫）| 不易預測 |
| **適用油品** | 原油（多成分） | 各種高溫油品 |

## 防制措施

| 對策 | 說明 |
|------|------|
| 監控熱波位置 | 定時測量槽壁溫度 |
| 排水作業 | 排除槽底積水 |
| 泡沫覆蓋 | 以泡沫覆蓋油面（不用水直射）|
| 安全距離 | 預測沸溢前及時撤離 |

## 記憶口訣

> **「沸溢：底部有水+熱波傳到→大噴發」**
> **「濺溢：射水進油面→小濺出」**
>
> 沸溢很久後才發生（但超猛）
> 濺溢射水就發生（但較小）
