## 定義

地下建築物：建築物地下層面積合計達 **1,000m²** 以上者。

## 為什麼特別危險？

| 危險因素 | 說明 |
|---------|------|
| 排煙困難 | 無自然通風開口，煙霧累積快 |
| 避難困難 | 只能向上避難，距離長 |
| 搶救困難 | 消防車無法直接進入 |
| 高溫累積 | 熱無法散逸，溫度上升快 |
| 通訊困難 | 地下無線電通訊不良 |

## 加嚴設備要求

地下建築物比一般場所需要**更多、更嚴格**的消防設備：

| 設備 | 要求 |
|------|------|
| 自動撒水設備 | **必設**（面積≥1000m²）|
| 排煙設備 | **必設** |
| 緊急照明 | **必設** |
| 緊急廣播 | **必設** |
| 連結送水管 | **必設** |
| 無線電通信輔助設備 | **必設** |
| 防煙區劃 | 每500m²一區 |

## 防煙區劃特別規定

- 每一防煙區劃 ≤ **500m²**
- 防煙壁下垂 ≥ **50cm**
- 排煙量每區 ≥ 120 m³/min

## 避難設施

- 步行距離至安全梯 ≤ **30m**
- 至少兩方向避難
- 設置緊急進口（供消防搶救進入）

## 記憶口訣

> **「地下1000m²就是地下建築物」**
>
> 地下五難：排煙難、避難難、搶救難、高溫、通訊差
> 撒水+排煙+照明+廣播+送水管+無線電 = 全都必設
