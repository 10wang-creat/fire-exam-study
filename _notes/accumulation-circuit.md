## 蓄積功能

### 定義
受信總機收到探測器信號後，**延遲一定時間**（蓄積時間）才發出警報，用以排除誤報。

### 蓄積 vs 非蓄積

| 項目 | 蓄積 | 非蓄積 |
|------|------|--------|
| 動作 | 延遲後才發警報 | **立即**發警報 |
| 蓄積時間 | **≤ 60秒** | 0秒 |
| 用途 | 一般場所（排除誤報）| 高危險場所 |
| 優點 | 減少誤報 | 最快速反應 |

### 必須使用非蓄積的場所

| 場所 |
|------|
| 地下層 |
| 無開口樓層 |
| 11層以上 |
| 其他高危險場所 |

> 高危險場所不能等，必須立即警報！

## 迴路規定

### P型受信總機迴路

| 型式 | 最大迴路數 | 適用 |
|------|----------|------|
| P型第一種 | ≤ **7** 迴路 | 中型建築 |
| P型第二種 | ≤ **2** 迴路 | 小型場所 |

### 每一迴路的探測器數量

- 每一迴路連接探測器 ≤ 一定數量
- 過多探測器在同一迴路 → 定位困難

### 迴路監視

| 功能 | 說明 |
|------|------|
| 斷線監視 | 偵測配線是否斷裂 → 異常燈亮 |
| 短路監視 | 偵測配線是否短路 |
| 接地監視 | 偵測配線是否接地 |

## 配線施工規定

| 項目 | 規定 |
|------|------|
| 耐燃/耐熱 | 消防配線須使用耐燃或耐熱配線 |
| 專用迴路 | 消防設備應有**專用配線** |
| 不得與一般配線共用 | 避免干擾 |
| 末端電阻 | 迴路末端設終端電阻（斷線監視用）|
| 配管 | 金屬導線管或耐火電纜 |

## 記憶口訣

> **「蓄積≤60秒排誤報，地下/無開口/11F必須非蓄積」**
>
> P1≤7迴路，P2≤2迴路
> 配線末端設終端電阻=斷線監視
> 消防配線專用，不得與一般電線共管
