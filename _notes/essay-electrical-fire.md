## 一、電氣火災三大原因

```
電氣火災
├─ ① 短路（最常見）
│     └→ 絕緣劣化→導體接觸→大電流→電弧→起火
├─ ② 過載（過負載）
│     └→ 電流超過額定→導體過熱→絕緣熔化→起火
└─ ③ 積污導電（漏電追蹤）
      └→ 插頭積灰+濕氣→微小電流→碳化路徑→短路→起火
```

---

## 二、短路（Short Circuit）

### 原理

1. 電線**絕緣層劣化**（老化、鼠咬、擠壓、釘穿）
2. 正負極導體直接接觸
3. 回路電阻驟降 → 電流**瞬間飆升**（可達額定數十倍）
4. 產生高溫**電弧**（溫度可達 **3,000~6,000°C**）
5. 電弧噴濺熔融金屬粒子 → 引燃周圍可燃物

### 特徵

- 發生**瞬間**，時間極短
- 產生**電弧**痕跡（鑑定依據）
- 導體末端有**熔融珠**（bead）

---

## 三、積污導電（Tracking）

### 原理

1. 電器插頭長期插在插座上，**積聚灰塵**
2. 灰塵吸附空氣中**濕氣**
3. 灰塵+水分形成微弱**導電路徑**
4. 產生**微小漏電流**，持續發熱
5. 熱量使絕緣材料表面逐漸**碳化**
6. 碳化物導電性更好 → 電流增大 → **正回饋**循環
7. 最終碳化路徑形成短路 → **起火**

### 特徵

- 發展**緩慢**（數月至數年）
- 不需要大電流，一般**斷路器不會跳脫**
- 常見於長期不拔的電器插頭（電視、冰箱後方）
- 火災鑑定可見插頭表面**碳化痕跡**

### 預防

- 定期拔除不用之插頭
- 保持插頭清潔乾燥
- 使用具**追蹤指數（CTI）**高的絕緣材料

---

## 四、過載（Overload）

### 延長線過載原理（114年考題）

1. 延長線額定容量有限（通常 **1,650W** = 110V × 15A）
2. 同時插接多個高功率電器 → 總功率超過額定
3. 通過導體之電流超過額定值
4. 依焦耳定律：**Q = I²Rt**（發熱量與電流平方成正比）
5. 導體溫度持續升高
6. 超過絕緣層耐溫 → 絕緣**軟化、熔化**
7. 相鄰導體絕緣層熔融接觸 → 短路 → 起火

### 焦耳定律

> **Q = I²Rt**
>
> Q：發熱量（J）
> I：電流（A）
> R：電阻（Ω）
> t：時間（s）

### 特徵

- 發展有**漸進過程**（先升溫、後熔化）
- 斷路器可能會跳脫（若正常運作）
- 延長線外皮可能先出現**變色、變形**

## 五、其他電氣火災原因（109年考題）

109年考題要求「除過負載外」的原因：

| 原因 | 說明 |
|------|------|
| **半斷線** | 導線內部部分斷裂→截面積減小→局部電阻增大→過熱 |
| **接觸不良** | 接頭鬆動→接觸電阻增大→局部過熱→起火 |
| **電弧** | 開關接觸不良/導線斷裂時產生電弧→高溫 |
| **漏電** | 絕緣劣化→電流經非正常路徑流向大地→發熱 |
| **靜電放電** | 電荷累積後瞬間放電→火花→引燃可燃氣體/粉塵 |

## 六、鋰電池火災（114年考題延伸）

### 原因

- **內部短路**：隔離膜破損→正負極接觸
- **過充電**：電壓超過安全值→電解液分解→產氣膨脹
- **熱失控**：溫度超過臨界點→自加速反應→爆炸

### 特性

- 燃燒溫度極高
- 產生有毒氣體（HF、CO）
- **水系統**可有效冷卻抑制（大量水吸收熱量，降溫至熱失控溫度以下）

### 水滅鋰電池原理（114年第2題）

1. **冷卻**：大量水吸收熱量，降低電池溫度
2. 水的蒸發潛熱極大（**2,257 kJ/kg**）
3. 使電池溫度降至**熱失控臨界溫度以下**
4. 阻止鄰近電池的**熱傳播**（thermal propagation）

## 記憶口訣

> **「短路瞬間弧，積污慢慢碳，過載焦耳熱」**——三大原因
> **「Q = I²Rt」**——過載發熱
> **「3000~6000度電弧」**——短路溫度
> **「灰塵+濕氣→碳化→正回饋→短路」**——積污導電四步
> **「半斷線、接觸不良、電弧、漏電、靜電」**——其他五原因
> **「鋰電池熱失控，大量水冷卻」**
