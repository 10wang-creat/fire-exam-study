## 滅火器種類與適用

| 種類 | 適用火災 | 滅火原理 |
|------|---------|---------|
| 水滅火器 | A | 冷卻 |
| 泡沫滅火器 | A、B | 窒息+冷卻 |
| CO₂滅火器 | B、C | 窒息+冷卻 |
| 乾粉滅火器（ABC） | A、B、C | 抑制連鎖反應 |
| 乾粉滅火器（BC） | B、C | 抑制連鎖反應 |
| 鹵化烷滅火器 | A、B、C | 抑制連鎖反應 |

## 火災分類對照

| 類別 | 火災對象 | 標示顏色 |
|------|---------|---------|
| A類 | 普通火災（木材、紙張）| 白色 |
| B類 | 油類火災（汽油、油脂）| 黃色 |
| C類 | 電器火災 | 藍色 |
| D類 | 金屬火災（鎂、鈉）| 無統一標示 |

## 效能值

滅火器依滅火效能分級，設置時以**效能值**計算：

### 設置基準

| 場所 | 每一滅火效能值涵蓋面積 |
|------|---------------------|
| 甲類場所 | ≤ **100 m²** |
| 乙類場所 | ≤ **200 m²** |
| 其他 | ≤ **200 m²** |

### 計算範例

甲類場所面積 500m²：
- 需要效能值 = 500 / 100 = 5
- 若每支滅火器效能值為 2A → 需 3 支（取整數進位）

## 設置規定

| 項目 | 規定 |
|------|------|
| 步行距離 | ≤ **20m**（任一點至最近滅火器）|
| 安裝高度 | 離地面 **1.5m** 以下 |
| 標示 | 設置「滅火器」標示，紅底白字 |
| 不得上鎖 | 滅火器箱不得上鎖 |
| 放置面 | 不得直接放於地面（潮濕腐蝕）|

## 檢查要點

| 檢查項目 | 內容 |
|---------|------|
| 壓力錶 | 指針在綠色範圍 |
| 安全插銷 | 完好且附封條 |
| 外觀 | 無腐蝕、變形、損傷 |
| 噴嘴 | 無堵塞 |
| 藥劑有效期限 | 未過期 |
| 蓄壓式壓力 | 壓力錶在正常範圍 |

## 記憶口訣

> **「步行20，掛高1.5，甲100乙200」**
>
> 任何位置走20m內有滅火器
> 滅火器底部離地≤1.5m
> 效能值：甲類場所100m²/1效能值
