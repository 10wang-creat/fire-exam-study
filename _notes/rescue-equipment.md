## 消防搶救上之必要設備

設置標準所列的第四大類設備，供**消防人員搶救使用**（非一般人員使用）。

## 設備種類

| 設備 | 功能 | 設置條件 |
|------|------|---------|
| 連結送水管 | 消防車送水至高樓層 | 7層以上 |
| 消防專用蓄水池 | 供消防車取水 | 依面積/用途 |
| 排煙設備 | 排除煙霧利於搶救 | ≥500m²樓層 |
| 緊急電源插座 | 供消防搶救用電 | 11層以上 |
| 無線電通信輔助設備 | 地下搶救通訊 | 地下建築物 |
| 緊急昇降機間 | 消防員專用電梯 | 11層以上 |

## 緊急電源插座

| 項目 | 規定 |
|------|------|
| 設置樓層 | 11層以上各樓層 |
| 設置位置 | 樓梯間或其附近 |
| 電壓 | 單相交流110V |
| 容量 | ≥ 1.5 kVA |
| 電源 | 連接緊急發電機 |

## 緊急昇降機

| 項目 | 規定 |
|------|------|
| 設置條件 | 11層以上建築物 |
| 載重 | ≥ 900 kg（含消防員+裝備）|
| 速度 | ≥ 60 m/min |
| 專用鑰匙 | 消防人員持有 |
| 不受停電影響 | 連接緊急電源 |
| 前室(排煙室) | 設有排煙設備 |

## 與一般消防設備的差異

| 類別 | 使用者 | 目的 |
|------|--------|------|
| 滅火設備 | 一般人員 | 初期滅火 |
| 警報設備 | 自動/一般人員 | 通報火災 |
| 避難設備 | 一般人員 | 逃生避難 |
| **搶救設備** | **消防人員** | **專業搶救** |

## 記憶口訣

> **「搶救設備是消防員用的，不是一般人」**
>
> 7層→連結送水管
> 11層→緊急電源插座+緊急昇降機
> 地下→無線電通信
