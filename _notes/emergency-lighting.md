## 功能

停電時自動點亮，提供避難路徑的最低照明，確保人員安全疏散。

## 基本規格

| 項目 | 規定 |
|------|------|
| 照度 | 地面照度 ≥ **1 lux** |
| 持續時間 | ≥ **30 分鐘** |
| 啟動方式 | 停電後**自動點亮** |
| 電源 | 蓄電池或緊急發電機 |

## 設置場所

- 甲類場所之居室
- 走廊、樓梯間、通道
- 地下層
- 無開口樓層
- 11樓以上建築物

## 設置位置

- 走廊、樓梯間、通道的地面
- 轉角處
- 出入口附近

## 免設條件

- 居室面積 < 一定面積且有直通室外之出口
- 設有自動照明（如連接緊急發電機的一般照明）

## 與標示設備的比較

| 項目 | 緊急照明 | 出口標示燈 |
|------|---------|----------|
| 功能 | 照亮地面 | 標示出口位置 |
| 照度 | ≥1 lux | 依規格 |
| 常態 | 停電才亮 | **常時亮燈** |
| 持續時間 | ≥30分鐘 | ≥30分鐘 |
| 設置位置 | 通道、走廊 | 出口門上方 |

## 檢修要點

| 項目 | 內容 |
|------|------|
| 外觀 | 燈具有無破損、變色 |
| 點亮測試 | 切斷電源→確認自動點亮 |
| 照度測試 | 地面照度≥1 lux |
| 持續時間 | 蓄電池容量能維持30分鐘 |
| 切換裝置 | 停電後40秒內自動切換 |

## 記憶口訣

> **「緊急照明：1lux地面亮30分鐘」**
>
> 停電自動亮（出口標示燈則是常時亮）
> 設在走廊、樓梯、通道
