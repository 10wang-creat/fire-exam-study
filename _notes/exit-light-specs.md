## 出口標示燈分級

| 等級 | 有效距離 | 表面亮度 |
|------|---------|---------|
| **A級（大型）** | ≤ **30m** | 最亮 |
| **B級（中型）** | ≤ **20m** | 中等 |
| **C級（小型）** | ≤ **10m** | 較暗 |

## 亮度規格

| 項目 | 規定 |
|------|------|
| 文字面亮度 | ≥ **50 cd/m²**（白底綠字）或依規格 |
| 均勻度 | 最亮與最暗處亮度比 ≤ **10:1** |
| 辨識性 | 在規定距離內可清楚辨識 |

## 避難方向指示燈分級

| 等級 | 設置間距 | 適用 |
|------|---------|------|
| **A級（大型）** | ≤ **20m** | 大型建築 |
| **B級（中型）** | ≤ **15m** | 一般場所 |
| **C級（小型）** | ≤ **10m** | 小型場所 |

## 設置位置與高度

### 出口標示燈
| 項目 | 規定 |
|------|------|
| 設置位置 | 出口門**上方**或**側邊** |
| 標示面 | 面向避難方向 |
| 常態 | **常時亮燈**（不可平時關閉）|
| 緊急供電 | 停電後≥ **20分鐘**（常亮型）|

### 避難方向指示燈
| 項目 | 規定 |
|------|------|
| 設置高度 | 距地面 **1m 以下** |
| 設置位置 | 走廊、通道、轉彎處 |
| 常態 | **常時亮燈** |

## 免設出口標示燈條件

- 居室面積 < **400m²**
- 出口**可直接觀察到**
- 各居室直接面向主要出口

## 標示內容

| 種類 | 內容 | 配色 |
|------|------|------|
| 出口標示 | 「出口」或「EXIT」| 綠底白字 或 白底綠字 |
| 方向指示 | 箭頭 + 避難方向 | 綠底白色箭頭 |
| 觀眾席引導 | 引導方向 | 綠色 |

## 緊急照明照度規定

| 項目 | 規定 |
|------|------|
| 照度 | 地面 ≥ **1 lux** |
| 測量位置 | 地面 |
| 持續時間 | ≥ **30 分鐘** |
| 燈具位置 | 走廊、樓梯間、通道 |

## 記憶口訣

> **「出口標示ABC: A大30m、B中20m、C小10m」**
> **「方向指示ABC: A大20m、B中15m、C小10m」**
>
> 出口標示=常時亮，停電繼續亮20-30分鐘
> 方向指示裝在1m以下（靠近地面，煙少處）
> 緊急照明：1 lux地面亮30分鐘
