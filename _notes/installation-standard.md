## 法規地位

「各類場所消防安全設備設置標準」是消防法規中**最重要**的子法，考試出題比例最高。

## 場所分類

### 甲類場所（高風險、公眾使用）

| 分類 | 用途舉例 |
|------|---------|
| 甲-1 | 電影院、歌廳、KTV |
| 甲-2 | 舞廳、酒家、夜總會 |
| 甲-3 | 餐廳、飲食店 |
| 甲-4 | 百貨商場、超級市場 |
| 甲-5 | 旅館、飯店、民宿 |
| 甲-6 | 醫院、療養院、安養中心 |
| 甲-7 | 三溫暖、游泳池 |

### 乙類場所

| 分類 | 用途舉例 |
|------|---------|
| 乙-1 | 車站、航空站 |
| 乙-2 | 學校、補習班 |
| 乙-3 | 圖書館、博物館 |
| 乙-4 | 寺廟、教堂 |
| 乙-5 | 辦公室、事務所 |
| 乙-6 | 集合住宅、住宅 |

### 丙類場所
- 工廠

### 丁類場所
- 倉庫（高架、一般）

## 設備分四大類

1. **滅火設備**：滅火器、室內消防栓、自動撒水...
2. **警報設備**：火警自動警報、手動報警、緊急廣播...
3. **避難逃生設備**：標示設備、緊急照明、避難器具...
4. **消防搶救上之必要設備**：排煙設備、連結送水管...

## 設置基準原則

依據場所用途、樓地板面積、樓層數來決定需設置哪些設備。

**一般原則**：
- 面積越大 → 設備要求越多
- 甲類場所 → 設備要求最嚴格
- 地下層 / 無開口樓層 → 額外要求

## 記憶口訣

> **「甲公乙雜丙工丁倉」**
>
> 甲 = 公眾娛樂場所，乙 = 一般公共場所
> 丙 = 工廠，丁 = 倉庫
