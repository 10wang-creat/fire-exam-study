## 一、排煙設備兩大類型

```
排煙設備
├─ 自然排煙（窗戶）
│   → 靠浮力自然排出
│   → 窗戶有效開口面積 ≥ 防煙區劃面積 × 2%
│
└─ 機械排煙（排煙機+風管）
    → 排煙口開啟 → 連動排煙機
    → 排煙量依面積計算
```

---

## 二、應設排煙設備的場所

| 條件 | 說明 |
|------|------|
| 甲類場所 | 樓地板面積 ≥ **500** m²（集合住宅等特定場所除外） |
| 居室 | 面積 ≥ **300** m² + 天花板下方80cm有效通風面積未達面積 **2%** |
| 無開口樓層 | 面積 ≥ **1,000** m² |
| 11F以上 | 全部（除依規定免設者） |
| 地下建築物 | 全部 |

## 三、防煙區劃

| 項目 | 規定 |
|------|------|
| 每區面積 | ≤ **500** m² |
| 防煙壁下垂 | ≥ **50** cm（地下通道 ≥ **80** cm） |
| 排煙口位置 | 防煙區劃內任一位置至排煙口水平距離 ≤ **30** m |

## 四、自然排煙——窗戶設置規定（109年考題）

### 有效開口面積

**有效開口面積 ≥ 防煙區劃面積 × 2%**

### 窗戶規定

| 項目 | 規定 |
|------|------|
| 位置 | 天花板高度 **1/2 以上**之範圍內 |
| 手動開關 | 距樓地板 **80~150** cm |
| 材質 | 與煙接觸部分使用**不燃材料** |

### 窗戶面積計算（考題常考！）

外推窗有效面積依開啟角度折減：

| 開啟方式 | 有效面積 |
|---------|---------|
| 正向外推 90° | 窗面積 × **1.0**（上推）或 × **sin90°=1**|
| 正向外推 45° | 窗面積 × **sin45° ≒ 0.71** |
| 側推（平推）| 窗面積 × 開啟比例 |

> **109年計算範例**：
> 候機室 500 m²，需有效開口 = 500 × 2% = **10 m²**
> 天花板下方80cm內共 20 扇正向外推 90° 窗
> 每扇有效面積 = 10 ÷ 20 = **0.5 m²**

## 五、機械排煙——排煙機風量計算

### 居室空間排煙量

| 情境 | 排煙量 |
|------|--------|
| 最低排煙量 | ≥ **120** m³/min |
| 一個防煙區劃 | ≥ 該區面積 × **1** m³/(min·m²) |
| 二區以上 | ≥ **最大區面積** × **2** m³/(min·m²) |
| 地下通道總排煙量 | ≥ **600** m³/min |

> **計算範例**：
> 某樓層三區：450、400、350 m²
> 排煙量 = max(450) × 2 = **900** m³/min
> 需確認 ≥ 120 → **900** m³/min（取大值）

### 排煙口規定

| 項目 | 規定 |
|------|------|
| 位置 | 天花板高度 **1/2 以上** |
| 材質 | 不燃材料 |
| 開啟方式 | 手動+探測器連動自動 |
| 手動操作部 | 距樓地板 **80~150** cm |

## 六、排煙室排煙（特別安全梯/緊急昇降機間）

### 6-1 直接面向戶外（窗戶）

| 項目 | 特別安全梯 | 兼用排煙室 |
|------|----------|----------|
| 窗戶有效開口 | ≥ **2** m² | ≥ **3** m² |
| 位置 | 天花板高度 1/2 以上 |
| 窗與煙接觸部分 | 不燃材料 |
| 手動開關 | 距樓地板 **80~150** cm |

### 6-2 機械排煙（風管方式）

| 項目 | 特別安全梯排煙室 | 兼用排煙室 |
|------|----------------|----------|
| 排煙量 | ≥ **4** m³/s | ≥ **6** m³/s |
| 進風量 | ≥ **4** m³/s | ≥ **6** m³/s |
| 排煙口位置 | 天花板高度 1/2 以上 |
| 進風口位置 | 天花板高度 1/2 以下 |
| 排煙口 ≠ 進風口 | **不得同側** |
| 連動 | 排煙口開啟 → 排煙機自動啟動 |

## 七、排煙閘門風速計算（112年考題）

### 公式

排煙室風速檢驗：

**排煙量 (m³/s) = 排煙閘門面積 (m²) × 風速 (m/s)**

因此：**風速 = 排煙量 ÷ 排煙閘門面積**

### 合格標準

| 排煙室類型 | 排煙量 | 排煙閘門面積 | 應達風速 |
|-----------|--------|------------|---------|
| 特別安全梯 | 4 m³/s | A m² | 4/A m/s |
| 兼用 | 6 m³/s | A m² | 6/A m/s |

> **112年計算範例**：
> 兼用排煙室，排煙閘門面積 6 m²
> 風速 = 6 ÷ 6 = **1.0** m/s（應符合設計值）

## 八、排煙機性能檢查（108年考題）

### 檢查方法

| 項目 | 方法 |
|------|------|
| 外觀 | 確認無變形、損傷、腐蝕、異物附著 |
| 回轉方向 | 確認葉片回轉方向正確 |
| 電動機 | 確認接地、絕緣電阻 |
| 開關 | 手動/自動啟停正常 |
| 風量 | 以風速計測排煙口風速 |

### 判定方法

- 運轉中無異常聲音及振動
- 風道無異常振動
- 回轉方向正常
- 運轉電流在規定範圍內
