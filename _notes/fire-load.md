## 定義

火載量（Fire Load）：某空間內所有可燃物完全燃燒所釋放的**總熱量**。

## 計算公式

### 火載量密度

`q = Σ(mᵢ × Hᵢ) / A`

- mᵢ = 第 i 種可燃物質量 (kg)
- Hᵢ = 第 i 種可燃物發熱量 (MJ/kg)
- A = 樓地板面積 (m²)
- 單位：MJ/m²

### 等價火載量（木材等價）

將所有可燃物換算成等量木材：

`W_eq = Σ(mᵢ × Hᵢ) / H_wood`

- H_wood = 木材發熱量 ≈ **16.7-18.8 MJ/kg**
- 單位：kg（等價木材重量）

## 常見物質發熱量

| 物質 | 發熱量 (MJ/kg) |
|------|---------------|
| 木材 | 16.7-18.8 |
| 紙張 | 16.7 |
| 棉花 | 16.7 |
| 塑膠（PE）| 46.5 |
| 汽油 | 46.0 |
| 天然氣 | 50.0 |

## 火災持續時間

`t = q / (Q̇ × A)`

或簡化：火載量密度越大 → 火災持續時間越長

## 用途

- 評估建築物火災嚴重度
- 決定耐火構造等級
- 設計消防設備容量
- 火災後結構安全評估

## 典型場所火載量密度

| 場所 | 火載量密度 (MJ/m²) |
|------|------------------|
| 住宅 | 500-800 |
| 辦公室 | 400-600 |
| 圖書館 | 1,500-2,500 |
| 倉庫 | 變化極大 |

## 記憶口訣

> **「火載量 = 所有可燃物的總熱量 ÷ 面積」**
>
> 塑膠發熱量是木材的近3倍！
> 火載量大→火災燒更久→結構要求更嚴
