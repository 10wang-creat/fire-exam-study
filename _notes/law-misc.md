## 義勇消防（民力運用）

| 項目 | 規定 |
|------|------|
| 法源 | 消防法第28條 |
| 編組 | 直轄市/縣市政府得編組義勇消防組織 |
| 任務 | 協助消防、緊急救護工作 |
| 假別 | 訓練、演習、服勤期間 → 所屬機關給予**公假** |
| 傷亡 | 比照公務人員因公撫卹 |

## 消防設備人員法（新法）

| 項目 | 規定 |
|------|------|
| 執業執照 | 有效期間 **6年** |
| 執業方式 | 不以同一執業機構為限（可多處執業）|
| 繼續教育 | 每6年須完成一定積分 |
| 業務範圍 | 設計/監造（師）、裝置/檢修（師+士）|

## 爆竹煙火管理

| 項目 | 規定 |
|------|------|
| 法源 | 爆竹煙火管理條例 |
| 一般爆竹煙火 | 分為多種類 |
| 舞台煙火 | 需申請許可 |
| 儲存 | 達管制量→適用公危物品管理 |

## 消防安全設備檢修專業機構

| 項目 | 規定 |
|------|------|
| 法源 | 消防安全設備檢修專業機構管理辦法 |
| 人員 | 須聘僱一定數量專技人員 |
| 合格證書 | 有效期限 **3年**，屆滿前可申請延展 |
| 設備 | 須具備執行檢修必要器具 |
| 完成標示 | 檢修完成貼標示（底色依規定）|
| 變更 | 事實發生日起一定天數內辦理 |

## 過廊（視為另一場所）

建築物間設有過廊，符合下列規定者，可視為另一場所（分開計算設備設置）：

| 條件 |
|------|
| 過廊僅供通行或搬運用途 |
| 無通行之障礙 |
| 有效寬度在 **6m 以下** |
| 過廊設有防火門 |

## 防災中心

| 項目 | 規定 |
|------|------|
| 設置場所 | 高層建築物（50m以上）|
| 功能 | 集中監控消防/防災設備 |
| 設備 | 受信總機、廣播、排煙控制、CCTV等 |
| 人員 | 24小時須有服勤人員 |

## 原有合法建築物

| 項目 | 規定 |
|------|------|
| 定義 | 法令修正前已存在的合法建築物 |
| 原則 | 應於一定期限內改善消防設備 |
| 特例 | 11層以上室內裝修用耐燃一級→可放寬撒水設置面積 |

## 消防栓維護

| 項目 | 規定 |
|------|------|
| 設置費用 | 自來水事業機構負擔 |
| 保養維護 | 由直轄市/縣市**消防機關**負責 |
| 維護費用 | 由自來水事業機構負擔 |

## 記憶口訣

> **「義消服勤給公假，設備人員法執照6年」**
>
> 檢修機構證書3年，消防栓維護消防機關做+自來水出錢
> 過廊寬≤6m+有防火門 → 可分開計算
