## 安全梯種類

| 種類 | 構造 | 設置條件 |
|------|------|---------|
| 安全梯（一般）| 防火門+防火構造樓梯間 | 5層以上+面積條件 |
| 戶外安全梯 | 設於建築物外部 | 開放式 |
| **特別安全梯** | 安全梯 + **排煙室（前室）** | 15層以上 |

## 特別安全梯構造

```
室內 → [排煙室(前室)] → [樓梯間]
        ↑正壓防煙     ↑防火構造
        ↑甲種防火門   ↑甲種防火門
```

### 排煙室（前室）規定

| 項目 | 規定 |
|------|------|
| 面積 | 各層 ≥ 一定面積 |
| 防火門 | 兩端均設**甲種防火門**（60min）|
| 排煙設備 | 設有排煙設備或正壓送風 |
| 正壓 | 維持正壓，防止煙侵入 |
| 牆壁 | 防火構造 |

## 設置條件

| 建築物 | 安全梯種類 |
|--------|----------|
| 5層以下（一般）| 一般樓梯即可 |
| 6-14層 | **安全梯** |
| **15層以上** | **特別安全梯** |
| 地下3層以上 | **特別安全梯** |

## 兩方向避難

| 項目 | 規定 |
|------|------|
| 原則 | 每一樓層應有**2座以上**安全梯 |
| 步行距離 | 居室任一點至安全梯 ≤ **30m**（設有撒水可放寬至40m）|
| 重複距離 | 兩條避難路徑的重複部分 ≤ 總距離的 **1/2** |

## 防火門要求

| 位置 | 門種 | 方向 |
|------|------|------|
| 安全梯入口 | 甲種防火門(60min) | 向避難方向開啟 |
| 排煙室兩端 | 甲種防火門 | 向避難方向開啟 |
| 常閉式 | 自動關門裝置 | - |

## 記憶口訣

> **「14層以下安全梯，15層以上特別安全梯」**
>
> 特別安全梯 = 安全梯 + 排煙室（前室）
> 排煙室兩端都是甲種防火門(60min)
> 任一點到安全梯≤30m（有撒水40m）
