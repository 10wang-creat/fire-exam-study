## 為什麼需要計算收容人數？

避難器具的設置數量、緊急廣播的揚聲器數量等，都取決於**各樓層收容人數**。

## 收容人數計算公式

`收容人數 = 樓地板面積 ÷ 每人所佔面積`

### 每人所佔面積（依場所用途）

| 場所用途 | 每人面積 (m²) |
|---------|-------------|
| 電影院（固定席位）| 依席位數 |
| 舞廳、歌廳 | **3 m²** |
| 餐廳、飲食店 | **3 m²** |
| 百貨商場 | **4 m²** |
| 旅館客房 | 依房間數 |
| 辦公室 | **5-6 m²** |
| 學校教室 | 依座位數或 **1.5 m²** |
| 醫院病房 | 依病床數 |

## 避難器具設置數量

| 樓層收容人數 | 避難器具數量 |
|------------|-----------|
| ≤ 30 人 | 1 具 |
| 31-100 人 | 2 具 |
| 101-200 人 | 3 具 |
| 每增加100人 | +1 具 |

## 計算範例

百貨商場某樓層面積 800m²：
- 收容人數 = 800 ÷ 4 = **200人**
- 避難器具 = 3 具

## 相關設備數量決定

| 設備 | 數量依據 |
|------|---------|
| 避難器具 | 收容人數（如上表）|
| 揚聲器 | 樓層面積 ÷ 涵蓋面積 |
| 出口標示燈 | 出口數量 |
| 緊急照明 | 走廊/通道面積 |

## 特殊場所

| 場所 | 收容人數算法 |
|------|------------|
| 有固定席位 | 依**席位數** |
| 旅館 | 客房**房間數 × 2** |
| 醫院 | **病床數** |
| 無隔間大空間 | 面積 ÷ 每人面積 |

## 記憶口訣

> **「舞廳餐廳3m²，百貨4m²，辦公6m²」**
>
> 收容人數 = 面積 ÷ 每人面積
> 避難器具：30人1具，每多100人加1具
