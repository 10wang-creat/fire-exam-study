## 一、三者定義

### Boilover（沸溢）

儲槽內含水分之重質油品（原油、重油）長時間燃燒，表面**熱波（heat wave）**逐漸向下傳導，當熱波到達儲槽底部**水層**時，水被加熱至100°C以上急速**汽化膨脹**（體積膨脹約1,700倍），將上方燃燒中的油品**猛烈噴射**至槽外的現象。

### Slopover（濺溢/溢流）

消防人員對燃燒中的油品儲槽進行**射水滅火**時，水進入高溫油層表面（超過100°C），水接觸高溫油後急速汽化，導致油品**溢出或濺出**儲槽的現象。

### Spillover（濺射）

儲槽火災中，燃燒的油品因沸騰等原因從儲槽**溢流而出**的現象。通常發生在儲槽液面**過高**或油品**輕質成分沸騰**時。

---

## 二、三者比較表（核心必背）

| 比較項目 | Boilover 沸溢 | Slopover 濺溢 | Spillover 濺射 |
|---------|-------------|-------------|--------------|
| **原因** | 熱波到達底部水層 | 滅火射水進入高溫油面 | 油品沸騰/液面過高 |
| **水的來源** | 儲槽底部**原有積水** | **消防射水**進入油層 | 非水引起 |
| **發生時機** | 長時間燃燒後（數小時） | 消防作業**當下** | 燃燒過程中 |
| **前兆** | 有（異常聲響、液面升降） | **無**明顯前兆 | 液面上升 |
| **規模** | **極大**（油品噴出數十米） | 較小（局部溢出） | 中等 |
| **危險性** | **最高**（大量燃燒油噴射） | 中等 | 中等 |
| **適用油品** | 含水分之**重質油**/原油 | 各種油品 | 含輕質成分油品 |

## 三、Boilover 熱波機制（詳細）

```
燃燒開始
│
├─ 表面油品燃燒，溫度 300~400°C
│
├─ 表面高溫油向下對流
│   形成「熱波（Heat Wave）」
│   熱波以約 30~90 cm/hr 速率下降
│
├─ 熱波前緣溫度 > 100°C
│
├─ 到達底部水層
│   水被加熱至 100°C 以上
│   ↓
│   水急速汽化（1→1,700 倍體積）
│   ↓
│   💥 大量蒸氣將上方油品猛烈噴射
│   ↓
│   燃燒油品噴出高度可達儲槽直徑的數倍
```

### Boilover 發生條件

| 條件 | 說明 |
|------|------|
| 油品含水 | 儲槽底部有**積水層** |
| 重質油品 | 沸點範圍寬（原油、重油） |
| 長時間燃燒 | 需足夠時間讓熱波傳導至底部 |
| 油品黏度高 | 高黏度阻止水蒸氣上升→壓力累積 |

### Boilover 前兆

1. 儲槽壁面溫度突然**異常升高**
2. 儲槽發出**異常聲響**（水沸騰聲）
3. 液面**不穩定升降**
4. 油品表面出現**氣泡**

---

## 四、Slopover 發生機制

```
消防隊對燃燒儲槽射水
    ↓
水進入高溫油層表面（油溫 > 100°C）
    ↓
水接觸高溫油後急速汽化
    ↓
汽化產生的蒸氣將周圍油品帶出
    ↓
燃燒油品溢出或濺出儲槽
```

### 特徵

- **隨時可能發生**（只要射水就有風險）
- 規模較小但可能傷及消防人員
- 可透過**泡沫滅火**（非水柱）降低風險

---

## 五、防範對策

| 現象 | 防範對策 |
|------|---------|
| **Boilover** | 預估熱波到達時間→提前撤離、監測槽壁溫度、排放底部積水 |
| **Slopover** | 使用**泡沫**而非直接射水、保持安全距離 |
| **Spillover** | 監控液面高度、設防液堤 |

### 熱波速率估算

- 熱波下降速率約 **30~90 cm/hr**
- 可由油品種類和儲槽高度估算到達底部時間
- 消防人員據此決定**撤離時機**

## 記憶口訣

> **「沸溢=底部水+熱波+噴射，時間長最危險」**
> **「濺溢=射水+高溫油+溢出，沒前兆」**
> **「濺射=沸騰+液面高+溢流」**
> **「水汽化膨脹1,700倍」**
> **「熱波速率30-90 cm/hr」**
