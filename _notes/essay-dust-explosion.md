## 一、粉塵爆炸定義

可燃性固體物質以**微細粉末**狀態分散於空氣中，當其濃度達到爆炸範圍內，遇點火源即發生**急速燃燒**，產生高溫高壓之爆炸現象。

### 爆炸五要素（缺一不可）

```
        ① 可燃性粉塵
              ↓
    ② 助燃性氣體（氧氣）
              ↓
     ③ 點火源（足夠能量）
              ↓
   ④ 粉塵濃度在爆炸範圍內
              ↓
    ⑤ 密閉或半密閉空間
         ↓↓↓
      💥 粉塵爆炸
```

| 要素 | 說明 |
|------|------|
| ① 可燃性粉塵 | 金屬粉（鋁、鎂）、有機物（麵粉、糖粉、木粉、煤粉） |
| ② 氧氣 | 空氣中氧濃度需足夠 |
| ③ 點火源 | 火花、靜電、高溫表面、摩擦熱 |
| ④ 濃度範圍 | 需在爆炸下限（LEL）以上 |
| ⑤ 密閉空間 | 壓力才能累積 |

> 與氣體爆炸不同：粉塵爆炸多了第⑤項「密閉空間」要素

## 二、何謂二次粉塵爆炸（103年考題核心）

### 發生機制

```
第一次爆炸（初始爆炸）
    ↓ 爆風衝擊
堆積在設備/地面的粉塵被揚起
    ↓ 形成高濃度粉塵雲
第一次爆炸的火焰成為點火源
    ↓
💥💥 第二次爆炸（二次爆炸）
    → 規模遠大於第一次
    → 可能引發連鎖爆炸
```

### 特點

| 比較 | 第一次爆炸 | 第二次爆炸 |
|------|----------|----------|
| 規模 | 較小 | **遠大於**第一次 |
| 範圍 | 局部 | 可能擴及整個廠房 |
| 原因 | 局部粉塵雲 | 堆積粉塵被揚起 |
| 點火源 | 原始火花等 | 第一次爆炸的火焰 |
| 傷亡 | 較少 | **主要傷亡**來源 |

## 三、影響粉塵爆炸的因素

| 因素 | 影響 |
|------|------|
| **粒徑越小** | 表面積/體積比越大 → 爆炸越劇烈 |
| **濃度** | 過低（<LEL）或過高（>UEL）不爆炸 |
| **含水量越高** | 抑制爆炸（水分吸熱+惰化） |
| **氧濃度越高** | 爆炸越劇烈 |
| **點火能量越大** | 越容易引爆 |
| **紊流（湍流）** | 加速燃燒速率 → 爆炸更猛 |

## 四、粉塵爆炸防護對策（103+113年都考）

### 預防措施（防止爆炸發生）

| 對策 | 說明 |
|------|------|
| **消除點火源** | 防靜電（接地、穿防靜電衣鞋）、禁止明火、防止機械火花 |
| **控制粉塵濃度** | 局部排風集塵、定期清掃堆積粉塵、防止粉塵飛揚 |
| **惰性化** | 以氮氣或CO₂置換空氣，降低氧濃度至限氧濃度以下 |
| **控制粒徑** | 避免產生過細粉塵 |
| **濕式作業** | 加濕抑制粉塵飛揚 |
| **定期清掃** | 防止粉塵堆積（**防二次爆炸最重要措施**） |

### 防護措施（減輕爆炸損害）

| 對策 | 說明 |
|------|------|
| **洩爆裝置** | 裝設洩爆口/洩爆膜，將爆炸壓力導向安全方向 |
| **抑爆裝置** | 偵測到壓力上升 → 瞬間噴射滅火劑抑制爆炸 |
| **隔爆裝置** | 在管路/設備間設置止火閥、旋轉閥，防止火焰傳播 |
| **耐爆設計** | 設備/建築物設計能承受爆炸壓力 |
| **防止二次爆炸** | 定期清掃、平滑表面設計（減少粉塵堆積面） |
