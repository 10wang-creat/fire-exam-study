## 手動報警機

### 功能
人員發現火災時，手動按壓發出火警信號至受信總機。

### 設置規定

| 項目 | 規定 |
|------|------|
| 步行距離 | 任一點至手動報警機 ≤ **25m** |
| 安裝高度 | 距地面 **0.8m-1.5m** |
| 設置位置 | 走廊、樓梯口、出入口附近 |
| 標示 | 紅色「火警發信機」標示 |
| 操作方式 | 按壓或打破玻璃 |

### 與探測器的關係
- 手動報警機是**人為操作**的警報啟動
- 火警探測器是**自動偵測**的警報啟動
- 兩者都連接至受信總機

## 119火災通報裝置

### 功能
火警確認後，自動或手動撥打119通報消防機關。

### 種類

| 種類 | 方式 |
|------|------|
| 自動通報 | 受信總機確認火警 → 自動撥打119 |
| 手動通報 | 人員按壓按鈕 → 撥打119 |

### 設置場所
- 甲類場所
- 地下建築物
- 11層以上建築物

## 火警警報整體流程

```
火災發生 → 探測器動作 或 手動報警機按壓
   ↓
受信總機接收信號 → 顯示火災區域
   ↓
地區音響裝置鳴動 + 緊急廣播啟動
   ↓
119火災通報裝置撥打消防局
```

## 記憶口訣

> **「手動報警機：走25m、掛0.8-1.5m」**
>
> 手動靠人按，自動靠探測器
> 最後都送到受信總機
