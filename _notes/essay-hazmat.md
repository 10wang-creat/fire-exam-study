## 一、六類公共危險物品分類（必背表）

| 類別 | 名稱 | 特性 | 代表物質 |
|------|------|------|---------|
| **第一類** | 氧化性固體 | 提供氧氣助燃 | 過氯酸鉀、過氧化鈉 |
| **第二類** | 可燃性固體 | 火焰接觸即燃 | 硫磺、鐵粉、鎂粉 |
| **第三類** | 發火性液體/固體、禁水性物質 | 遇空氣/水自燃 | 鈉、鉀、烷基鋁 |
| **第四類** | 易燃液體 | 閃火點低 | 汽油、甲苯、柴油 |
| **第五類** | 自反應物質/有機過氧化物 | 分解爆炸 | TNT、硝化纖維 |
| **第六類** | 氧化性液體 | 強氧化劑 | 過氧化氫、硝酸 |

### 第四類閃火點分級

| 分級 | 閃火點 | 代表 |
|------|--------|------|
| 特殊易燃物 | <-20°C 或 ≤100°C 沸點 | 乙醚 |
| 第一石油類 | <21°C | 汽油、丙酮 |
| 醇類 | — | 甲醇、乙醇 |
| 第二石油類 | 21~70°C | 煤油、柴油 |
| 第三石油類 | 70~200°C | 重油 |
| 第四石油類 | 200~250°C | 齒輪油 |
| 動植物油類 | <250°C | 花生油 |

> **高閃火點物品** = 閃火點 ≥ **100°C** 之第四類公共危險物品

## 二、場所分類

```
公共危險物品場所
├─ 製造場所
├─ 儲存場所
│   ├─ 室內儲存場所（建築物內，非儲槽）
│   ├─ 室外儲存場所（建築物外，非儲槽，限特定物品）
│   ├─ 室內儲槽場所（建築物內，容量 > 600L，不可移動）
│   ├─ 室外儲槽場所（建築物外，容量 > 600L，不可移動）
│   └─ 地下儲槽場所（地面下，容量 > 600L）
├─ 處理場所
│   ├─ 第一種販賣場所（< 管制量15倍）
│   ├─ 第二種販賣場所（15~40倍）
│   └─ 一般處理場所
└─ 容器串接使用場所（LPG ≥ 80kg 或 128kg）
```

## 三、一般處理場所構造設備（108年考題）

### 構造規定

| 項目 | 規定 |
|------|------|
| 牆壁/柱/樑/地板 | **防火構造** |
| 窗戶及出入口 | **30分鐘**以上防火時效之防火門窗 |
| 延燒之虞開口 | **1小時**以上防火時效之常時關閉式防火門 |
| 屋頂 | 不燃材料 |
| 地面 | 不滲透構造，適當傾斜，設集液設施 |
| 排放 | 液體六類物品處理設備周圍設 ≥ **15cm** 圍阻措施 |
| 通風/排出設備 | 可燃蒸氣有效排至屋簷以上或室外距地面 **4m** 以上 |

### 罰則

| 違規 | 罰則 |
|------|------|
| 未符合構造設備規定 | 限期改善 → 屆期未改善 → **2萬~10萬**，按次處罰 |
| 致發生火災 | 管理權人 **1年~7年** 有期徒刑 |

## 四、室外儲存場所（109年考題）

### 限儲物品

室外儲存場所**僅限**儲存以下物品：
- 第二類之**硫磺**（塊狀）
- 第四類之**第二石油類**（閃火點 21~70°C，限桶裝）
- 第四類之**第三石油類**、**第四石油類**、**動植物油類**

### 架臺規定

| 項目 | 規定 |
|------|------|
| 材質 | 不燃材料 |
| 高度 | ≤ **6** m |
| 存放容器 | 不得超出架臺 |
| 固定 | 防止容器掉落之措施 |

## 五、公共危險物品場所設火警自動警報設備（107、108年考題）

### 設置條件

| 場所 | 設置條件 |
|------|---------|
| **製造場所** | 總樓地板面積 ≥ **500** m² |
| **一般處理場所** | 總樓地板面積 ≥ **500** m² |
| **室內儲存場所** | 總樓地板面積 ≥ **150** m²（或達管制量100倍） |
| **室內儲槽場所** | 達**顯著滅火困難**者 |
| **室外儲槽場所** | 依防護面積/設備判斷 |

### 緊急通報裝置設置條件

| 條件 | 說明 |
|------|------|
| 達管制量 **10倍** 以上 | 平時有人之場所 |
| 未達火警自動警報設置條件 | 但有人值守 |

### 手動報警設備

| 條件 | 說明 |
|------|------|
| 達管制量 **10倍** 以上 | 平時有人，未設火警自動警報者 |

## 六、LPG 容器串接使用場所（114年考題）

### 定義

使用液化石油氣作燃氣來源，串接使用量 ≥ **80 kg**（或 **128 kg**）之場所

### 安全設施（依串接量）

| 串接量 | 設施 |
|--------|------|
| 80~120 kg | 容器室外放置、防傾倒、嚴禁煙火、滅火器 |
| 120~300 kg | 上述 + **漏氣警報器** + 緊急遮斷裝置 |
| 300~600 kg | 上述 + 防護牆或防爆牆 |
| ≥ 600 kg | 上述 + 符合儲存場所規定 |

### 不得供氣之情形

- 超過期限之容器
- 檢驗不合格容器
- 未標示合格之容器
- 串接使用量超過申報量

## 七、滅火設備分類（場所適用）

| 種類 | 設備 |
|------|------|
| 第一種 | 室內/室外消防栓設備 |
| 第二種 | 自動撒水設備 |
| 第三種 | 水霧/泡沫/CO2/乾粉滅火設備 |
| 第四種 | 大型滅火器 |
| 第五種 | 滅火器、水桶、水槽、乾燥砂、膨脹蛭石/珍珠岩 |

### 場所危險等級與滅火設備

| 場所等級 | 應設滅火設備 |
|---------|------------|
| 顯著滅火困難 | 第一種+第四種+第五種 |
| 一般滅火困難 | 第四種+第五種 |
| 其他 | 第五種 |
