## 能見度公式

`V = K / Cs`

| 參數 | 說明 |
|------|------|
| V | 能見度 (m) |
| K | 常數（反射式標誌≈2-4，發光式標誌≈5-10）|
| Cs | 減光係數 (1/m)，越大表示煙越濃 |

## 避難安全判定

| 能見度 | 狀態 |
|--------|------|
| > 20m | 安全 |
| 10-20m | 注意 |
| 5-10m | 困難 |
| **< 5m** | **危險（一般人無法避難）** |
| < 3m | 極度危險 |

## 減光係數 Cs

- Cs 越大 → 煙越濃 → 能見度越低
- Cs 與煙粒子濃度成正比
- 不同燃燒物產生的煙濃度不同

| 材料 | 煙產生量 |
|------|---------|
| 木材 | 中等 |
| PVC塑膠 | **大量黑煙** |
| 聚氨酯(PU) | 大量+有毒 |
| 紙張 | 較少 |

## 煙層下降與避難

- 煙層下降速度影響可用避難時間
- 天花板越高 → 煙層下降越慢 → 避難時間越長
- 能見度降至5m以下 → 必須匍匐前進

## 光學密度

另一個常用指標：
`OD = log₁₀(I₀/I) / L`

- I₀ = 初始光強度
- I = 穿透後光強度
- L = 光路長度

## 記憶口訣

> **「能見度 = K÷Cs，Cs越大煙越濃」**
>
> 能見度<5m = 無法安全避難
> 發光式K較大(5-10)→同煙濃度下看得比較遠
