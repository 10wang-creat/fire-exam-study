## 粉塵爆炸五要素

普通火災需要「火三角」（燃料+氧+熱源），粉塵爆炸需要**五要素**（火五角形）：

1. **可燃性粉塵**（燃料）
2. **助燃性氣體**（氧氣）
3. **點火源**（熱能）
4. **粉塵飛散懸浮**（分散狀態）
5. **密閉空間**（壓力累積）

> 缺少任一要素就不會爆炸

## 影響爆炸嚴重度的因素

| 因素 | 影響 |
|------|------|
| 粒徑越**小** | 表面積越大 → 爆炸越劇烈 |
| 濃度在爆炸範圍內 | 偏下限附近最危險 |
| 含氧量越高 | 反應越劇烈 |
| 含水率越高 | 爆炸危險**降低**（水有抑制效果）|
| 空間越密閉 | 壓力累積越大 |
| 粉塵揮發分越高 | 越容易爆炸 |

## Kst 值（爆炸指數）

| Kst等級 | Kst值 (bar·m/s) | 爆炸強度 |
|---------|----------------|---------|
| St 0 | 0 | 不爆炸 |
| St 1 | 0-200 | 弱 |
| St 2 | 200-300 | 強 |
| St 3 | >300 | 非常強 |

## 防範對策

### 消除五要素

| 對策 | 針對要素 |
|------|---------|
| 使用濕式作業、灑水降塵 | 消除懸浮 |
| 以惰性氣體（N₂）置換 | 消除氧氣 |
| 使用防爆電氣設備 | 消除點火源 |
| 良好通風排塵 | 防止濃度累積 |
| 設置洩爆口（爆炸洩壓裝置）| 減輕密閉壓力 |

### 工程防護

- **洩爆口**：面積依 NFPA 68 計算
- **抑爆系統**：偵測壓力上升 → 瞬間噴灑抑制劑
- **隔爆裝置**：防止爆炸沿管路傳播

## 二次爆炸

第一次爆炸震動 → 揚起堆積粉塵 → 形成新的粉塵雲 → **二次爆炸**（通常比第一次更劇烈）

## 記憶口訣

> **「燃氧熱飛密」= 粉塵爆炸五要素**
>
> 可燃粉塵 + 氧 + 熱源 + 飛散懸浮 + 密閉空間
