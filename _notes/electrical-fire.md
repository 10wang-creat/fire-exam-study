## 台灣火災統計

電氣因素是台灣火災原因**第一名**（約30-35%）。

## 四大電氣起火原因

### 1. 短路（短絡）

- 電線絕緣劣化，正負極直接接觸
- 瞬間大電流 → 高溫引燃
- 常見於老舊配線

### 2. 過負載（過電流）

- 用電量超過電線安全容量
- 焦耳熱：`Q = I²Rt`
- 常見於延長線串接、一插多用

### 3. 積污導電（碳化導電）

- 插頭長期插在插座，累積灰塵+水氣
- 灰塵受潮 → 微小電流 → 碳化 → 導電路徑形成 → 起火
- 又稱「追蹤現象」（Tracking）

### 4. 半斷線（接觸不良）

- 電線內部部分斷裂，截面積變小
- 同一電流通過更小截面 → 電阻增大 → 局部過熱
- R = ρL/A，A減小 → R增大

## 焦耳定律

`Q = I²Rt = V²t/R`

| 參數 | 說明 |
|------|------|
| Q | 發熱量（焦耳 J） |
| I | 電流（安培 A） |
| R | 電阻（歐姆 Ω） |
| t | 時間（秒 s） |

**重點**：發熱量與電流的**平方**成正比，電流加倍 → 發熱量變 4 倍。

## 電氣火災的分類

屬於 **C 類火災**（電器火災）。

- 滅火前應先**斷電**
- 未斷電時使用：CO₂、乾粉滅火器
- **不可**使用水、泡沫（導電危險）

## 防護措施

| 措施 | 對應原因 |
|------|---------|
| 定期更換老舊配線 | 短路 |
| 避免延長線串接 | 過負載 |
| 定期拔除不用的插頭、清潔 | 積污導電 |
| 檢查電線接頭是否鬆動 | 半斷線 |
| 裝設漏電斷路器（ELCB）| 全部 |
