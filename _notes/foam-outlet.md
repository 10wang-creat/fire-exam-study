## 儲槽用泡沫放出口型式

### I型放出口
- 設於儲槽**側板上方**
- 泡沫從槽壁上方注入，沿槽壁流下覆蓋油面
- 適用：固定頂式儲槽

### II型放出口
- 設於儲槽**底部**，泡沫從底部注入上升
- 又稱「底部注入式」
- 泡沫穿過油層到達表面
- 適用：不易在頂部設置放出口時

### III型（特殊型）
- 設於**外浮頂式儲槽**
- 泡沫注入浮頂與槽壁之間的環形空間
- 適用：外浮頂式儲槽

### 選擇原則

| 儲槽類型 | 適用放出口 |
|---------|----------|
| 固定頂式 | I型（側板上方）|
| 內浮頂式 | I型或II型 |
| **外浮頂式** | **III型（特殊型）** |

## 飛機庫泡沫噴頭

| 項目 | 規定 |
|------|------|
| 放射量 | ≥ **6.5 L/min·m²**（泡水噴頭）|
| 泡沫種類 | AFFF（水成膜）最常用 |
| 設置方式 | 天花板下設置泡沫噴頭 |
| 適用 | 飛機庫、汽車修護廠、停車場 |

## 泡沫設備放射時間

| 場所 | 放射時間 |
|------|---------|
| 停車場 | ≥ **10 分鐘** |
| 飛機庫 | ≥ **10 分鐘** |
| 儲槽 | ≥ **30 分鐘**（大型儲槽更長）|

## 排油煙管滅火設備

| 項目 | 規定 |
|------|------|
| 適用 | 餐廳廚房排油煙管及煙罩 |
| 原理 | 感溫裝置偵測→自動噴射藥劑 |
| 藥劑 | 強化液或乾粉 |
| 風速限制 | 排油煙管內風速 > **5 m/min** 時，需在警戒長度外側設啟動裝置 |
| 連動 | 啟動滅火時自動關閉**閘門**（停止排風）|

## 電動機規定

| 項目 | 規定 |
|------|------|
| 連續運轉 | 額定輸出連續運轉 **8小時** 不得異狀 |
| 絕緣電阻 | 符合用戶用電設備裝置規則 |
| 軸封 | 幫浦軸封不得漏水 |
| 防護等級 | 依場所環境選擇（防水/防塵）|

## 防止水溫上升排放裝置

消防幫浦締切運轉時，水在幫浦內循環→溫度上升→損壞幫浦。

| 項目 | 規定 |
|------|------|
| 功能 | 自動排放過熱水、補入冷水 |
| 排放管口徑 | ≥ **20mm** |
| 啟動 | 水溫上升至設定值自動排放 |

## 記憶口訣

> **「固定頂I型、浮頂III型(特殊型)」**
>
> 飛機庫泡沫：6.5 L/min·m²放射10分鐘
> 排油煙管風速>5m/min→加設啟動裝置
> 電動機連續運轉8小時不得異狀
