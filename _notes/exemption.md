## 免設概念

符合特定條件時，可不設置某些消防設備。常考題型：哪些場所可免設？哪些不行？

## 撒水設備免設

| 免設場所 | 原因 |
|---------|------|
| 浴室、廁所 | 用水場所 |
| 室外通道、露台 | 開放空間 |
| 主要出入口 | 消防搶救需要 |
| 室內停車場設有泡沫滅火 | 替代設備 |
| 樓梯間 | 避難通道 |

## 室內消防栓免設

- 設有**自動撒水設備**的場所
- 已設**水霧**或**泡沫**滅火設備的場所（部分替代）

## 火警探測器免設

| 免設場所 | 原因 |
|---------|------|
| 廁所、浴室 | 蒸氣干擾 |
| 游泳池上方 | 水氣干擾 |
| 室外通道 | 開放空間 |
| 冷藏庫（-10°C以下）| 極低溫不適用 |

## 標示設備免設

- 居室面積 < **400m²** 且出口可直接觀察到者
- 直通主要出口的情況

## 避難器具免設

| 免設條件 |
|---------|
| 設有**兩方向**避難路徑（安全梯2座以上）|
| 樓層人數未達設置門檻 |
| 設有特別安全梯之樓層 |
| 直通安全梯之走廊 |

## 排煙設備免設

| 免設條件 |
|---------|
| 設有**自動撒水設備**且有效防護 |
| 室內面積 < 500m² 且直接開向戶外 |
| 樓層有自然排煙開口（面積≥2%）|

## 減設（設備等級降低）

| 情境 | 減設內容 |
|------|---------|
| 設有撒水 → 消防栓 | 第一種可改為第二種 |
| 設有撒水 → 防火區劃 | 面積可加倍 |
| 防火構造 → 撒水頭 | 間距可放寬 |

## 替代設備

| 原設備 | 可替代為 |
|--------|---------|
| 自動撒水 | 水霧、泡沫（特定場所）|
| 室內消防栓 | 自動撒水（部分替代）|
| 排煙設備 | 自然排煙開口（面積≥2%）|

## 記憶口訣

> **「浴廁泳池免探測，撒水替栓排煙」**
>
> 免設原則：已有替代設備、場所特性不適用
> 設有撒水可以免很多東西（消防栓降級+排煙免設+區劃加倍）
