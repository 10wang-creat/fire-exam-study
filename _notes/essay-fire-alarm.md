## 一、P型 vs R型受信總機（104年考題）

| 項目 | P型 | R型 |
|------|-----|-----|
| **原理** | 探測器動作→回路電流變化→總機判讀分區 | 探測器帶地址→數位信號→總機定位到個別探測器 |
| **定址能力** | ✗ 僅知分區 | ✓ 可定位到單一探測器 |
| **配線量** | 多（每分區獨立回路） | 少（多個探測器共用回路） |
| **回路規模** | 每回路探測器數少 | 每回路可掛數十至百個 |
| **維修成本** | 低（構造簡單） | 高（需專用探測器） |
| **建築物規模** | 中小型 | 大型、高層 |
| **探測器價格** | 便宜 | 較貴（含定址模組） |

### P型配線規定

| 項目 | 規定 |
|------|------|
| 公用線供應分區數 | ≤ **7** 個 |
| 探測器回路電阻 | ≤ **50** Ω |

## 二、受信總機設置規定（110年考題）

| 項目 | 規定 |
|------|------|
| 位置 | **值日室**等經常有人處；設有防災中心者，設於該中心 |
| 壁掛型操作高度 | **0.8** m 以上 **1.5** m 以下（座式 0.6 m 以上） |
| 日光 | **不得**直接照射 |
| 傾斜 | 避免傾斜，外殼應**接地** |
| 前方空間 | ≥ **1.5** m（供檢查操作） |
| 緊急電源 | 蓄電池，容量 ≥ **10** 分鐘有效動作 |

### 蓄積時間

| 設備 | 蓄積時間合計 |
|------|------------|
| 偵煙式探測器分區 | ≤ **60** 秒 |
| 其他探測器分區 | ≤ **20** 秒 |

## 三、火警分區劃定

| 項目 | 規定 |
|------|------|
| 每分區 | ≤ **1** 樓層、≤ **600** m² |
| 觀察全角落者 | 得增至 **1,000** m² |
| 上下二層合計 ≤ 500 m² | 得**二層共用**一分區 |
| 每分區任一邊長 | ≤ **50** m（光電分離型 ≤ **100** m） |
| 樓梯/管道間 | 水平距離 **50** m 內+頂層相差 ≤ 2層 → 一分區 |
| 樓梯垂直 | 每 **45** m 一分區；地下層另一分區 |

---

## 四、分區鳴動方式

### 適用：5F以上且總樓地板面積 ≥ 3,000 m²

| 起火層 | 鳴動範圍 |
|--------|---------|
| 地上2F以上 | 該層 + 直上**2層** + 直下**1層** |
| 地面1F | 該層 + 直上**1層** + 地下**各層** |
| 地下層 | 地面層 + 地下**各層** |
| 10F以上起火 | **全棟**鳴動 |

> 分區鳴動 **10分鐘**內或再收到火災信號 → 立即**全區鳴動**

## 五、探測器選擇——依場所

| 場所特性 | 適用探測器 | 不適用 |
|---------|----------|--------|
| 一般場所 | 差動式/偵煙式 | — |
| 廚房/鍋爐房（高溫） | **定溫式** | 差動式、偵煙式 |
| 結露場所 | **定溫式** | 差動式 |
| 排放廢氣場所 | 定溫式、火焰式 | 偵煙式 |
| 塵埃粉末場所 | 火焰式、定溫式特種 | 偵煙式 |
| 腐蝕性氣體 | **火焰式** | — |
| 大空間 (>10m) | 火焰式、光電分離型、差動分布型 | 局限型 |

### 探測器裝置面高度限制

| 高度 | 可用探測器 |
|------|----------|
| < 4m | 全部可用 |
| 4~8m | 差動式局限型1種、偵煙式1種2種、定溫式特種 |
| 8~15m | 偵煙式1種、差動式分布型、火焰式 |
| 15~20m | 光電式分離型1種、火焰式 |
| > 20m | **火焰式**（唯一） |

## 六、綜合檢查（111年考題）

### 同時動作試驗

**方法**：不復舊狀態下，使任意 **5** 回路（15回路以下場所）或 **全回路1/3**（15回路以上）同時動作

**判定**：
- 火災燈及地區表示燈：應正常亮燈
- 主音響及地區音響：應正常鳴動
- 受信總機不得發生異常

### 綜合動作試驗

**方法**：選取任一回路，使探測器實際動作（加熱/加煙），確認全系統連動

**判定**：
- 探測器動作 → 受信總機火災表示 → 地區音響鳴動
- 連動設備（排煙口、防火門、撒水等）應正常動作
- 緊急廣播應自動啟動

### 二信號式受信總機

| 信號 | 動作 |
|------|------|
| 第一信號 | 火災燈亮 + 副音響（或主音響）鳴動 |
| 第二信號 | 主音響 + 地區音響**全部**鳴動 + 地區表示燈亮 |
