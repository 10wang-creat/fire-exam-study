## 避難層定義

具有出入口直接通往室外之樓層。通常為**1樓（地面層）**，但高層建築可能有多個避難層。

## 避難層的功能

- 人員最終避難匯集處
- 消防人員進入搶救的入口
- 消防車輛接近的樓層

## 避難層設備要求

| 設備 | 要求 |
|------|------|
| 出入口 | 直接通往室外，不得有障礙 |
| 寬度 | 足以容納避難人流 |
| 標示 | 出口標示燈常時亮 |
| 緊急照明 | 設置 |
| 內部裝修 | **不燃材料** |

## 直通樓梯到避難層

| 項目 | 規定 |
|------|------|
| 原則 | 安全梯應直通避難層 |
| 步行距離 | 避難層至室外出口 ≤ 規定距離 |
| 不得阻塞 | 通道上不得堆放物品 |

## 高層建築的避難層

超高層建築（>50m）可能設置**中間避難層**（避難樓層/臨時避難區）：

| 項目 | 規定 |
|------|------|
| 設置間隔 | 通常每15-25層設一處 |
| 功能 | 暫時等待救援，分散避難人流 |
| 設備 | 排煙、廣播、緊急照明 |
| 耐火 | 防火構造 2-3 小時 |

## 避難動線原則

```
居室 → 走廊 → 安全梯 → 避難層 → 室外
                                    ↑
                              直接通往室外
```

- 全程不得有上鎖的門
- 防火門向避難方向開啟
- 避難方向指示燈引導全程

## 記憶口訣

> **「避難層 = 有出口直通室外的樓層」**
>
> 通常是1F，高樓可能有中間避難層
> 安全梯直通避難層，出口不得阻塞
