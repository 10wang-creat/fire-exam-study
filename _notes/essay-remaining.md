## 一、室內消防栓——一般場所 vs 危險物品場所（109年）

| 項目 | 一般場所 第一種 | 一般場所 第二種 | 危險物品場所 |
|------|--------------|--------------|------------|
| **栓口徑** | 38 或 50 mm | 25 mm | 38 或 50 mm |
| **放水壓力** | ≥ **1.7** kgf/cm² | ≥ **2.5** kgf/cm² | ≥ **2.5** kgf/cm² |
| **放水量** | ≥ **130** L/min | ≥ **60** L/min | ≥ **260** L/min |
| **放水壓力上限** | ≤ 7 kgf/cm² | ≤ 7 kgf/cm² | ≤ 7 kgf/cm² |
| **同時使用數** | ≤ 2 支 | ≤ 2 支 | ≤ 4 支 |
| **水源容量** | 2支×30min | 2支×30min | 4支×30min |
| **緊急電源** | 30 min | 30 min | **45 min** |

> 石化場所=危險物品場所，傢俱展示=一般場所低度危險→可用第二種

### 第二種消防栓適用場所
- 低度危險工作場所
- 集合住宅
- 傢俱展示販售場

## 三、簡易自動滅火設備（112年）

### 適用場所
廚房排油煙管及煙罩（含未達自動撒水設備設置規模之場所）

### 設置規定

| 項目 | 規定 |
|------|------|
| 噴頭涵蓋 | 煙罩及排油煙管均須涵蓋 |
| 儲存鋼瓶 | 溫度 ≤ **40°C** |
| 噴頭位置 | 不得因藥劑放射使可燃物飛散 |
| 防護範圍 | 噴頭應**一齊放射** |
| 排油煙管風速 > **5 m/s** | 警戒長度外側設啟動裝置+連動閉鎖閘門 |
| 警戒長度 | 煙罩與排油煙管接合處往內 **10 m** |
| 手動啟動 | 設於廚房出入口附近 |

### 性能檢查（蓄壓式）
- 藥劑量與標示差異 ≤ **5%**
- 壓力錶指針在綠色範圍

## 四、標示設備——光度與照度（105年）

### 出口標示燈等級與光度

| 等級 | 標示面光度 | 步行有效距離 |
|------|----------|------------|
| **A級** | ≥ **50** cd | 60 m |
| **B級** | ≥ **10** cd | 30 m |
| **C級** | ≥ **1.5** cd | 15 m |

> 不易看清時，有效範圍減為 **10** m

### 避難方向指示燈

| 等級 | 光度 | 步行有效距離 |
|------|------|------------|
| A級 | ≥ 25 cd | 20 m |
| B級 | ≥ 13 cd | 15 m |
| C級 | ≥ 5 cd | 10 m |

### 樓梯/坡道照度

避難方向指示燈設於樓梯或坡道者，樓梯級面/坡道表面照度 ≥ **1 lx**（勒克司）

### 觀眾席引導燈

通道地面水平照度 ≥ **0.2 lx**

### 認可基準要點

| 項目 | 規定 |
|------|------|
| 平均亮度 | 綠色部分/白色部分均需符合 |
| 亮度比 | 綠色:白色 ≤ **10:1** 且 ≥ **1:10** |
| 緊急電源持續 | ≥ **20** 分鐘（內藏蓄電池） |

## 五、耐熱保護 vs 耐燃保護（106、113年）

### 定義

| 保護等級 | 說明 | 承受溫度 |
|---------|------|---------|
| **耐燃保護** | 火災中能維持電路不短路 | 380°C × 15min |
| **耐熱保護** | 火災中能維持電路功能 | 840°C × 30min |

> 耐熱 > 耐燃（耐熱保護等級較高）

### 各設備配線保護等級

| 配線區間 | 保護等級 |
|---------|---------|
| **電源回路**（受信總機至電源） | **耐燃** |
| **受信總機→探測器**（一般） | **耐熱** |
| **受信總機→定址式探測器** | **耐燃** |
| **受信總機→火警警鈴** | **耐熱** |
| **擴音機→揚聲器** | **耐熱** |
| **排煙設備→排煙口** | **耐燃** |
| **滅火設備控制盤→啟動裝置** | **耐燃** |
| **緊急照明（非內置電池）** | **耐燃** |

### 替代方式

| 方式 | 條件 |
|------|------|
| MI電纜 | 可取代耐燃/耐熱 |
| 金屬導線管+**20mm**混凝土 | 取代**耐燃** |
| 金屬導線管+**15mm**混凝土 | 取代**耐熱** |
| 600V耐熱電線 | 用於緊急電源回路/操作回路 |

## 六、系統架構方塊圖（113年）

### 火警自動警報設備

```
探測器 ──→ 受信總機 ──→ 地區音響
                    ├──→ 火災表示
                    ├──→ 緊急廣播（連動）
                    ├──→ 排煙設備（連動）
                    └──→ 防火門/撒水（連動）
```

### 緊急廣播設備

```
啟動裝置 ──→ 擴音機 ──→ 揚聲器（各層分區）
            ↑
      火警自動警報連動
```

### 排煙設備

```
手動開關 ──→ 排煙口開啟 ──→ 排煙機連動啟動
  ↑
探測器連動
```
