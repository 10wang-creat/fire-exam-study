## 一、揚聲器等級與音壓

| 等級 | 距1m處音壓 | 適用廣播區域面積 |
|------|----------|----------------|
| **L級** | ≥ **92** dB | > 100 m² |
| **M級** | ≥ **87** dB（<92dB） | 50~100 m² |
| **S級** | ≥ **84** dB（<87dB） | ≤ 50 m² |

### 廣播區域面積與揚聲器對應

| 面積 | 可設揚聲器 |
|------|----------|
| ≤ 50 m² | L、M 或 S 級 |
| 50~100 m² | L 或 M 級 |
| > 100 m² | 限 **L 級** |

> 性能設計時可不受 10m 限制，但需標注音響功率位準及指向特性

---

## 二、揚聲器設置規定

| 項目 | 規定 |
|------|------|
| 水平距離 | 各廣播區域任一點至揚聲器 ≤ **10** m |
| 樓梯/斜坡 | 至少垂直距離每 **15** m 設一個 L 或 M 級 |
| 配線 | 任一層短路或斷線 → **不得影響**其他樓層 |
| 音量調整器 | 設有時應為 **三線式**配線 |

---

## 三、廣播分區

| 項目 | 規定 |
|------|------|
| 每一分區 | ≤ **1** 樓層 |
| 安全梯/特別安全梯 | 垂直距離每 **45** m 單獨一分區 |
| 安全梯地下層部分 | **另設**一分區 |
| 挑空構造 | 音壓符合規定 → 得為一分區 |

---

## 四、擴音機及操作裝置（105年考題）

### 擴音機設置規定

| 項目 | 規定 |
|------|------|
| 位置 | 值日室等經常有人處；設有防災中心者，設於該中心 |
| 操作開關高度 | **0.8** m 以上 **1.5** m 以下（座式 0.6 m 以上） |
| 選擇功能 | 具有選擇**必要樓層或區域**廣播之性能 |
| 短路標示 | 各廣播分區配線有短路時，應有短路信號之標示 |
| 與火警連動 | 火警信號 → 自動切換為緊急廣播 |
| 緊急電源 | 蓄電池，容量 ≥ **10** 分鐘有效動作 |

### 操作裝置其他規定

- 與其他設備共用 → 火災時應能**遮斷其他廣播**
- 音響警報以**語音方式**播放
- 設有**麥克風**可直接廣播

---

## 五、啟動裝置（109年考題）

### 設置規定

| 項目 | 規定 |
|------|------|
| 步行距離 | 各樓層任一點至啟動裝置 ≤ **50** m |
| 高度 | 距樓地板 **0.8** m 以上 **1.5** m 以下 |
| 外觀 | 應設於明顯易見、操作容易之處 |
| 標示燈 | 與裝置面成 **15°** 角，在 **10** m 距離內明顯可見 |

### 啟動裝置性能檢查

**檢查方法**：
1. 操作啟動裝置，確認擴音機啟動
2. 確認語音警報播放正常
3. 確認選擇樓層廣播功能
4. 確認與火警自動警報設備之連動

**判定方法**：
- 啟動裝置操作後，擴音機應確實啟動
- 所選擇區域之揚聲器應正常播放
- 音壓符合各等級規定
- 連動時應自動遮斷其他廣播

## 六、揚聲器斷線影響（109年考題）

### 三線式配線優勢

```
三線式配線：
擴音機 ──── 公共線(C) ──────── 揚聲器群
       ──── 音量調整線(V) ─┤
       ──── 信號線(L) ────┘

→ 某層斷線時，其他層仍可正常廣播
```

### 斷線影響

| 情形 | 影響 |
|------|------|
| 某層揚聲器配線斷線 | **該層無法廣播** |
| 其他樓層 | **不受影響**（獨立回路） |
| 短路 | 擴音機應有**短路信號標示** |

> 法規要求：「任一層之揚聲器或配線有短路或斷線時，**不得影響其他樓層之廣播**」

---

## 七、綜合檢查（105年考題）

### 檢查方法

1. 使用**緊急電源**狀態
2. 操作啟動裝置或由火警自動警報連動
3. 確認各區域揚聲器音壓
4. 確認語音警報內容清晰
5. 確認選擇廣播功能

### 判定方法

| 項目 | 合格標準 |
|------|---------|
| 音壓 | L≥92, M≥87, S≥84 dB |
| 語音 | 清晰可辨 |
| 連動 | 火警信號後自動啟動 |
| 選擇 | 可正確選擇樓層廣播 |
| 遮斷 | 緊急時自動遮斷一般廣播 |

---

## 八、配線規定

| 項目 | 規定 |
|------|------|
| 配線方式 | 依屋內線路裝置規則 |
| 不得共用管槽 | 但弱電回路（60V以下）可共用 |
| 絕緣電阻 | 直流 250V 測定 |
| 對地電壓 ≤ 150V | ≥ **0.1** MΩ |
| 對地電壓 > 150V | ≥ **0.2** MΩ |
