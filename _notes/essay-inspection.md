## 一、三種檢查層次——萬用答題架構

```
┌──────────────────────────────┐
│  外觀檢查（目視+簡易工具）      │
│  → 看：變形/損傷/腐蝕/標示      │
│  → 確認：位置/周圍/障礙物       │
├──────────────────────────────┤
│  性能檢查（操作單一設備）        │
│  → 測：壓力/藥劑量/動作時間     │
│  → 計器：壓力錶/秤/加熱器       │
├──────────────────────────────┤
│  綜合檢查（全系統模擬火災）      │
│  → 連動：探測→受信→警報→滅火   │
│  → 測：放水壓力/放射量/時間     │
└──────────────────────────────┘
```

> **答題公式**：每種檢查都答「①檢查方法 ②判定方法 ③注意事項」三段。

## 二、外觀檢查——通用判定要點

不論什麼設備，外觀檢查的判定項目幾乎一致：

| 判定項目 | 合格標準 |
|---------|---------|
| 外形 | 無變形、損傷、顯著腐蝕 |
| 標示 | 清晰無污損，字體完整可辨 |
| 周圍狀況 | 無檢查或使用上之障礙物 |
| 配管 | 無洩漏、變形、損傷 |
| 閥類 | 開關位置正確（標示方向） |
| 壓力錶（蓄壓式）| 指針在綠色範圍內 |
| 設置位置 | 符合法規距離（高度/間距）|

## 三、性能檢查——各設備關鍵數字速查

### 3-1 水系統

| 設備 | 檢查項目 | 合格數值 |
|------|---------|---------|
| 第一種室內消防栓 | 放水壓力 | **1.7** kgf/cm² 以上、7 以下 |
| 第二種室內消防栓 | 放水壓力 | **2.5** kgf/cm² 以上、7 以下 |
| 危險物品場所消防栓 | 放水壓力 | **2.5** kgf/cm² 以上、7 以下 |
| 密閉式撒水（幫浦）| 末端查驗閥放水壓力 | **1** kgf/cm² 以上（幫浦啟動）|
| 補助撒水栓 | 放水壓力 | **1** kgf/cm² 以上、**6** 以下 |
| 水霧（一般）| 噴頭放水壓力 | **2.7** kgf/cm² 以上 |
| 水霧（電氣）| 噴頭放水壓力 | **3.5** kgf/cm² 以上 |
| 水道連結型撒水 | 放水壓力 | **0.5** kgf/cm² 以上、**10** 以下 |

### 3-2 化學系統

| 設備 | 檢查項目 | 合格數值 |
|------|---------|---------|
| CO2 音響警報 | 距1m音壓 | ≥ **90** dB |
| CO2 遲延裝置 | 動作時限 | ≥ **20** 秒 |
| 乾粉藥劑 | 暫停檢查條件 | 溫度 >**40**°C 或 濕度 >**60**% |
| 乾粉藥劑 | 手握搓揉落下（高度）| **50** cm，應呈粉狀 |
| 泡沫 25%還原時間 | 蛋白質 | ≥ **120** 秒 |
| 泡沫 25%還原時間 | 合成界面活性 | ≥ **60** 秒 |
| 泡沫 25%還原時間 | 水成膜 | ≥ **60** 秒 |
| 鹵化烴藥劑量 | 與充填值差異 | ≤ **10**% |
| 簡易自動滅火（蓄壓式）| 藥劑量與標示差異 | ≤ **5**% |

### 3-3 警報系統

| 設備 | 檢查項目 | 合格數值 |
|------|---------|---------|

| 火焰式 | 動作試驗器 | ≤ **30** 秒 |
| 地區音響裝置 | 距1m音壓 | ≥ **70** dB |
| 瓦斯檢知區域警報 | 距1m音壓 | ≥ **70** dB |

## 四、綜合檢查——各設備放射試驗速查

| 設備 | 綜合檢查方法 | 關鍵數字 |
|------|------------|---------|
| 密閉濕式撒水 | 末端查驗閥放水 | 壓力 1~7 kgf/cm² |
| 泡沫（低發泡固定式）| 逐區放水試驗 | 每次選 ≥**10%** 放射區域 |
| CO2（高壓式）| 放射試驗 | 瓶數 × **5%** 以上（進1） |
| CO2（低壓式）| 放射試驗 | 藥劑量 × **5%** 以上 |
| 乾粉 | 以空氣或N2替代放射 | — |
| 鹵化烴（高壓）| 以空氣或N2放射 | 藥劑量 × **10%** 核算 |
| 排煙設備 | 排煙口開啟→排煙機連動 | 運轉電流紀錄 |
| 瓦斯漏氣 | 加瓦斯試驗器→受信總機 | 60秒/65秒/80秒/85秒內 |

## 六、申論答題模板

**題目範例**：「請說明○○設備綜合檢查之檢查方法、判定方法及注意事項」

**作答架構**：

```
一、檢查方法
   1. 使用正常電源/緊急電源供電狀態
   2. 藉由手動/自動啟動裝置操作
   3. 確認系統之性能是否正常
   4. （設備特有步驟）

二、判定方法
   1. ○○裝置應正常動作
   2. 壓力/放射量/音壓/照度應在規定值以上
   3. 各設備連動應正常
   4. 無異常聲音、振動

三、注意事項
   1. 有困難之場所得使用常用電源（醫院等）
   2. 應事先通知管理權人及相關人員
   3. 檢查後應確實復歸
   4. 紀錄運轉電流等數據
```
