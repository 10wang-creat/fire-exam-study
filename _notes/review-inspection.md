## 法源

消防法第 **10 條**：供公眾使用建築物之消防安全設備圖說，應由直轄市、縣（市）消防機關於主管建築機關許可開工前，審查完畢。

## 審查與查驗流程

```
建築設計 → 消防圖說審查 → 許可開工 → 施工 → 竣工查驗 → 核發使用執照
```

### 1. 消防圖說審查（開工前）

| 項目 | 說明 |
|------|------|
| 時機 | 建築許可開工**前** |
| 審查機關 | 直轄市/縣市**消防機關** |
| 審查內容 | 消防安全設備設計是否符合設置標準 |
| 圖說簽證 | 須由**消防設備師**簽證 |

### 2. 竣工查驗（使用前）

| 項目 | 說明 |
|------|------|
| 時機 | 建築完工、核發使用執照**前** |
| 查驗機關 | 直轄市/縣市**消防機關** |
| 查驗內容 | 實際設備是否與核准圖說相符 |
| 不合格 | 不得核發使用執照 |

## 簽證制度

| 項目 | 消防設備師 | 消防設備士 |
|------|----------|----------|
| 圖說設計簽證 | ✅ 可以 | ❌ 不可以 |
| 竣工查驗簽證 | ✅ 可以 | ❌ 不可以 |
| 檢修申報簽證 | ✅ 可以 | ✅ 一定規模以下 |

## 變更設計

- 消防設備變更設計時，須重新送審
- 未經審查擅自變更 → 罰鍰

## 記憶口訣

> **「開工前審圖，完工前查驗」**
>
> 圖說審查只有消防設備**師**能簽
> 審查+查驗都是消防機關負責
