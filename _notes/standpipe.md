## 功能

連結送水管是供**消防人員**使用的滅火設備，消防車從建築物外的送水口加壓送水，消防人員在各樓層出水口接水帶滅火。

## 設置條件

- **7層以上**建築物
- **地下層3層以上**之建築物
- 建築物高度超過一定高度者

## 乾式 vs 濕式

| 項目 | 乾式 | 濕式 |
|------|------|------|
| 平時管內 | **空管** | **充水** |
| 加壓方式 | 消防車從送水口加壓 | 建築物自有加壓幫浦+消防車 |
| 適用 | 一般7層以上 | 11層以上或特殊場所 |
| 優點 | 簡單、無凍結問題 | 反應較快 |

## 組成

| 組件 | 說明 |
|------|------|
| **送水口** | 設於建築物外牆，消防車接管處 |
| 立管 | 垂直配管，貫穿各樓層 |
| **出水口** | 設於各樓層，消防員接水帶處 |
| 閥門 | 各層出水口設止水閥 |

## 送水口規定

| 項目 | 規定 |
|------|------|
| 口徑 | **63mm** 雙口陽式快速接頭 |
| 離地高度 | **1m 以下** |
| 位置 | 消防車容易接近之外牆面 |
| 標示 | 「送水口」標示 |

## 出水口規定

| 項目 | 規定 |
|------|------|
| 口徑 | 63mm |
| 離地高度 | 0.5m-1.5m |
| 設置位置 | 樓梯間或其附近 |
| 標示 | 「出水口」標示 |

## 記憶口訣

> **「7層以上設連結送水管」**
>
> 乾式=空管靠消防車送水
> 濕式=預充水+幫浦
> 送水口63mm在外牆1m以下
