## 法源

消防法第 **11 條**：地面樓層達11層以上建築物、地下建築物及中央主管機關指定之場所，其管理權人應使用附有防焰標示之地毯、窗簾、布幕、展示用廣告板及其他指定之防焰物品。

## 應使用防焰物品的場所

- 地面樓層達 **11層以上** 建築物
- **地下建築物**
- 中央主管機關指定場所（電影院、KTV、醫院等甲類場所）

## 防焰物品種類

| 物品 | 說明 |
|------|------|
| 地毯 | 含地墊、人工草皮 |
| 窗簾 | 含布簾、紗簾 |
| 布幕 | 舞台布幕、隔間布幕 |
| 展示用廣告板 | 合板、纖維板 |
| 網目狀窗簾 | 施工用帆布、防塵網 |
| 其他指定物品 | 沙發、床墊布料等 |

## 防焰性能標準

防焰物品須通過**燃燒試驗**：
- 餘焰時間 ≤ **3秒**（火焰移開後殘餘燃燒時間）
- 餘燼時間 ≤ **5秒**（無焰燃燒時間）
- 碳化面積 ≤ **30 cm²**
- 碳化距離 ≤ **20 cm**

## 防焰標示

- 經認證的防焰物品貼有**防焰標示**
- 標示由內政部消防署核發
- 未貼防焰標示者不得使用

## 罰則

- 未使用防焰物品：**1-5萬元**罰鍰
- 經通知改善仍未改善：得連續處罰
