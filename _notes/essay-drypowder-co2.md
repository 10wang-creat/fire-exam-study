## 一、乾粉滅火藥劑四種分類（114年考題核心）

| 種類 | 主成分 | 顏色 | 適用火災 | 化學式 |
|------|--------|------|---------|--------|
| **第一種** | 碳酸氫鈉 NaHCO₃ | 白色/淡藍 | BC | 2NaHCO₃ → Na₂CO₃ + H₂O + CO₂ |
| **第二種** | 碳酸氫鉀 KHCO₃ | 紫色系 | BC | 2KHCO₃ → K₂CO₃ + H₂O + CO₂ |
| **第三種** | 磷酸二氫銨 NH₄H₂PO₄ | 粉紅色 | **ABC** | NH₄H₂PO₄ → HPO₃ + NH₃ + H₂O |
| **第四種** | 碳酸氫鉀+尿素 | 灰色 | BC | — |

> **室內停車場** → 限用第三種乾粉（磷酸二氫銨，ABC通用）

### 滅火原理（114年考題）

1. **窒息作用**：分解產生 CO₂ 及水蒸氣排擠氧氣
2. **冷卻作用**：吸熱分解降溫
3. **抑制作用（負催化）**：鉀/鈉離子捕捉自由基，中斷連鎖反應 ← **最主要機制**
4. **遮斷輻射**：粉末雲遮斷熱輻射回饋

## 二、加壓式乾粉系統裝置（107年考題核心）

```
加壓用N2/CO2鋼瓶
    ↓
 容器閥 ──→ 壓力調整裝置（調壓至≤25 kgf/cm²）
    ↓
 ┌──┤
 │  遲延裝置（延遲放出，讓人撤離）
 │  ↓
 │  定壓動作裝置（壓力達設定值才開啟放出閥）
 │  ↓
 │  放出閥 → 配管 → 噴頭
 │
 └─ 排出裝置（放射後清除管內殘留藥劑）
    └─ 清洗裝置（用N2清洗配管）
```

### 各裝置作用

| 裝置 | 作用 |
|------|------|
| **遲延裝置** | 啟動後延遲放射，確保人員撤離 |
| **定壓動作裝置** | 確保儲存容器壓力達設定值後才開啟放出閥，保證藥劑均勻放射 |
| **壓力調整裝置** | 調整加壓氣體壓力至 ≤ **25** kgf/cm²，防止管路過壓 |
| **清洗裝置** | 放射後以N₂或空氣清洗配管內殘留藥劑 |
| **排出裝置** | 將配管內殘壓及殘留氣體排出 |

### 定壓動作裝置類型

| 類型 | 原理 |
|------|------|
| **膜片式** | 壓力達設定值→膜片破裂→開啟放出閥 |
| **彈簧式** | 壓力達設定值→克服彈簧力→開啟 |

| 種類 | 每m³防護區域 | 每m²開口追加 |
|------|------------|------------|
| 第一種 | **0.60** kg/m³ | **4.5** kg/m² |

## 四、CO2 全區放射藥劑量（105年考題）

### 基本設計濃度

| 場所 | 設計濃度 | 每m³藥劑量 |
|------|---------|----------|
| 電信機械室 | 50% | **1.0** kg/m³ |
| 電氣設備 | 50% | **1.0** kg/m³ |
| 鍋爐房（可燃液體） | 50% | **1.0** kg/m³ |
| 印刷機房 | 65% | **1.6** kg/m³ |
| 總機室 | 50% | **1.0** kg/m³ |

### 計算公式

基本體積因素 Kb = 2.66/(S+1)

其中 S = V_a/V_s（CO2體積/空間體積）

| 設計濃度(C) | 體積因素 |
|------------|---------|
| 34% | 0.72 kg/m³ |
| 50% | 1.00 kg/m³ |
| 66% | **約 1.95 kg/m³** |

> **105年計算**：防護乙炔，C=66%
> 每m³所需CO₂ = 2.66 × (66/(100-66)) = 2.66 × 1.94 ≒ **5.16 kg/m³**
> （或依表查基本量+開口追加量計算）

### 放射時間

| 場所 | 時間 |
|------|------|
| 電信機械室 | ≤ **3.5** 分鐘（210 秒） |
| 總機室 | ≤ **3.5** 分鐘 |
| 其他全區 | ≤ **1** 分鐘 |
| 局部放射 | ≤ **30** 秒 |

## 五、啟動裝置手動/自動切換（106年考題）

### 設置規定

| 項目 | 規定 |
|------|------|
| 手動啟動 | 操作一個操作裝置即可啟動 |
| 自動啟動 | 探測器動作 → 遲延 → 自動放射 |
| 切換裝置 | 設於防護區域外之操作箱 |
| 切換位置 | 明確標示「自動」/「手動」 |
| 平時狀態 | 依場所使用型態決定 |

### 遲延裝置

| 系統 | 遲延時間 |
|------|---------|
| CO2 全區放射 | ≥ **20** 秒 |
| 乾粉 | 依設計 |
| 鹵化烴 | ≥ **20** 秒 |

### 切換裝置性能檢查

1. 確認切換開關操作容易
2. 自動位置：探測器動作→遲延→容器閥開→放射
3. 手動位置：探測器動作→僅警報→**不放射**
4. 手動操作啟動裝置→放射
5. 切換標示清晰可辨

## 六、CO2 綜合檢查安全注意事項（110年考題核心）

### 檢查方式（高壓式）

1. **拆除**：先拆電磁閥→再拆操作管→最後拆容器閥
2. **放射試驗**：瓶數 × **5%** 以上（小數進1）
3. 替代試驗：以空氣或N₂進行，量= 藥劑量 × 5%，每公斤核算 **45 L**

### 檢查前準備

| 項目 | 內容 |
|------|------|
| 人員疏散 | 確認防護區域內**無人** |
| 通知管理權人 | 事先書面通知 |
| 拆除順序 | 電磁閥→操作管→氣動式開放裝置 |
| 門窗 | 確認自動關閉裝置正常 |
| 通風換氣 | 試驗後開啟排放裝置 |

### 檢查中注意

| 項目 | 內容 |
|------|------|
| **禁止進入** | 放射中及放射後防護區域**嚴禁進入** |
| 氧氣濃度 | 放射後 CO2 會排擠氧氣，可能致命 |
| 排放 | 機械排放：排風機排氣量 ≥ **5次/hr** 換氣 |
| 自然排放 | 面向外氣開口（樓地板面2/3以下）≥ 面積 **10%** |
| 復歸 | 試驗後確實復歸所有裝置 |
| 記錄 | 記錄放射壓力、時間等數據 |

## 七、乾粉藥劑性能檢查（109年考題）

### 藥劑量檢查方法

| 儲存方式 | 方法 |
|---------|------|
| 蓄壓式 | 以台秤測重量 |
| 加壓式 | 以台秤測重量或目視液面計 |

### 判定方法

- 藥劑量與設計量差異 ≤ **充填值 5%**
- 無變質、固化、結塊
- 手握搓揉，自 **50 cm** 高度落下應呈粉狀

### 注意事項

- 溫度 > **40°C** 或 濕度 > **60%** → **暫停檢查**
- 防潮處理
- 不得以有機溶劑清理
