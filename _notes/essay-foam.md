| 年份 | 題目 |

| 跨年 | 多次選擇題考泡沫種類/濃度/還原時間數值 |

## 一、泡沫原液種類與濃度

| 泡沫原液 | 混合濃度 | 特性 |
|---------|---------|------|
| **蛋白質** | 3% 或 6% | 耐熱性佳，穩定性好 |
| **合成界面活性** | 3% 或 6% | 流動性好，發泡快 |
| **水成膜** | 3% 或 6% | 在油面形成薄膜，隔絕最佳 |
| **酒精型** | 依廠牌 | 耐醇類溶劑 |

---

## 二、發泡倍率分類

| 類型 | 發泡倍率 | 適用 |
|------|---------|------|
| **低發泡** | < 20 倍 | 一般場所、停車場 |
| **中發泡** | 20~200 倍 | — |
| **高發泡** | 80~1,000 倍 | 大空間全區放射 |

---

## 三、綜合檢查——泡沫試料採集（108年考題核心）

### 3-1 試料採集方法

1. 選擇放射區域進行**放水試驗**
   - 設置泡沫頭者：每次選 ≥ **10%** 放射區域
2. 在泡沫頭下方放置**採集器**（金屬盤等）
3. 採集足夠量之泡沫試料
4. 同時採集泡沫噴頭噴出之**泡沫水溶液**（用於測混合比率）

### 3-2 發泡倍率測定

**定義**：發泡倍率 = 泡沫體積 ÷ 泡沫水溶液體積

**測定方法**：

```
① 準備量筒（1,000 mL 具刻度）
② 將採集之泡沫裝入量筒至標線
③ 靜置後量測泡沫水溶液析出之體積（V液）
④ 計算：
   發泡倍率 = 量筒容積(1000mL) ÷ V液(mL)
```

### 發泡倍率合格標準

| 泡沫種類 | 合格標準 |
|---------|---------|
| 蛋白質泡沫 | ≥ **5** 倍 |
| 合成界面活性 | ≥ **5** 倍 |
| 水成膜泡沫 | ≥ **5** 倍 |
| 移動式泡沫 | ≥ **5** 倍 |

### 3-3  25%還原時間測定

**定義**：泡沫體積的 25% 還原（析出）為水溶液所需的時間

**測定方法**：

```
① 將泡沫裝入量筒至標線（1,000 mL）
② 開始計時
③ 觀察底部析出之水溶液量
④ 當析出量 = 量筒容積 × 25% = 250 mL 時
⑤ 記錄經過時間 = 25% 還原時間
```

### 25% 還原時間合格標準

| 泡沫種類 | 合格標準 |
|---------|---------|
| **蛋白質**泡沫 | ≥ **120** 秒（2 分鐘） |
| **合成界面活性** | ≥ **60** 秒（1 分鐘） |
| **水成膜** | ≥ **60** 秒（1 分鐘） |

---

## 四、混合比率測定

確認泡沫原液與水之混合比例是否正確

| 測定方法 | 說明 |
|---------|------|
| **糖度計法** | 以糖度計（折射計）測定泡沫水溶液之折射率 |
| **比色計法** | 比較泡沫水溶液顏色與標準色 |

---

## 五、泡沫頭放射量

| 泡沫原液 | 每 m² 放射量 |
|---------|------------|
| 蛋白質 | ≥ **6.5** L/(min·m²) |
| 合成界面活性 | ≥ **8** L/(min·m²) |
| 水成膜 | ≥ **3.7** L/(min·m²) |

---

## 六、固定式低發泡泡沫綜合檢查

### 放水試驗

| 項目 | 規定 |
|------|------|
| 區域選擇 | 每次 ≥ **10%** 放射區域 |
| 測定項目 | 放射分布、放射壓力 |
| 泡沫品質 | 發泡倍率+25%還原時間 |
| 混合比率 | 糖度計法或比色計法 |

### 判定方法

- 泡沫應均勻覆蓋防護面積
- 發泡倍率 ≥ 5 倍
- 25%還原時間符合各原液標準
- 混合比率在規定濃度範圍內
- 加壓送水裝置正常動作
- 一齊開放閥正常開啟

---

## 七、移動式泡沫滅火設備

| 項目 | 規定 |
|------|------|
| 瞄子放射量 | ≥ **130** L/min（室內）/ **200** L/min（室外） |
| 放射壓力 | ≥ **3.5** kgf/cm² |
| 水帶接頭至防護對象 | ≤ **15** m |
| 水源容量 | 2具瞄子同時放水 × **15** 分鐘 |
| 箱內配置 | 15m 水帶 2 條 + 泡沫瞄子 1 具 |

---

## 八、泡沫原液儲槽性能檢查

| 項目 | 判定 |
|------|------|
| 液面計 | 應能確認藥劑量 |
| 壓力錶（加壓式） | 指示正常 |
| 泡沫原液採樣 | 上/中/下三處，無變質/污損 |
| 溫度 | ≤ **40°C**，無日光曝曬 |
| 防震 | 有效防震措施 |

## 記憶口訣

> **「發泡5倍起跳」**——三種泡沫都要≥5倍
> **「蛋白120秒、水成膜合成60秒」**——25%還原時間
> **「每次選10%區域放水試驗」**——綜合檢查
> **「蛋白6.5、合成8、水成膜3.7」**——每m²放射量
> **「移動式130/200、壓力3.5、距離15m、放水15min」**
