## 第一種 vs 第二種比較

| 項目 | 第一種 | 第二種 |
|------|--------|--------|
| **操作人數** | **2人** | **1人** |
| 水帶口徑 | 38mm 或 50mm | **25mm** |
| 水帶長度 | 15m | 15m 或 20m |
| 瞄子口徑 | 13mm | 8mm |
| **放水量** | ≥ **130 L/min** | ≥ **60 L/min** |
| **放水壓力** | ≥ 1.7 kgf/cm²（0.17 MPa）| ≥ 2.5 kgf/cm²（0.25 MPa）|
| 同時使用數 | **2支** | **2支** |
| 水源容量 | ≥ 130×2×20min = **5.2 m³** | ≥ 60×2×20min = **2.4 m³** |
| 適用場所 | 一般場所 | 11樓以上、地下層等 |

## 設置規定

| 項目 | 規定 |
|------|------|
| 水帶接頭高度 | 離地面 **0.3m-1.5m** |
| 步行距離 | 任何位置 ≤ **25m**（水帶有效範圍）|
| 箱面標示 | 紅色「消防栓」字樣 |
| 啟動裝置 | 箱旁設啟動表示燈（紅色）|

## 水源容量計算

`水源 = 放水量 × 同時使用數 × 20 分鐘`

| 類型 | 計算 |
|------|------|
| 第一種 | 130 × 2 × 20 = **5,200 L = 5.2 m³** |
| 第二種 | 60 × 2 × 20 = **2,400 L = 2.4 m³** |

## 幫浦（加壓送水裝置）

- 出水量 ≥ 同時使用消防栓數量的放水量
- 第一種：≥ 130 × 2 = 260 L/min
- 第二種：≥ 60 × 2 = 120 L/min

## 使用步驟

1. 按下箱旁啟動按鈕（啟動幫浦）
2. 打開消防栓箱門
3. 取出水帶，展開
4. 接上瞄子
5. 打開閥門
6. 射水滅火

## 記憶口訣

> **「一種雙人130，二種單人60」**
>
> 第一種：2人操作、38/50mm、130 L/min、水源5.2噸
> 第二種：1人操作、25mm、60 L/min、水源2.4噸
> 都是同時2支、持續20分鐘
