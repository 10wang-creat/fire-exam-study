## 高層建築定義

- 高度超過 **50m** 或 **16層以上** 之建築物

## 火災特性

### 1. 煙囪效應嚴重
- 電梯井、樓梯間等垂直管道形成煙囪
- 煙垂直上升速度 3-5 m/s
- 幾分鐘內煙可到達頂層

### 2. 避難困難
- 人員疏散距離長、時間久
- 行動不便者難以走樓梯
- 避難方向與煙流方向衝突（人往下、煙往上）
- 恐慌造成推擠

### 3. 消防搶救困難
- 雲梯車最高約 50m（約15-16層）
- 超過此高度需靠**室內消防設備**
- 水壓送至高層需大揚程幫浦
- 消防員體力消耗極大

### 4. 火勢蔓延快
- 幕牆玻璃破裂 → 火焰沿外牆向上延燒
- 管道間、線槽成為火勢蔓延路徑
- 風速隨高度增加 → 助長火勢

## 防護對策

| 對策 | 功能 |
|------|------|
| 特別安全梯+排煙室 | 防煙避難通道 |
| 樓梯間正壓防煙 | 阻止煙進入逃生梯 |
| 防火區劃 | 阻止火勢水平蔓延 |
| 連結送水管 | 供消防員高樓滅火用水 |
| 緊急昇降機 | 消防人員專用電梯 |
| 自動撒水設備 | 初期滅火 |
| 緊急廣播 | 分區引導避難 |

## 記憶口訣

> **「高樓四難：煙快、逃慢、救不到、火沿牆」**
>
> 雲梯車極限50m，超過靠室內設備
> 煙3-5m/s上升，人跑不贏
