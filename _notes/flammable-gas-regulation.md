## 法源

公共危險物品**及可燃性高壓氣體**製造儲存處理場所設置標準暨安全管理辦法。

## 定義

### 高壓氣體
- 常溫下壓力 ≥ **10 kgf/cm²** 的壓縮氣體
- 溫度35°C時壓力 ≥ 10 kgf/cm² 的壓縮氣體
- 液化氣體：常溫下壓力 ≥ 2 kgf/cm² 的液化氣體

### 可燃性高壓氣體
高壓氣體中具有可燃性者：氫氣、甲烷、乙烷、丙烷、丁烷、乙烯、乙炔等。

## 場所分類

| 場所 | 定義 |
|------|------|
| 製造場所 | 壓縮、液化等製造高壓氣體之場所 |
| 儲存場所 | 儲存高壓氣體之場所 |
| 處理場所 | 使用高壓氣體進行加工處理之場所 |
| 販賣場所 | 販賣高壓氣體之場所 |

## 設置規定重點

### 安全距離

| 儲存量 | 與第一類保護物距離 | 與第二類保護物距離 |
|--------|-----------------|-----------------|
| 較小量 | 依規定距離 | 依規定距離 |
| 較大量 | 距離加大 | 距離加大 |

### 構造規定

| 項目 | 規定 |
|------|------|
| 屋頂 | 輕質構造（洩爆用）|
| 地板 | 防止氣體滯留 |
| 通風 | 設置充分通風設備 |
| 電氣 | **防爆型**電氣設備 |
| 照明 | 防爆型照明 |

### 防爆型電氣設備
在可能有可燃性氣體洩漏的場所，所有電氣設備須為**防爆構造**，避免電火花引燃氣體。

## 安全管理

| 項目 | 規定 |
|------|------|
| 標示 | 設置「可燃性高壓氣體」警告標示 |
| 滅火器 | 設置適當滅火器 |
| 漏氣檢知 | 設置瓦斯漏氣檢知警報設備 |
| 防止洩漏 | 定期檢查管路、閥門 |
| 火源管制 | 嚴禁煙火、明火作業須許可 |
| 緊急處置 | 訂定緊急應變計畫 |

## 容器（鋼瓶）管理

| 項目 | 規定 |
|------|------|
| 檢驗 | 定期送檢驗機構 |
| 合格標示 | 須有檢驗合格標示 |
| 堆放 | 防止傾倒、碰撞 |
| 溫度 | 避免高溫（≤40°C）|
| 通風 | 儲放處保持通風 |
| 分離 | 可燃性與氧化性氣體**分開儲放** |

## 與LPG的關係

LPG（液化石油氣）是可燃性高壓氣體的一種，但消防法第15-1條另有特別規定。

## 記憶口訣

> **「高壓氣體10 kgf/cm²，可燃的要防爆」**
>
> 防爆電氣+充分通風+漏氣警報=三大安全要求
> 可燃與氧化性氣體必須分開放
