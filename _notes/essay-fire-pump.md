## 一、消防幫浦性能規定

### 出水量

| 設備 | 最低出水量 |
|------|----------|
| 第一種室內消防栓 | 全部栓同時使用（≤2支×130 L/min = **260** L/min） |
| 第二種室內消防栓 | 最多2支×60 L/min = **120** L/min |
| 室外消防栓 | 最多2支×350 L/min = **700** L/min |
| 自動撒水（10F↓一般） | 15個×80 L/min = **1,200** L/min |
| 水霧（單區） | ≥ **1,200** L/min |
| 水霧（二區以上） | ≥ **2,400** L/min |

> 多設備共用幫浦時：出水量取各系統**合計值**，全揚程取**最大值**

### 全揚程

**全揚程 = 消防水帶摩擦損失 + 配管摩擦損失 + 落差 + 放水壓力（換算水柱高）**

### 性能曲線規定

| 項目 | 規定 |
|------|------|
| 額定出水量時 | 全揚程 = 標示揚程（100%~110%） |
| 出水量 **150%** 時 | 全揚程 ≥ 額定 × **65%** |
| **全閉揚程** | ≤ 額定全揚程 × **140%** |

### 幫浦本體耐壓

- 能耐最高水壓之 **1.5** 倍以上
- 加壓 **3** 分鐘後各部位無洩漏

### 電動機規定

- 額定輸出連續運轉 **8** 小時不異常
- 超過額定輸出 **15%** 運轉 **1** 小時不過熱
- 額定負荷狀態下能順利啟動

## 二、呼水裝置（110年考題）

### 必要機件

```
       ┌── 自動給水裝置（自來水/屋頂水箱→球塞閥）
       │
呼水槽 ─┼── 減水警報裝置（浮筒開關/電極式）
(≥100L)│        → 水量降至 1/3 前發警報
       │
       ├── 呼水管（連接幫浦吸水側）
       │        管徑依距離：≤1m用40A、>1m用50A
       │
       └── 溢水用排水管（口徑 ≥ 50A）
```

| 機件 | 規定 |
|------|------|
| **呼水槽** | 鋼板製+防銹處理（或防火塑膠槽）、有效儲存量 ≥ **100** L |
| **減水警報** | 水位降至容量 **1/3** 前發警報至有人駐在處 |
| **自動給水** | 自來水或屋頂水箱經球塞自動給水 |
| **呼水管** | 槽底與逆止閥中心線距離 ≤ 1m → 管徑 **40A**；> 1m → **50A** |
| **溢水排水管** | 口徑 ≥ **50A** |

## 三、啟動用水壓開關裝置（111年考題）

### 構造

```
幫浦出水側 ──逆止閥──┬── 配管系統
                      │
                 啟動用壓力水槽
                 (≥ 100 L)
                      │
                 壓力開關 ──→ 控制盤 → 啟動幫浦
```

### 規定

| 項目 | 規定 |
|------|------|
| 壓力水槽容量 | ≥ **100** L |
| 配管口徑 | ≥ **25** mm |
| 連接位置 | 幫浦出水側逆止閥之**二次側** |
| 中途裝設 | **止水閥** |
| 壓力開關 | 裝設於壓力水槽上或近傍 |
| 壓力錶 | 壓力水槽上應裝設 |
| 試驗閥 | 設試驗幫浦啟動用之**排水閥** |

### 動作原理

1. 撒水頭/消防栓開啟 → 管內壓力下降
2. 壓力水槽感知壓降 → **壓力開關**動作
3. 信號傳至控制盤 → **自動啟動幫浦**

## 四、防止水溫上升用排放裝置（111年考題）

### 用途

幫浦全閉運轉時（無出水），水在幫浦內循環導致**水溫上升**，須排放散熱

### 規定

| 項目 | 規定 |
|------|------|
| 排放管口徑 | ≥ **15** mm |
| 水溫上升限制 | 幫浦全閉連續運轉時，內部水溫不得升高至 **40°C** 以上 |
| 排放方式 | 以逆止閥+排放管回流至水源（或排水） |
| 排放量 | 足以維持幫浦內水溫不超過限值 |

---

## 五、加壓試驗

| 設備 | 試驗壓力 | 持續時間 |
|------|---------|---------|
| 室內消防栓 | ≥ 全閉揚程 × **1.5** 倍 | **2** 小時無漏水 |
| 自動撒水 | ≥ 全閉揚程 × **1.5** 倍 | **2** 小時無漏水 |
| 連結送水管 | ≥ 送水設計壓力 × **1.5** 倍 | **2** 小時無漏水 |
| 防火水幕 | ≥ 全閉揚程 × **1.5** 倍 | **2** 小時無漏水 |

---

## 六、加壓送水裝置區劃

- 使用消防幫浦之加壓送水裝置，至少以 **2小時** 防火時效之牆壁、樓地板及防火門窗**區劃分隔**

## 記憶口訣

> **「呼水槽100L、減水1/3、給水球塞、排水50A」**——呼水四件
> **「壓力水槽100L、配管25mm、二次側連」**——水壓開關
> **「排放管15mm、水溫不超40度」**——防水溫上升
> **「150%出水65%揚程、全閉140%」**——幫浦性能三點
> **「耐壓1.5倍×2小時」**——加壓試驗
