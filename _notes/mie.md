## 定義

MIE（Minimum Ignition Energy）：使可燃混合氣或粉塵雲著火所需的**最小能量**，單位為 **mJ**（毫焦耳）。

## 常見物質MIE

| 物質 | MIE (mJ) | 危險評估 |
|------|----------|---------|
| 氫氣 | **0.019** | 極低→靜電即可引燃 |
| 乙炔 | 0.019 | 極低 |
| 甲烷 | 0.28 | 低 |
| 丙烷 | 0.25 | 低 |
| 汽油蒸氣 | 0.24 | 低 |
| 粉塵（一般）| 10-100 | 中等 |
| 人體靜電 | **10-30** | 可引燃氣體，不一定引燃粉塵 |

## 重要觀念

- MIE越**小** → 越容易被點燃 → 越危險
- 氫氣和乙炔的MIE極低（0.019 mJ）→ **微小靜電就能引燃**
- 人體摩擦靜電約10-30 mJ → 足以引燃大部分可燃氣體
- 粉塵的MIE通常比氣體大 → 但堆積粉塵仍可被靜電引燃

## 影響MIE的因素

| 因素 | 影響 |
|------|------|
| 濃度 | 接近化學計量濃度時MIE最低 |
| 溫度 | 溫度升高→MIE降低 |
| 壓力 | 壓力升高→MIE降低 |
| 氧濃度 | O₂濃度升高→MIE降低 |
| 粒徑（粉塵）| 粒徑越小→MIE越低 |

## 靜電引火風險

| 靜電源 | 產生能量 | 能否引燃氣體？ | 能否引燃粉塵？ |
|--------|---------|-------------|-------------|
| 人體 | 10-30 mJ | ✅ 可以 | 可能 |
| 衣物摩擦 | 1-10 mJ | ✅ 可以 | 較難 |
| 液體流動 | 數十 mJ | ✅ 可以 | ✅ 可以 |
| 皮帶傳動 | 數百 mJ | ✅ 可以 | ✅ 可以 |

## 防靜電措施

| 措施 | 說明 |
|------|------|
| 接地 | 設備/管路接地消除靜電 |
| 增加濕度 | 相對濕度>65%→靜電不易累積 |
| 防靜電衣物 | 禁止化學纖維衣物 |
| 流速控制 | 液體管路流速不超過限值 |
| 靜候時間 | 充填後靜候讓靜電消散 |

## 記憶口訣

> **「MIE越小越容易點，氫氣0.019最危險」**
>
> 人體靜電10-30 mJ→足以引燃所有可燃氣體
> 防靜電三招：接地、加濕、穿棉衣
