## 水霧滅火原理

將水以高壓噴成極細霧滴：

| 機制 | 說明 |
|------|------|
| **冷卻** | 細小水滴表面積大，吸熱效率高 |
| **窒息** | 水滴蒸發膨脹1700倍，排擠氧氣 |
| **乳化** | 水霧打散油面，形成乳化層阻斷蒸發 |
| **稀釋** | 稀釋可燃蒸氣濃度 |

## 適用場所

| 場所 | 原因 |
|------|------|
| 變電設備 | 水霧不導電（滴粒間有空氣絕緣）|
| 油浸變壓器 | 冷卻+乳化 |
| 柴油發電機房 | B類火災有效 |
| 停車場 | 油類火災+通風良好場所 |
| 廚房油鍋 | 乳化效果 |

## 水霧 vs 撒水

| 項目 | 水霧 | 自動撒水 |
|------|------|---------|
| 水滴大小 | 極細（<1mm）| 較大 |
| 用水量 | **少**（約撒水的1/10~1/3）| 多 |
| 壓力 | 較高 | 較低 |
| 導電性 | **不導電** | 導電 |
| 水損害 | 小 | 大 |
| 適用 | 精密設備、電氣、油類 | 一般場所 |

## 細水霧分類

| 分類 | 壓力 |
|------|------|
| 低壓 | < 12.5 bar |
| 中壓 | 12.5-35 bar |
| 高壓 | > 35 bar |

## 記憶口訣

> **「水霧四效：冷卻、窒息、乳化、稀釋」**
>
> 水霧不導電→可用於電氣設備
> 用水少、水損小，但系統較貴
