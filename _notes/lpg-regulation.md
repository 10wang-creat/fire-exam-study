## 法源

消防法第 **15-1 條**、公共危險物品及可燃性高壓氣體製造儲存處理場所設置標準暨安全管理辦法。

## LPG基本特性

| 特性 | 說明 |
|------|------|
| 主成分 | 丙烷(C₃H₈)、丁烷(C₄H₁₀) |
| 比重 | 約1.5-2.0（比空氣**重**→沉地面）|
| 爆炸範圍 | 約1.8-9.5% |
| 供應方式 | 鋼瓶（桶裝瓦斯）|

## 容器（鋼瓶）管理

| 項目 | 規定 |
|------|------|
| 容器檢驗 | 定期送檢驗機構檢驗 |
| 合格標示 | 須有檢驗合格標示 |
| 使用年限 | 超過年限須報廢 |
| 充填量 | 不得超過容器標示之最大充填量 |

## 儲存場所規定

### 零售場所

| 項目 | 規定 |
|------|------|
| 儲存量 | ≤ **128 kg**（8桶×16kg）|
| 防止傾倒 | 鋼瓶須固定防傾倒 |
| 通風 | 設置良好通風設備 |
| 距離 | 與火源保持安全距離 |

### 分裝場所
- 須取得許可
- 設置氣體漏氣檢知警報設備
- 嚴禁煙火

## 配管規定

- 使用金屬管或經認可之軟管
- 軟管長度不得超過一定限制
- 接頭須使用防脫落裝置

## 罰則

| 違規 | 罰鍰 |
|------|------|
| 未經許可經營 | 2-5萬元 |
| 逾期未檢驗容器繼續使用 | 1-5萬元 |
| 違反儲存規定 | 1-5萬元 |

## 記憶口訣

> **「LPG比空氣重往下沉，鋼瓶要固定、要檢驗」**
>
> 零售儲存≤128kg（8桶）
> LPG洩漏沉地面→檢知器裝地板
