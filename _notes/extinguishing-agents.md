## 滅火四原理

| 原理 | 說明 |
|------|------|
| 冷卻 | 降低溫度至燃點以下 |
| 窒息 | 隔絕氧氣（O₂<15%不燃） |
| 移除 | 移除可燃物 |
| 抑制連鎖反應 | 負催化作用，中斷自由基連鎖 |

## 各藥劑比較

| 藥劑 | 主要原理 | 適用 | 不適用 | 特點 |
|------|---------|------|--------|------|
| 水 | **冷卻** | A類 | B(油浮)、C(導電)、D(反應) | 最便宜、冷卻效果最好 |
| 泡沫 | **窒息**+冷卻 | A、B類 | C(導電)、D | 覆蓋油面隔氧 |
| CO₂ | **窒息**+冷卻 | B、C類 | A(復燃)、D | 不導電、不殘留 |
| 乾粉 | **抑制連鎖反應** | 視種類 | D類 | 速效但會復燃 |
| 海龍(Halon) | **抑制連鎖反應** | A、B、C | - | 已禁用(破壞臭氧層) |
| HFC系列 | 抑制+少量冷卻 | A、B、C | - | 海龍替代品、潔淨 |
| IG-541 | **窒息** | A、B、C | - | 惰性氣體混合(N₂+Ar+CO₂) |

## 海龍替代品

海龍（Halon 1301、1211）因破壞臭氧層已被蒙特婁議定書禁用，替代品：

| 替代品 | 組成 | 特點 |
|--------|------|------|
| HFC-227ea（FM-200）| 七氟丙烷 | 最常用替代品，低毒 |
| HFC-236fa | 六氟丙烷 | 類似FM-200 |
| IG-541（Inergen）| 52%N₂+40%Ar+8%CO₂ | 天然氣體，最環保 |
| IG-55 | 50%N₂+50%Ar | 惰性氣體 |
| FK-5-1-12（Novec 1230）| 氟化酮 | ODP=0, GWP極低 |

## 水的滅火特性

- 比熱容最大（4.2 kJ/kg·°C）→ 冷卻效果最佳
- 汽化潛熱大（2,260 kJ/kg）→ 吸收大量熱能
- 1 kg水從20°C→100°C蒸發 = 吸收 **2,594 kJ**
- 水蒸氣體積是液態的**1,700倍** → 窒息效果
