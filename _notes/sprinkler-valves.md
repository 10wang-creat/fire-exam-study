## 流水檢知裝置（Alarm Check Valve）

### 功能
偵測管路中的水流動 → 確認撒水頭已開放 → 發出警報信號。

### 原理
撒水頭開放→水流動→閥瓣被推開→啟動壓力開關→警報

### 組成

| 組件 | 功能 |
|------|------|
| 閥體 | 主體，含閥瓣 |
| 壓力開關 | 水流推動→電氣信號→通知受信總機 |
| 延遲裝置 | 防止水錘造成誤報（延遲5-60秒）|
| 排水閥 | 試驗排水用 |

### 設置
- 每一樓層或每一防護區域設一組
- 設於容易維修操作的位置

## 末端查驗閥（Test Valve）

### 功能
在配管**最末端**設置的測試閥，模擬一個撒水頭開放，用於測試系統是否正常。

### 測試流程
1. 打開末端查驗閥
2. 水流出 → 管內壓力降低
3. 流水檢知裝置應動作
4. 壓力開關發出信號
5. 消防幫浦應自動啟動（如有設定）
6. 確認整個系統連動正常

### 規格

| 項目 | 規定 |
|------|------|
| 開口面積 | 等同一個撒水頭開口 |
| 設置位置 | 配管最末端（最不利點）|
| 排水 | 設有排水管（不噴灑在室內）|

## 一齊開放閥（Deluge Valve）

### 功能
控制**開放式撒水系統**和**泡沫系統**，接收信號後一次打開，讓水/泡沫同時從所有噴頭噴出。

### 特點

| 項目 | 說明 |
|------|------|
| 適用系統 | 開放式撒水、水霧、泡沫系統 |
| 平時 | 關閉狀態，管路下游無水 |
| 啟動方式 | 火警探測器連動 或 手動啟動 |
| 動作 | 一次打開 → 全區放水 |

### 啟動方式

| 方式 | 原理 |
|------|------|
| 電動式 | 探測器信號→電磁閥→開放 |
| 氣動式 | 感溫管破裂→壓力釋放→開放 |
| 手動式 | 人員操作手動開放 |

## 三閥比較

| 閥類 | 用於 | 功能 |
|------|------|------|
| 流水檢知 | 密閉式系統 | 偵測水流→報警 |
| 末端查驗 | 密閉式系統 | 末端測試 |
| 一齊開放 | 開放式系統 | 控制全區放水 |

## 記憶口訣

> **「流水報警、末端測試、一齊開放」**
>
> 流水檢知 = 水流→報警（密閉式用）
> 末端查驗 = 最遠端打開測試系統
> 一齊開放 = 開放式/泡沫全區一次放
