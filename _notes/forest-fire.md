## 森林火災種類

| 種類 | 位置 | 特性 |
|------|------|------|
| **地表火** | 地面 | 燒茅草、灌木，最常見，強度增強可轉為樹冠火 |
| **樹冠火** | 樹頂 | 多由地表火引起，針葉樹（含油脂）易發，蔓延快 |
| **地中火** | 地下 | 燃燒地下有機物（泥炭），難以發現，悶燒持續數月 |
| **飛火** | 遠距 | 火品飛散引燃遠處，火線強度≥**2.5 MW/m**時可能出現 |

## 森林火災特性

- 地表火呈**花斑狀**蔓延
- 針葉樹比闊葉樹更易發生樹冠火（含樹脂）
- 地中火無煙無焰，極難偵測
- 火旋風可能在大型森林火災中出現

## 微火源

### 定義
能量較小但能引燃可燃物的火源。

### 種類

| 微火源 | 溫度/特性 |
|--------|----------|
| **香煙（菸蒂）** | 中心約700-800°C，可引燃棉被（陰燃）|
| **線香/蚊香** | 可燃燒7-8小時 |
| **火花/火星** | 高溫但短暫，對下風可燃物有引火危險 |
| 火柴 | 小型明火 |

### 非微火源
- **打火機** → 屬於明火（非微火源）
- 焊接火焰 → 明火

## 火源分類

| 分類 | 範例 |
|------|------|
| 明火（裸火）| 打火機、蠟燭、爐火 |
| **微火源** | 香煙、線香、火星 |
| 高溫表面 | 電熱器、排氣管 |
| 電氣火花 | 開關火花、靜電放電 |
| 化學反應 | 自然發火、混合發火 |

## 實效濕度

日本用於預測火災危險的指標：

`H_eff = H₀ + 0.7×H₁ + 0.7²×H₂ + ...`

- H₀ = 當日相對濕度
- H₁ = 前一天
- H₂ = 再前一天
- 實效濕度 < **60%** → 日本發布火災警報

## 記憶口訣

> **「地表最常見，樹冠最猛烈，地中最難找」**
>
> 香煙是微火源，打火機不是（打火機=明火）
> 飛火在火線強度≥2.5 MW/m時出現
> 實效濕度<60%→火災警報
