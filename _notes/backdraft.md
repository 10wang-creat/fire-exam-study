## 定義

密閉空間火災因缺氧而進入悶燒狀態，當突然開啟門窗引入新鮮空氣時，高溫可燃氣體與氧氣急速混合，產生**爆炸性燃燒**。

## 發生機制（四步驟）

1. 密閉空間火災 → O₂消耗 → 濃度降至 **<15%**
2. 火焰熄滅但高溫持續 → **悶燒（陰燃）**
3. 室內充滿高溫未燃可燃氣體（CO等）
4. **開啟門窗** → O₂湧入 → 可燃氣體與O₂混合 → **爆燃**

## 發生前兆

| 前兆 | 說明 |
|------|------|
| 門窗異常高溫 | 觸摸門板燙手 |
| 濃煙從門縫脈動式滲出 | 煙像呼吸般一陣一陣 |
| 門窗向外鼓脹 | 內部正壓 |
| 窗戶玻璃變色/變黑 | 高溫碳化 |
| 煙色轉為黃棕色 | 高溫不完全燃燒產物 |

## 閃燃 vs 複燃 完整比較

| 項目 | 閃燃 Flashover | 複燃 Backdraft |
|------|---------------|----------------|
| **根本原因** | **熱能**累積 | **氧氣**補充 |
| **O₂狀態** | 充足 | 嚴重不足→突然補充 |
| **空間狀態** | 通風良好 | **密閉** |
| **觸發時機** | 成長期→最盛期 | 衰退期或悶燒中 |
| **觸發動作** | 自然發生(熱累積) | **人為開門窗** |
| **現象** | 全面燃燒 | **爆炸性燃燒（火焰噴出）** |
| **火焰方向** | 室內向上擴展 | **從開口向外噴出** |
| **發生機率** | 較高 | 較低 |
| **對消防員危險** | 高 | **極高**（無預警爆炸）|

## 消防員應對

- 進入前先用**紅外線測溫儀**偵測門板溫度
- 觀察煙的脈動現象
- 不可貿然開門
- 先在門上方開小孔通風（破壞式通風）
- 使用水霧降溫後再進入

## 記憶口訣

> **「複燃 = 缺氧悶燒 + 開門補氧 = 爆」**
>
> 前兆：門燙、煙脈動、門鼓脹、玻璃黑
> 閃燃看「熱」，複燃看「氧」
