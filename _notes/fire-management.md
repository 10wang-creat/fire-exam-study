## 法源

消防法第 **13 條**：一定規模以上之場所，應由管理權人遴用防火管理人。

## 防火管理人

### 資格
- 經中央消防機關認可之訓練機構施予**初訓合格**
- 初訓時數 ≥ 12 小時
- 每**3年**接受**複訓** ≥ 6 小時

### 職責
1. 制定**消防防護計畫**
2. 實施滅火、通報及避難訓練
3. 維護消防安全設備
4. 火災及其他災害發生時的應變
5. 防止縱火措施

## 消防防護計畫

### 必備內容

| 項目 | 內容 |
|------|------|
| 自衛消防編組 | 各班編組人員名單及任務 |
| 防火避難設施維護 | 防火門、避難通道管理 |
| 滅火及通報訓練 | 每半年至少1次 |
| 用火用電管理 | 吸菸管理、電器使用 |
| 防止縱火對策 | 加強巡邏、管理出入口 |
| 收容人員避難管理 | 避難引導、弱勢人員協助 |

## 自衛消防編組

| 編組 | 任務 |
|------|------|
| 滅火班 | 初期滅火 |
| 通報班 | 通報119、通報大樓管理 |
| 避難引導班 | 引導人員避難 |
| 安全防護班 | 關閉防火門/捲門 |
| 救護班 | 傷者急救 |

## 滅火通報訓練

- 每**半年**至少實施 **1 次**
- 事先通報消防機關
- 訓練紀錄保存

## 甲類場所加強規定

- 員工人數 ≥ 10 人：編組自衛消防隊
- 應設場所專用滅火、通報、避難設備
- 每年實施**綜合訓練**

## 罰則

| 違規 | 罰鍰 |
|------|------|
| 未遴用防火管理人 | 1-5 萬元 |
| 未提報消防防護計畫 | 1-5 萬元 |
| 未實施訓練 | 1-5 萬元 |

## 記憶口訣

> **「防火管理人做五件事：計畫、訓練、維護、應變、防縱火」**
>
> 初訓12hr，每3年複訓6hr
> 訓練每半年1次
> 自衛編組：滅火、通報、避難、安全、救護
