## 一、閃燃（Flashover）

### 定義

火災成長期中，室內可燃物表面受**輻射熱回饋**加熱，當達到臨界條件時，所有可燃物表面**幾乎同時起火**，火災從局部燃燒瞬間轉變為**全面燃燒**的過渡現象。

### 發生機制

```
局部火源燃燒
    ↓ 熱煙層蓄積於天花板
    ↓ 天花板熱煙層溫度升高
    ↓ 輻射熱回饋至室內所有表面
    ↓ 可燃物表面溫度持續上升
    ↓ 各表面達到熱裂解溫度
    ↓ 揮發性可燃氣體大量釋出
    ↓↓↓
💥 閃燃（全面起火）
```

### NFPA 265 量化標準（105年考題核心）

| 條件（符合**任一**即判定為閃燃） | 數值 |
|-------------------------------|------|
| 天花板處熱氣流溫度 | ≥ **600°C** |
| 地板面熱通量 | ≥ **20 kW/m²** |
| 地板到天花板的火焰延伸 | 火焰到達**天花板** |
| 紙指示物 | 紙指示物**起火** |

### 閃燃特徵

- 發生在火災**成長期→全盛期**轉換時
- 室內空間有**足夠氧氣**
- 轉換極為迅速（秒級）
- 閃燃後溫度可達 **800~1,000°C**
- 閃燃後為**通風控制燃燒**

---

## 二、複燃（Backdraft）

### 定義

在**通風不足**的密閉空間中，火災因缺氧而進入衰退悶燒階段，室內蓄積大量**可燃性熱裂解氣體及不完全燃燒產物**（CO等）。當突然提供新鮮空氣（如開門/破窗）時，可燃氣體與氧氣混合後**爆燃**的現象。

### 發生機制

```
密閉空間火災
    ↓ 氧氣逐漸耗盡
    ↓ 火勢衰退但持續悶燒
    ↓ 室內蓄積CO、可燃氣體、高溫
    ↓ 空間處於「富燃料、貧氧」狀態
    ↓
開門/破窗引入新鮮空氣
    ↓ 氧氣與可燃氣體混合
    ↓ 達到燃燒條件
    ↓↓↓
💥 複燃（爆燃）
    → 火球噴出開口
    → 巨大壓力波
```

### 複燃徵兆（消防員入室前判斷）

| 徵兆 | 說明 |
|------|------|
| 門窗**高溫** | 觸摸門把燙手 |
| 濃黑煙從縫隙**脈動式**噴出 | 煙呈規律「呼吸」狀 |
| 門窗玻璃**油煙汙黑** | 可燃氣體凝結 |
| 煙層**壓至極低** | 幾乎到地面 |
| **無火焰**但高溫 | 典型缺氧燃燒 |

## 三、閃燃 vs 複燃比較表（112年考題核心）

| 比較項目 | 閃燃 Flashover | 複燃 Backdraft |
|---------|---------------|---------------|
| **發生階段** | 成長期→全盛期 | 衰退期→突然復燃 |
| **空間氧氣** | **充足** | **不足**（缺氧） |
| **觸發條件** | 輻射熱回饋達臨界 | **供氧**（開門/破窗） |
| **機制** | 全部表面同時起火 | 可燃氣體與新空氣**爆燃** |
| **火焰方向** | 由上往下蔓延 | **由內往外噴出**（火球） |
| **壓力效應** | 無明顯壓力波 | 有**壓力波**（類似爆炸） |
| **溫度條件** | 天花板 ≥ 600°C | 室內高溫但火焰衰退 |
| **預防** | 早期滅火、限制可燃物 | 入室前**先排煙通風** |
| **對消防員威脅** | 困在室內者極危險 | 開門瞬間**最危險** |

## 四、通風控制 vs 燃料控制（相關概念）

| 燃燒類型 | 限制因素 | 特徵 |
|---------|---------|------|
| **燃料控制** | 可燃物量 | 氧氣充足，火勢大小取決於可燃物 |
| **通風控制** | 空氣供應量 | 閃燃後常轉為通風控制 |

> 閃燃前＝燃料控制，閃燃後＝通風控制
