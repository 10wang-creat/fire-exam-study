## 管理權人定義

| 身分 | 定義 |
|------|------|
| **管理權人** | 對建築物具有管理權限之人 |
| 通常為 | 所有人、使用人或管理人 |

- 建築物**所有人** = 產權持有者
- 建築物**使用人** = 承租使用者
- 建築物**管理人** = 受託管理者

## 管理權人的法定責任

| 責任 | 法源 |
|------|------|
| 設置消防安全設備 | 消防法§6 |
| 維護消防安全設備 | 消防法§6 |
| 定期**檢修申報** | 消防法§9 |
| 使用**防焰物品** | 消防法§11 |
| 遴用**防火管理人** | 消防法§13 |
| 制定**消防防護計畫** | 消防法§13 |
| 實施**滅火通報訓練** | 消防法§13 |
| 接受消防機關檢查 | 消防法§6 |

## 共同管理權人（多人共有）

### 問題
一棟建築物有多個管理權人時（如百貨公司各專櫃、辦公大樓各層租戶），誰負責？

### 處理方式

| 項目 | 規定 |
|------|------|
| 共用部分 | 由**建築物所有人**或**管理委員會**負責 |
| 各自使用部分 | 由各管理權人自行負責 |
| 協議 | 各管理權人應**協議**共同維護公共設備 |
| 協議不成 | 由消防機關指定 |

### 常見情境
- 百貨公司：各專櫃為使用人，整棟設備由百貨公司管理
- 辦公大樓：各層租戶各自負責，公共消防設備由管委會負責
- 複合用途：不同用途各管理權人負責各自部分

## 管理權人 vs 防火管理人

| 項目 | 管理權人 | 防火管理人 |
|------|---------|----------|
| 角色 | **雇主/老闆** | **被指派的員工** |
| 責任 | 法律上的最終責任人 | 實際執行消防管理 |
| 設置 | 自然存在 | 由管理權人遴用 |
| 罰則對象 | **管理權人**被罰 | 僅管理權人被罰 |

> 防火管理人做事，但出事是**管理權人**被罰！

## 記憶口訣

> **「管理權人是最終責任人，出事他被罰」**
>
> 設置+維護+檢修+防焰+管理=全是管理權人的事
> 多人共有→協議分擔→協議不成→消防機關指定
