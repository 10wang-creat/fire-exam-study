## 滅火設備設置門檻

### 滅火器

| 場所 | 門檻 |
|------|------|
| 甲類場所 | **全部** |
| 乙類場所 | **全部** |
| 其他 | ≥ 150 m² |

### 室內消防栓

| 場所 | 5層以下 | 6層以上 |
|------|---------|--------|
| 甲類 | ≥ **300 m²** | ≥ **150 m²** |
| 乙類 | ≥ **500 m²** | ≥ **300 m²** |
| 丙類 | ≥ 500 m² | ≥ 300 m² |
| 丁類 | ≥ 500 m² | ≥ 500 m² |

### 自動撒水設備

| 場所 | 門檻 |
|------|------|
| 甲-1~甲-3（娛樂場所） | ≥ **300 m²** |
| 甲-4（百貨商場） | ≥ **1,000 m²** |
| 甲-5（旅館） | ≥ **300 m²**（11層以上全設）|
| 甲-6（醫院、安養） | ≥ **300 m²** |
| 地下層 | ≥ **1,000 m²** |
| 11層以上 | **全部** |

### 泡沫/水霧/乾粉/CO₂

- 依特定場所用途設置（停車場、發電機房等）
- 不以面積為主要門檻，而是依**場所用途**

## 警報設備設置門檻

### 火警自動警報設備

| 場所 | 門檻 |
|------|------|
| 甲類場所 | ≥ **300 m²** |
| 乙類場所 | ≥ **500 m²** |
| 11層以上 | **全部** |
| 地下建築物 | **全部** |

### 緊急廣播設備

| 場所 | 門檻 |
|------|------|
| 甲類 | ≥ **300 m²** |
| 11層以上 | **全部** |
| 地下建築物 | **全部** |

## 避難設備設置門檻

### 標示設備（出口標示燈+避難方向指示燈）

| 場所 | 門檻 |
|------|------|
| 甲、乙類場所 | **全部** |
| 地下層 | **全部** |

### 緊急照明

| 場所 | 門檻 |
|------|------|
| 甲類場所 | **全部**（居室+通道）|
| 地下層 | **全部** |
| 無開口樓層 | **全部** |
| 11層以上 | **全部** |

### 避難器具

| 場所 | 門檻 |
|------|------|
| 甲類 2F以上 | 依收容人數 |
| 3F以上 | 各樓層依收容人數 |
| 設有兩座安全梯 | **免設** |

## 搶救設備設置門檻

| 設備 | 門檻 |
|------|------|
| 連結送水管 | **7層**以上 |
| 緊急電源插座 | **11層**以上 |
| 緊急昇降機 | **11層**以上 |
| 無線電通信 | 地下建築物(≥1000m²) |
| 排煙設備 | ≥ **500 m²** 樓層 |

## 常考「11層以上」設備

11層以上建築物須設置：
- 自動撒水設備
- 緊急廣播
- 緊急照明
- 緊急電源插座
- 緊急昇降機
- 防焰物品
- 特別安全梯（15層以上）

## 記憶口訣

> **「甲類300、乙類500是基本門檻」**
> **「11層以上什麼都要設」**
>
> 室內消防栓：甲300/150、乙500/300
> 撒水：娛樂旅館醫院300，百貨1000
> 火警：甲300、乙500
