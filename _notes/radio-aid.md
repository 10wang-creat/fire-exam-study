## 功能

確保消防人員在地下層或鋼筋混凝土建築內部能使用無線電通訊，克服電波遮蔽問題。

## 設置場所

- **地下建築物**（地下層≥1000m²）
- **地下層3層以上**且面積≥1000m²
- 其他電波遮蔽嚴重場所

## 系統組成

| 組件 | 功能 |
|------|------|
| 洩波同軸電纜 | 沿走廊佈設，發射/接收電波 |
| 增幅器(中繼器) | 放大信號，克服衰減 |
| 無線電接頭 | 供消防員連接手持無線電 |
| 分配器 | 分配信號至各區域 |
| 電源設備 | 含緊急電源 |

## 洩波同軸電纜原理

- 外層屏蔽開有間隔的**槽孔**
- 電波從槽孔「洩漏」出來 → 覆蓋周圍空間
- 同時也能接收消防員的發射信號
- 沿走廊、通道佈設，確保沿線通訊暢通

## 規格要求

| 項目 | 規定 |
|------|------|
| 使用頻率 | 消防專用頻率 |
| 涵蓋範圍 | 所有地下層走廊、樓梯間 |
| 緊急電源 | 停電後可運作 **30分鐘** 以上 |
| 接頭設置 | 各層均設無線電接頭 |

## 記憶口訣

> **「地下通訊靠洩波，電纜開孔漏信號」**
>
> 地下建築物≥1000m²必設
> 確保消防員地下搶救時通訊暢通
