## 中繼幫浦

### 為什麼需要？
高層建築中，地下幫浦送水至高樓層 → 壓力損失大 → 可能壓力不足。
設置中繼幫浦在中間樓層，接力加壓送水。

### 設置條件
- 建築物高度使消防幫浦無法直接送達足夠壓力時
- 通常在建築物的**中間技術層**

### 規格

| 項目 | 規定 |
|------|------|
| 出水量 | ≥ 與主幫浦相同 |
| 揚程 | 足以送達最上層 |
| 啟動方式 | 與主幫浦連動自動啟動 |
| 電源 | 連接緊急電源 |
| 位置 | 專用幫浦室或技術層 |

### 串聯/並聯

| 方式 | 效果 |
|------|------|
| **串聯** | 增加**揚程**（壓力疊加）|
| 並聯 | 增加**出水量** |

> 高層建築中繼幫浦通常是**串聯**→增加揚程克服高度

## 屋頂水箱（重力水箱）

### 功能
設於建築物最高處，利用重力提供消防設備初期用水，在幫浦啟動前即可供水。

### 容量

| 設備 | 屋頂水箱容量 |
|------|------------|
| 室內消防栓 | ≥ 所需水量的 **1/3** |
| 自動撒水 | ≥ 所需水量的 **1/3** |

### 設置位置

| 項目 | 規定 |
|------|------|
| 高度 | 建築物**最高處** |
| 落差 | 距最高層消防栓 ≥ 一定高度 |
| 壓力 | 每10m落差 ≈ 1 kgf/cm² |

### 優點
- 幫浦啟動前（0-30秒）即有水壓
- 不受停電影響（重力供水）
- 初期滅火用水保障

### 補水
- 自動補水裝置（浮球閥）
- 水位下降 → 自動補水
- 專用管路供水

## 供水系統完整架構

```
蓄水池 → 主幫浦 → (中繼幫浦) → 立管 → 各層出水口
                                    ↑
                              屋頂水箱（重力補充）
```

## 記憶口訣

> **「中繼串聯加揚程，屋頂水箱靠重力」**
>
> 中繼幫浦：中間接力加壓，串聯增壓
> 屋頂水箱：容量≥需求的1/3，停電也有水
> 10m落差≈1 kgf/cm²水壓
