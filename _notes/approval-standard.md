## 認可制度

消防機具器材及設備須經中央主管機關**認可**後，始得銷售、陳列或設置。

## 認可方式

| 方式 | 說明 | 適用 |
|------|------|------|
| **型式認可** | 對設備的設計、規格進行認可 | 大量生產設備 |
| **個別認可** | 每批或每個產品逐一檢驗 | 特殊/少量設備 |
| **型式試驗** | 抽樣進行破壞性測試 | 新產品上市前 |

## 需要認可的主要設備

| 設備 | 認可項目 |
|------|---------|
| 滅火器 | 藥劑量、放射距離、放射時間 |
| 探測器 | 動作溫度/濃度、反應時間 |
| 撒水頭 | 動作溫度、放水量、分布 |
| 消防幫浦 | 出水量、揚程、材質 |
| 緊急照明 | 照度、持續時間 |
| 避難器具 | 強度、下降速度 |
| 受信總機 | 迴路數、功能 |

## 滅火器認可基準重點

| 項目 | 規定 |
|------|------|
| 放射距離 | ≥ **3m**（乾粉10型）|
| 放射時間 | ≥ **15秒**（一般型）|
| 使用溫度 | -20°C ~ +40°C |
| 耐壓 | 試驗壓力 ≥ 工作壓力 × 1.5 |
| 有效期限 | 依藥劑種類（通常5-10年）|

## 探測器認可基準重點

| 探測器 | 認可重點 |
|--------|---------|
| 差動式 | 溫度上升率15°C/min時應動作 |
| 定溫式 | 在標示溫度±5°C內動作 |
| 偵煙式 | 減光率達設定值時動作 |
| 火焰式 | 在設定距離內偵測火焰 |

## 撒水頭認可基準重點

| 項目 | 規定 |
|------|------|
| 動作溫度 | 在標示溫度 ± 規定範圍 |
| 水量分布 | 均勻覆蓋防護面積 |
| 耐腐蝕 | 耐鹽水噴霧試驗 |
| 強度 | 耐水壓試驗 |

## 記憶口訣

> **「所有消防設備上市前都要認可」**
>
> 型式認可 = 設計OK，個別認可 = 每批檢驗
> 滅火器：放射≥3m、放射≥15秒
> 探測器：差動式15°C/min動作
