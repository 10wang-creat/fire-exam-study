## 原理

發光部與受光部**分離設置**（相對兩面牆），中間形成一條光線。煙進入光路 → 光線被遮蔽（減光）→ 受光量低於設定值 → 動作。

## 特點

| 項目 | 說明 |
|------|------|
| 偵測方式 | 減光式（煙遮光）|
| 適用 | **大空間、高天花板**（體育館、倉庫、工廠）|
| 監視距離 | 最遠 **100m** |
| 安裝高度 | 可達 **15-20m** |
| 優點 | 一組即可涵蓋大面積 |
| 缺點 | 灰塵、振動可能誤報 |

## vs 局限型偵煙探測器

| 項目 | 局限型（一般偵煙）| 分離型（線型）|
|------|---------------|-------------|
| 構造 | 發光+受光在同一個探測器 | 分離在兩面牆上 |
| 偵測範圍 | 小區域（單一房間）| 大空間（長距離）|
| 天花板高度 | ≤ 15m | **≤ 20m** |
| 適用 | 辦公室、走廊 | 倉庫、體育館、工廠 |
| 成本 | 每個便宜但需多個 | 單價高但數量少 |

## 設置規定

| 項目 | 規定 |
|------|------|
| 監視距離 | ≤ 100m |
| 距天花板 | 光軸距天花板 **80cm** 以內 |
| 距側牆 | ≥ 60cm |
| 光軸高度 | 距地板 **80%** 以上 |

## 記憶口訣

> **「分離型 = 大空間專用，發光受光隔兩牆」**
>
> 最遠100m，高度可到20m
> 體育館、倉庫、大廠房首選
