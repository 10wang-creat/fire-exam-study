# 🔥 消防設備士備考系統

歷屆考題分析 → 圖解筆記 → 練題 → 錯題回連

## 資料規模

- 100-114年（15年）× 4科 = 60份考題
- 2,402 題選擇題（含答案 99.9%）
- 96 題申論題（103-114年）
- 10+ 圖解筆記（持續擴充）

## 架構

- **Jekyll** 靜態網站產生器
- **Vercel** 部署
- 題庫資料存在 `assets/data/*.json`（每科一個檔案）
- 圖解筆記存在 `_notes/*.md`（每考點一個檔案）

## 本機開發

```bash
bundle install
bundle exec jekyll serve
```

## 部署

推到 GitHub 後 Vercel 自動部署。
